# 3.6 漂浮残骸系统

> 对标 Brotato 的"树"机制，提供战斗中额外的资源与回复途径

---

## 一、设计概述

### 1.1 系统定位

**漂浮残骸**是战场上可被摧毁的中立物体，类似 Brotato 中的"树"（Trees）。玩家可以在战斗中攻击并摧毁残骸，获取**晶体**、**护盾电池**（恢复护盾）或**补给箱**（随机模块）。

| 对标 | Brotato Trees | 星河战姬 漂浮残骸 |
|------|---------------|-------------------|
| 名称 | Trees（树） | 漂浮残骸 |
| 世界观 | 战场上的植被 | 太空中漂浮的废弃物/残骸 |
| **必定掉落** | **3材料** | **3-15 晶体（随波次增加）** |
| **主掉落物** | **水果（恢复HP）** | **护盾电池（80%，恢复护盾）** |
| **额外掉落** | **20%概率掉落箱子** | **补给箱（20%，随机模块）** |
| HP | 10 (+5/波) | 10 (+5/波) |
| 速度 | 0（静止） | 0（静止） |

### 1.2 核心机制 ⭐ 重点

```
漂浮残骸 → 玩家攻击 → 残骸被摧毁 
                ↓
    ┌──────────┴──────────┬────────────┐
    ↓                     ↓            ↓
 必定掉落             80%概率      20%概率
  晶体               护盾电池      补给箱
 (3-15个)           (恢复护盾)    (随机模块)
```

**晶体**：
- 100%掉落，数量随波次增加（3-15个）
- 直接计入玩家晶体

**护盾电池**：
- 80%概率掉落
- 只恢复护盾值（受护盾回复属性加成）
- 不提供晶体

**补给箱**：
- 20%概率掉落
- **只包含模块**（不含武器和晶体）
- 波次结束后统一开箱

### 1.3 与Brotato对标

| Brotato Trees | 星河战姬 漂浮残骸 |
|---------------|-------------------|
| 3材料（固定） | 3-15晶体（直接掉落，随波次增加） |
| 水果（100%掉落，恢复HP） | 护盾电池（80%掉落，恢复护盾） |
| 箱子（20%概率，含道具/材料） | 补给箱（20%概率，**只含模块**） |
| HP: 10 (+5/波) | 耐久: 10 (+5/波) |
| 消耗品恢复受相关属性影响 | 护盾恢复受相关属性影响 |

## 二、掉落物详解

### 2.1 晶体（必定掉落）⭐

残骸被摧毁时**必定掉落晶体**，数量随波次增加。

| 属性 | 数值 | 说明 |
|------|------|------|
| **基础数量** | 3 个 | 波次1的掉落量 |
| **每波增加** | +0.5 个 | 渐进增长 |
| **数量上限** | 15 个 | 波次20+时封顶 |
| **获得方式** | 直接获得 | 残骸摧毁时立即计入 |

**晶体掉落数量表**：

| 波次范围 | 晶体数量 | 说明 |
|----------|----------|------|
| **1-5** | 3-5 | 早期基础收益 |
| **6-10** | 5-7 | 中期稳定收益 |
| **11-15** | 8-11 | 后期高收益 |
| **16-20** | 12-15 | 最大收益 |

> ⚠️ **注意**：晶体在残骸摧毁时立即获得，不需要拾取

### 2.2 护盾电池（80%掉落）⭐

**护盾电池**是残骸被摧毁时80%概率掉落的恢复道具，需要玩家拾取。

| 属性 | 数值 | 说明 |
|------|------|------|
| **掉落概率** | 80% | 每个残骸独立判定 |
| **护盾恢复** | 8 点 | 基础恢复量，受"护盾回复"属性加成 |
| **拾取方式** | 接触拾取 | 需要玩家主动靠近 |
| **存在时间** | 永久 | 直到被拾取或波次结束 |
| **拾取范围** | 1.5 米 | 进入范围自动拾取 |
| **磁吸范围** | 3.0 米 | 进入范围开始飞向玩家 |

**护盾恢复计算**：
```
实际恢复 = 基础恢复(8) × (1 + 护盾回复%)
```

**护盾电池详细规则**：

| 规则 | 说明 |
|------|------|
| **拾取优先级** | 护盾电池不会被敌舰抢走，只有玩家可以拾取 |
| **上限检测** | 护盾恢复不会超过护盾上限 |
| **波次结束处理** | 波次结束时，场上未拾取的护盾电池自动消失 |
| **磁力牵引模块** | 装备后护盾电池会自动高速飞向玩家 |

> ⚠️ **重要**：护盾电池只恢复护盾，不提供晶体（晶体由残骸直接掉落）

### 2.3 补给箱（20%概率）⭐

残骸被摧毁时有20%概率掉落补给箱，**只包含模块**。

| 属性 | 数值 | 说明 |
|------|------|------|
| **掉落概率** | 20% | 每个残骸独立判定 |
| **内容物** | 随机模块 | **不含武器和晶体** |
| **拾取方式** | 接触拾取 | 需要玩家主动靠近 |
| **存在时间** | 永久 | 直到被拾取或波次结束 |
| **打开时机** | 波次结束后 | 统一在波次结算界面打开 |

**补给箱详细规则**：

| 规则 | 说明 |
|------|------|
| **只含模块** | 补给箱内容100%为模块，不会掉落武器或晶体 |
| **不立即打开** | 战斗中拾取的补给箱存入"待开箱"列表 |
| **波次结束打开** | 波次结束后统一显示所有待开箱子 |
| **选择机制** | 玩家可选择"获取模块"或"回收换晶体" |
| **波次结束处理** | 场上未拾取的补给箱自动消失 |

> ⚠️ **重要**：补给箱只包含模块，是获取模块的额外途径（商店外）

### 2.4 模块品阶掉落概率

**模块品阶概率表**（按波次）：

| 波次范围 | T1（普通） | T2（精良） | T3（稀有） | T4（传说） |
|----------|-----------|-----------|-----------|-----------|
| **1-3** | 100% | 0% | 0% | 0% |
| **4-7** | 75% | 25% | 0% | 0% |
| **8-12** | 55% | 35% | 9% | 1% |
| **13-20** | 40% | 40% | 17% | 3% |

> **设计说明**：模块品阶概率与 Brotato 箱子掉落一致，后期有机会获得高品质模块

## 三、波次结束流程 ⭐ 重点

### 3.1 流程概述

```
波次战斗结束
       ↓
  清算场上掉落物
  (未拾取的护盾电池和补给箱消失)
       ↓
  显示本波获得的补给箱
       ↓
  ┌────────────────────────────┐
  │ 补给箱开启界面              │
  │ ┌──────────┐ ┌──────────┐  │
  │ │ 箱子 1    │ │ 箱子 2    │  │
  │ │ 🔧 模块   │ │ 🔧 模块   │  │
  │ │ T2 引擎   │ │ T1 电池   │  │
  │ │ 回收: 12💎 │ │ 回收: 4💎 │  │
  │ │[获取][回收]│ │[获取][回收]│  │
  │ └──────────┘ └──────────┘  │
  │                            │
  │      [全部获取] [全部回收]   │
  └────────────────────────────┘
       ↓
  进入商店界面
```

### 3.2 补给箱开启界面

| 元素 | 说明 |
|------|------|
| **箱子列表** | 显示本波收集的所有补给箱（已预先决定内容） |
| **内容预览** | 每个箱子显示模块名称、品阶和回收价值 |
| **获取按钮** | 将模块加入装备栏 |
| **回收按钮** | 将模块转换为晶体（见回收价值表） |
| **批量操作** | "全部获取"和"全部回收"快捷按钮 |

### 3.3 回收机制（对标 Brotato）

#### 回收价值计算

**Brotato 原版机制**：回收返还道具**当前价格**的一定百分比

| 属性 | 默认值 | 说明 |
|------|--------|------|
| **基础回收率** | 25% | 默认返还模块价格的 25% |
| **回收效率加成** | 可叠加 | 通过道具提升 |

#### 回收计算公式

```
回收获得晶体 = 模块当前价格 × (基础回收率 + 回收效率加成)
```

**示例**（假设回收效率 +35%）：
| 模块品阶 | 当前价格 | 回收获得 |
|----------|----------|----------|
| T1 模块 | 15 晶体 | 15 × 60% = 9 晶体 |
| T2 模块 | 40 晶体 | 40 × 60% = 24 晶体 |
| T3 模块 | 80 晶体 | 80 × 60% = 48 晶体 |
| T4 模块 | 150 晶体 | 150 × 60% = 90 晶体 |

> **注意**：补给箱只含模块，所有内容都可以回收

### 3.4 决策指引

| 情况 | 建议操作 | 原因 |
|------|----------|------|
| **需要该模块** | 获取 | 直接使用 |
| **模块不适合当前Build** | 回收 | 换取晶体购买更合适的 |
| **模块栏位已满** | 回收 | 无法获取更多模块 |
| **有高回收效率属性** | 回收 | 能获得更多晶体 |
| **晶体紧缺** | 回收 | 优先保证商店购买力 |
| **模块稀有度高** | 慎重考虑 | T3/T4模块难以在商店遇到 |

### 3.5 特殊规则

| 规则 | 说明 |
|------|------|
| **内容预决定** | 补给箱内容在掉落时就已决定，不是开箱时随机 |
| **无法跳过** | 必须处理完所有补给箱才能进入商店 |
| **超时处理** | 如果60秒内无操作，自动"全部获取" |
| **无法撤销** | 一旦选择获取或回收，无法撤销 |

## 六、生成规则

### 6.1 生成时机

**与Brotato一致**：每10秒检查生成，数量受道具影响

```
基础生成：每10秒检查，生成数量 = 0.5 + 0.33 × 残骸相关道具数
```

### 6.2 每波残骸数量（简化设计）

| 阶段范围 | 平均数量 | 说明 |
|----------|----------|------|
| 全波次 | 3-5 个 | 保持稳定，与Brotato一致 |

### 6.3 生成位置

```
残骸生成规则：
1. 距离玩家当前位置 > 5 米（避免刷在脚下）
2. 距离其他残骸 > 2 米（避免堆叠）
3. 在竞技场范围内随机分布
```

## 八、经济影响分析

### 8.1 残骸收入预估

**假设**：每波平均摧毁 4 个残骸

| 来源 | 每波收入 | 20波总计 |
|------|----------|----------|
| 直接掉落晶体（平均5晶体×4） | 20 晶体 | 400 晶体 |
| 补给箱回收（20%×4×平均15晶体） | 12 晶体 | 240 晶体 |
| **残骸总收入** | **~32 晶体** | **~640 晶体** |

> **注意**：补给箱只含模块，如果选择获取模块则不计入晶体收入

### 8.2 与敌舰收入对比

| 收入来源 | 20波总收入 | 占比 |
|----------|------------|------|
| 敌舰击杀 | ~900 晶体 | ~58% |
| **漂浮残骸** | ~640 晶体 | **~42%** |
| 其他（波次奖励等） | ~40 晶体 | ~3% |

**结论**：残骸系统提供约 **40%** 的额外收入，是重要的资源来源。

### 8.3 护盾恢复价值

| 来源 | 每波恢复 | 20波总计 |
|------|----------|----------|
| 护盾电池（80%×4×8护盾） | 25.6 护盾 | 512 护盾 |

> 相当于额外的生存保障，鼓励玩家主动清理残骸

## 九、战略意义

### 9.1 风险-收益决策

```
残骸位置在战场边缘 → 需要冒险前往 → 获得护盾+晶体
残骸位置在敌群中 → 攻击残骸可能吸引注意 → 需要精准走位
残骸位置安全 → 优先摧毁 → 获得恢复和资源
```

### 9.2 与生存的关系

- **能量核心恢复护盾**：给低血量玩家续命机会
- **需要主动拾取**：创造"冒险拾取还是安全撤退"的决策
- **补给箱可能掉道具**：增加随机性和惊喜感
