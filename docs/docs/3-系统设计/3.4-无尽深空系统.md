## 二、核心公式：深空因子（Endless Factor）

### 2.1 计算公式（与Brotato完全一致）

```lua
function CalculateEndlessFactor(current_wave)
    -- 从21阶段开始计算
    local endless_wave = math.max(0, current_wave - 20)
    
    -- 36阶段后乘数增加
    local endless_mult = 2.0 + math.max(0, (current_wave - 35) * 0.2)
    
    -- 最终因子
    local endless_factor = math.max(0, 
        ((endless_wave * (endless_wave + 1)) / 2) / 100
    ) * endless_mult
    
    return endless_factor
end
```

## 五、经济系统变化

### 5.1 模块/武器价格

```
无尽模式物品价格 = 普通价格 × (1 + Factor × 系数)
```

| 阶段 | 价格增加 |
|------|----------|
| 30 | +22% |
| 40 | +126% |
| 50 | +465% |

### 5.2 刷新成本

```lua
刷新成本 = max(1, 上次刷新成本 + max(1, 0.5 × 阶段 × sqrt(1 + Factor)))
```

刷新成本增长极快，后期几乎无法刷新。

### 5.3 回收价值

**关键**：回收价值**不受**深空因子影响！

| 来源 | 回收比例 | 无尽模式 |
|------|----------|----------|
| 默认 | 25% | 25%（不变） |
| 回收模块 | 60% | 60%（不变） |
| 企业家战舰 | 85% | 85%（不变） |

这意味着后期回收价值远低于购买价格。

## 六、掉落机制变化

### 6.1 晶体掉落

```
晶体掉落率 = max(50%, 100% - (阶段 - 20) × 1.5%)
```

| 阶段 | 掉落率 |
|------|--------|
| 20 | 100% |
| 27 | 89.5% |
| 30 | 85% |
| 34+ | **50% (下限)** |

### 6.2 消耗品掉落

```
消耗品掉落率 = 基础率 × (1 + 运势/100) / (1 + Factor)
```

| 阶段 | Factor | 掉落率（相对） |
|------|--------|---------------|
| 20 | 0 | 100% |
| 30 | 1.10 | 47.6% |
| 40 | 6.30 | **13.7%** |
| 50 | 23.25 | 4.1% |

**例外**：走私船、精英、Boss 仍保证掉落消耗品。

### 6.3 传奇补给箱

- 20阶段后精英和Boss**不再保证**掉落传奇补给箱
- 掉落率受深空因子削弱

## 七、战斗机制变化

### 7.1 无敌帧

```
实际无敌帧 = 基础无敌帧 / max(1, Factor)
```

| 阶段 | Factor | 无敌帧比例 |
|------|--------|------------|
| 20 | 0 | 100% |
| 30 | 1.10 | 90.9% |
| 40 | 6.30 | **15.9%** |
| 50 | 23.25 | 4.3% |

### 7.2 百分比护盾伤害削弱

**从第39阶段开始**，百分比伤害效果被削弱：

```
实际百分比伤害 = floor(当前敌人护盾 × 伤害百分比 / max(1, Factor × 0.2))
```

**受影响武器/模块**：
- 等离子切割（类似Chainsaw）
- 重力冲击（类似Giant Belt）
- 星火炮（类似Greek Fire）

| 阶段 | Factor | 有效率 |
|------|--------|--------|
| 39 | 5.49 | 91% |
| 40 | 6.30 | **79%** |
| 50 | 23.25 | 22% |
| 100 | 485.40 | **1%** |

## 九、策略建议

### 9.1 构筑重点

| 阶段 | 重点 |
|------|------|
| 1-20阶段 | 普通构筑 + 为无尽准备 |
| 21-35阶段 | 提升固定伤害输出 |
| 36-50阶段 | 生存为主，放弃百分比伤害 |
| 50阶段+ | 极限生存，依赖无敌/规避 |

### 9.2 避免的陷阱

1. **不要过度投资资源回收** - 20阶段后反向增长
2. **不要依赖存钱罐模块** - 20阶段后失效
3. **不要依赖百分比伤害** - 39阶段后急剧削弱
4. **不要频繁刷新商店** - 成本增长极快
