## 十、实现建议

### 10.1 进度追踪系统设计

```lua
-- 累计统计（跨局保存）
累计统计 = {
    total_kills = 0,           -- 累计击杀数
    total_materials = 0,       -- 累计收集晶体
    danger_wins = {0,0,0,0,0,0}, -- 各危险等级通关记录
    ship_wins = {},            -- 各战舰通关记录（48个）
}

-- 单局统计（每局重置）
单局统计 = {
    current_materials = 0,     -- 当前持有晶体
    wave_healing = 0,          -- 本阶段治疗量
    wave_burn_kills = 0,       -- 本阶段烧死敌舰数
    consumables_collected = 0, -- 本局消耗品数
    weapons_recycled = 0,      -- 本局回收武器数
    turret_count = 0,          -- 当前召唤物数量
    unique_weapons = 0,        -- 不同武器种类数
    t4_weapons = 0,            -- T4武器数量
    t1_items = 0,              -- T1模块种类数
}
```

### 10.2 条件检测时机

| 检测时机 | 相关挑战 |
|----------|----------|
| 属性变化时 | 丰收、幻影、疾速、缓慢、鹰眼、坚韧、濒死、吸血 |
| 击杀时 | 幸存者系列、伐木、烟花、屠龙 |
| 晶体收集时 | 收集者系列、囤积、满血收集 |
| 阶段结束时 | 医疗、森林、鲁莽 |
| 进入商店时 | 回收 |
| 游戏结束时 | 危险等级系列、战舰胜利、不锁商店 |
| 消耗品拾取时 | 饥饿 |
| 召唤物放置时 | 建造、高科技、魔械 |
| 装备武器时 | 实验、贵族 |
