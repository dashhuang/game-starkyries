## 三、反馈系统设计 ⭐（对标Brotato核心设计）

> 清晰的反馈是优秀游戏体验的基础。参考Brotato的设计理念，我们需要在视觉、数值、进度三个维度提供即时反馈。

### 3.0 反馈设计总览

| 反馈类型 | 内容 | 设计目标 |
|----------|------|----------|
| **视觉反馈** | 伤害数字、击中特效、爆炸效果 | 让玩家"感受"到伤害 |
| **数值反馈** | 属性面板、DPS显示、变化预览 | 让玩家"理解"自己的强度 |
| **进度反馈** | 阶段进度、经验条、晶体数 | 让玩家"知道"当前状态 |

### 3.1 伤害数字系统 ⭐

伤害数字是最重要的战斗反馈，需要精心设计：

| 伤害类型 | 字体大小 | 颜色 | 特效 |
|----------|----------|------|------|
| **普通伤害** | 标准 | 白色 #FFFFFF | 无 |
| **暴击伤害** | ×1.5倍 | 金黄 #FFD700 | 放大弹跳 + 光芒 |
| **元素伤害** | 标准 | 对应元素色 | 对应元素粒子 |
| **DOT伤害** | ×0.8倍 | 橙色 #FF8800 | 持续飘出 |
| **护盾恢复** | 标准 | 绿色 #00FF88 | 向上飘动 |
| **受到伤害** | ×1.2倍 | 红色 #FF3B3B | 屏幕边缘闪红 |

**伤害数字动画**：
```
出现 → 放大(0.1秒) → 保持(0.3秒) → 向上飘动+淡出(0.5秒)
暴击额外：震动(0.05秒) + 光芒扩散
```

### 3.2 击中反馈系统

| 反馈元素 | 描述 | 持续时间 |
|----------|------|----------|
| **命中火花** | 子弹击中时的闪光球，颜色跟随武器类型 | 0.1秒 |
| **敌舰闪白** | 受击时全身变白 | 0.08秒 |
| **击退效果** | 部分武器击中后敌舰后退 | 取决于武器 |
| **屏幕震动** | 暴击/大伤害时轻微震动 | 0.05秒 |
| **顿帧效果** | 暴击时画面轻微暂停 | 0.02秒 |

### 3.3 击毁反馈系统

| 反馈元素 | 描述 |
|----------|------|
| **爆炸效果** | 圆形扩散光环 + 中心闪光 |
| **碎片飞溅** | 8-30个碎片粒子向外飞散，带重力下落 |
| **击毁音效** | 根据敌舰大小不同音效 |
| **掉落物弹出** | 晶体/修复包从爆炸中心弹出 |

### 3.4 属性变化反馈

| 场景 | 反馈方式 |
|------|----------|
| **舰桥升级** | 选择属性后，数值跳动+光芒 |
| **补给站预览** | 鼠标悬停时显示"当前值→变化后值" |
| **装备武器/模块** | 属性面板对应数值高亮闪烁 |
| **Buff激活** | 战舰周围显示对应Buff图标 |

### 3.5 进度反馈系统

| 元素 | 位置 | 反馈方式 |
|------|------|----------|
| **护盾条** | 顶部左 | 实时变化，低于30%变红 |
| **经验条** | 护盾下方 | 平滑增长，升级时闪光 |
| **阶段进度** | 顶部中 | 显示"阶段X/20"，倒计时 |
| **晶体数量** | 右下 | 获取时数字跳动+绿色 |
| **升级提示** | 中央 | 升级时金色光柱 |

## 四、特效设计

### 4.1 战斗特效

| 特效 | 描述 | 触发 |
|------|------|------|
| **激光轨迹** | 发光的线条，快速消散 | 激光武器攻击 |
| **导弹尾焰** | 橙红色拖尾 | 导弹发射 |
| **爆炸效果** | 圆形扩散，碎片飞溅 | 敌舰被击毁 |
| **护盾闪烁** | 蓝色六边形网格 | 受到伤害 |
| **暴击特效** | 放大数字+光芒+顿帧 | 精确打击 |
| **升级光环** | 向上的金色光柱 | 舰桥升级 |
| **拾取吸附** | 物品飞向战舰 | 拾取掉落 |
| **连杀特效** | 屏幕边缘金光 | 连续击毁5+敌舰 |

### 4.2 UI特效

| 特效 | 描述 | 触发 |
|------|------|------|
| **按钮发光** | 边缘呼吸光效 | 可交互按钮 |
| **面板滑入** | 从边缘滑入+淡入 | 界面切换 |
| **数字跳动** | 数字变化时弹跳 | 数值增减 |
| **稀有光效** | 彩色扫光 | 高品质物品 |
| **解锁绽放** | 光芒绽放 | 解锁新内容 |
| **属性变化** | 数值高亮+箭头指示 | 装备物品时 |

### 4.3 屏幕反馈

| 事件 | 视觉反馈 | 音效反馈 |
|------|----------|----------|
| **受到伤害** | 屏幕边缘红色闪烁 | 警告音 |
| **护盾耗尽** | 屏幕变红+震动 | 紧急警报 |
| **Boss出现** | 屏幕震动+警告文字 | Boss主题曲 |
| **暴击** | 屏幕轻微顿帧 | 暴击音效 |
| **通关** | 胜利特效+舰长语音 | 胜利音乐 |
| **升级** | 金色光柱+经验条闪光 | 升级音效 |
| **获得晶体** | 晶体数字跳动 | 拾取音效 |

### 4.4 击中反馈特效

| 特效 | 描述 | 参数 |
|------|------|------|
| **命中火花** | 子弹击中时的闪光球 | 颜色跟随武器，暴击更大 |
| **敌人闪白** | 受击时全身变白 | 持续0.08秒，intensity 3.0 |
| **玩家闪白** | 玩家战舰受击时闪白 | 持续0.08秒，intensity 0.8（战舰较大，降低强度避免过亮） |
| **击毁粒子** | 碎片向外飞散 | 8-30个，带重力 |
| **暴击震动** | 屏幕轻微震动 | 持续0.05秒 |
| **顿帧效果** | 画面短暂暂停 | 持续0.02秒 |

### 4.5 动态光照系统

| 光源 | 描述 | 参数 |
|------|------|------|
| **子弹光源** | 子弹携带点光源，照亮周围战舰 | 颜色跟随武器，亮度0.8（暴击1.5），范围4.0m（暴击6.0m） |
| **爆炸光源** | 爆炸时产生点光源并淡出 | 亮度3.0渐变至0，范围=爆炸尺寸×3 |
| **引擎光源** | 玩家战舰引擎火焰点光源 | 颜色为引擎色（蓝），亮度随移动0-2.0，范围4.0m |
| **场景主光** | 太空环境主光源 | 方向光，亮度0.8，带补光0.2 |
| **环境光** | Zone组件提供环境光 | ambientColor (0.08, 0.10, 0.15) |
