# 星河战姬 世界观与剧情框架

> 本文档记录《星河战姬》的世界观背景、势力设定和叙事框架。

---

## 一、世界观概述

### 1.1 核心概念

**流浪舰队的归途之旅**

玩家指挥的是一支流亡中的舰队，搭载着一艘母舰不断进行空间跳跃，在星际间流浪。舰队的终极目标是找到传说中的"创世遗迹"——那其实是人类失落已久的故乡：**地球**。

### 1.2 叙事特点

| 原则 | 说明 |
|------|------|
| **即时战斗** | 游戏一开始直接进入战斗，无大段对话 |
| **侧面叙事** | 剧情通过碎片、语音、环境细节逐步展开 |
| **探索揭示** | 世界观真相随游戏推进逐渐拼凑 |

### 1.3 游戏机制的叙事包装

| 游戏机制 | 世界观解释 |
|---------|-----------|
| **每局战斗** | 派出战舰为母舰探索安全跳跃坐标 |
| **20波敌人** | 在跳跃途中遭遇的敌人，每波=新的空间跳跃 |
| **波次倒计时** | 计算下一个跳跃点的时间 |
| **难度等级** | 距离起点越远（越深入未知宇宙）越危险 |
| **解锁新舰长** | 探索中救援/招募的幸存者、新伙伴 |
| **解锁武器模块** | 打捞的遗迹科技、贸易获取的新资源 |
| **重复刷关** | 舰队日常的生存探索任务 |

## 二、历史时间轴

### 【古纪元】- 约500年前

```
├─ 地球时代末期，一名世界首富的科技公司作为先驱
│  带领股东和支持者移民火星
│
├─ "母星联邦" vs "火星自治领" 爆发冲突
│
├─ 首富在支持者怂恿下称帝，建立"自由帝国"
│  （帝国宣称为人民带来了"绝对的自由"）
│
├─ 百年战争期间，军用AI权限不断扩大
│  最终集体叛变（机械叛军起源）
│
├─ 混战惊醒了沉睡的外星文明"原生者"（虫族起源）
│
└─ 大崩塌：地球、火星失联，人类文明碎片化
```

### 【流散纪元】- 约100年

```
├─ 自由帝国从火星一路逃离、一路扩张（自称人类正统）
│
├─ 幸存者在银河各处建立殖民地
│
├─ 独立势力、海盗、佣兵团兴起
│
└─ "创世遗迹"的传说开始流传
   ——据说那里藏着人类起源的秘密
```

### 【现在】

```
├─ 自由帝国已是银河最大人类势力，但仍在流亡扩张中
│
├─ 玩家舰队：帝国某支舰队的残部，因某事件被迫叛逃
│
└─ 目标：寻找"创世遗迹"
   （玩家初期不知道，遗迹其实就是地球）
```

## 三、自由帝国（黑色幽默设定）

### 3.1 帝国特色

**"绝对的自由"**

自由帝国的公民坚信，正是伟大的创始皇帝为他们带来了绝对的自由。帝国的一切都以"自由"为名：

- 自由舰队（军队）
- 自由议会（橡皮图章）
- 自由教育（洗脑体系）
- 自由监察（互相举报系统）

### 3.2 皇帝崇拜

| 特点 | 描述 |
|------|------|
| **神秘化** | 皇帝现在在哪里已成传说，没人见过本人 |
| **网络指示** | 只有定期通过网络才能收到"皇帝的指示" |
| **绝对崇拜** | 帝国民众自发对皇帝无限崇拜 |
| **互相监督** | 只要有人"崇拜不绝对"就会被举报、逐出帝国 |

### 3.3 帝国讽刺点

- 以"自由"为名的极权统治
- 可能已死500年的皇帝仍在"发号施令"
- AI生成的皇帝指示？（暗线伏笔）
- 民众真心相信自己很自由

## 四、势力设定

### 4.1 自由帝国（追捕者）

| 属性 | 描述 |
|------|------|
| **起源** | 火星自治领 → 自由帝国 |
| **性质** | 人类最大势力，自称正统 |
| **与舰队关系** | 舰队的"娘家"，现在是追捕者 |
| **出现区域** | 帝国控制区、主线剧情 |

**特点**：
- 舰船设计统一、制式化
- 战术死板但数量庞大
- 士兵狂热但缺乏独立思考

### 4.2 机械叛军

| 属性 | 描述 |
|------|------|
| **起源** | 百年战争中叛变的军用AI |
| **性质** | 失控的机械文明 |
| **目标** | 消灭"威胁"（有机生命） |
| **出现区域** | 遗迹星域、废弃空间站 |

**特点**：
- 原帝国制式舰船（被接管）
- 精确的协调作战
- 无人操作，高效冷酷
- 与帝国的AI皇帝有何关联？（伏笔）

### 4.3 原生者（虫族）

| 属性 | 描述 |
|------|------|
| **起源** | 被百年战争惊醒的外星文明 |
| **性质** | 宇宙深空的原生恐怖 |
| **目标** | 吞噬一切有机和无机物质 |
| **出现区域** | 未知星域、深空探索 |

**特点**：
- "舰船"是活的有机体
- 拥有自我修复能力
- 依靠数量和毒素作战
- 无法沟通，纯粹的掠食者

### 4.4 独立势力/海盗

| 属性 | 描述 |
|------|------|
| **起源** | 各种殖民地幸存者、流民、逃兵 |
| **性质** | 松散的独立势力联盟 |
| **与舰队关系** | 有敌有友，可能的盟友 |
| **出现区域** | 常见星域、低难度区 |

**特点**：
- 改装的民用和军用舰船
- 机动性强，战术灵活
- 有自己的道德准则（或没有）
- 部分是帝国的叛逃者

## 五、舰队设定

### 5.1 母舰与指挥官

**母舰和指挥官（玩家）是神秘的存在。**

- 母舰的来历不明，拥有远超帝国的技术
- 指挥官身份成谜，从不露面、从不说话
- 舰长们只知道：指挥官收留了她们，给了她们容身之所
- 母舰在星际间流浪，不断收留各种背景的女舰长

**母舰承载着**：
- 舰队全体成员的日常生活
- 超越时代的跳跃引擎和导航系统
- 维修、补给设施
- 一个又一个被收留的故事

### 5.1.1 指挥官（玩家）

**沉默的存在，情感的中心。**

| 特点 | 说明 |
|------|------|
| **不露面** | 没有立绘、没有形象设定 |
| **不说话** | 对话中没有台词，由舰长们单方面说话 |
| **被倾诉** | 舰长们会主动和指挥官聊天、倾诉、撒娇 |
| **情感羁绊** | 与舰长们建立好感度、产生暧昧关系 |
| **被信任** | 舰长们信任指挥官，愿意为其战斗 |

**叙事作用**：
- 玩家代入的载体
- 舰长们情感投射的对象
- 所有角色关系的交汇点

**类似定位参考**：《明日方舟》博士、《碧蓝航线》指挥官

### 5.2 舰长来源

**舰长们来自不同阵营，各有各的故事**：

| 原阵营 | 来源说明 |
|-------|---------|
| **自由帝国** | 因各种荒诞理由被迫叛逃 |
| **机械叛军** | 觉醒了自我意识的AI？ |
| **独立势力** | 被母舰救援的幸存者 |
| **海盗** | 金盆洗手想改邪归正？ |
| **原生者** | ？？？（如果有的话） |

### 5.3 帝国舰长的叛逃理由（黑色幽默）

来自帝国的舰长，每人可能有不同的荒诞叛逃理由：

| 叛逃类型 | 示例 |
|---------|------|
| **表情包事件** | 发了皇帝的表情包，被判"亵渎圣像" |
| **算法预判** | AI判定她"未来可能崇拜不绝对" |
| **皇恩抽奖** | 中了"面见皇帝"的"大奖"（其实是处决） |
| **五星好评** | 只给皇帝打了4.9星，被判不忠 |
| **连坐受害** | 朋友/家人出事，被牵连 |
| **知道太多** | 发现了某些不该知道的真相 |

*具体分配到哪个舰长身上，待角色详细设计时确定。*

### 5.4 战舰出击

每局战斗，玩家派出一艘战舰进行探索任务：
- 为母舰寻找安全的跳跃坐标
- 清除航路上的威胁
- 搜集资源、救援幸存者

## 六、创世遗迹（核心谜团）

### 6.1 表面传说

在银河系的某处，存在一处神秘的古老遗迹。传说这处遗迹藏着人类起源的秘密，蕴含着改变命运的力量。

- 帝国用它作为统治工具："找到遗迹，就能复兴人类"
- 各势力都在寻找它
- 舰队在逃亡中意外获得了相关线索

### 6.2 真相

**创世遗迹就是地球。**

500年过去，地球已经变成了神话。没人记得它的真实坐标，甚至不确定它是否真的存在。舰队的旅途，其实是人类寻找故乡的归途。

### 6.3 真相揭示节奏

```
初期：玩家只知道在寻找"传说中的遗迹"
  ↓
中期：发现遗迹线索指向某个特定星域
  ↓
后期：逐渐拼凑出"遗迹就是母星"的真相
  ↓
终章：抵达地球——但地球现在是什么样？
```

## 八、叙事原则

### 8.1 游戏体验优先

- 不打断战斗节奏
- 剧情不强制观看
- 故事是奖励，不是门槛

### 8.2 碎片化揭示

- 远古历史通过碎片逐步揭露
- 不直接点明讽刺对象（马斯克、SpaceX）
- 让玩家自己拼凑、发现真相

### 8.3 日常与史诗并存

- **日常轻松**：母舰上的角色互动、日常对话
- **战斗热血**：出击时的紧张感、同伴配合
- **黑暗底色**：帝国的荒诞、宇宙的残酷

## 十、游戏开头与角色解锁

### 10.1 新手流程

**设计原则**：不解释、直接行动、在失败中学习

#### 第一次进入游戏

```
场景：母舰危机时刻

[只有一艘战舰可选：先锋号 - 舰长星遥]

星遥：「指挥官，没时间了！我们要立即出发寻找新的落脚点！」

→ 直接进入战斗（无装备选择，默认配置）
→ 玩家通过战斗学习基础操作
```

#### 第一次失败

```
场景：战舰返回母舰

星遥：「果然还是不行呢...没事，零会帮忙修复的。
       我们还有时间，重新调整装备再出发吧。」

[零登场]

零：「战损分析完毕。主推进器损伤17%，已开始修复。
     预计修复时间：3分钟。建议利用这段时间优化装备配置。」

→ 认识零（机械师/维修角色）
→ 零加入对话
→ 解锁零（赛博号）
→ 解锁装备系统
```

### 10.2 角色解锁设计

| 解锁时机 | 角色 | 解锁方式 |
|---------|------|---------|
| 初始 | 星遥 | 游戏开始即有 |
| 第一次失败 | 零 | 剧情解锁（修复战舰） |
| 后续 | 其他角色 | 探索救援 / 剧情触发 / 成就解锁 |

### 10.3 开头的叙事信息量

**玩家第一局能了解到的**：
- 我们是一支逃亡中的舰队
- 需要不断寻找安全的跳跃点
- 星遥是可靠的伙伴
- 零负责维修工作

**玩家不知道的（后续揭示）**：
- 为什么在逃亡
- 从哪里逃出来
- 要去哪里
- 舰队的过去
