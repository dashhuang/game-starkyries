## 三、外形与气质

### 外形要素

| 要素 | 描述 |
|------|------|
| **年龄感** | 22-24岁，知性温婉 |
| **发型** | 淡紫色长发，编成精致的辫子 |
| **发色** | 淡紫色，如同水晶 |
| **瞳色** | 淡蓝色，清澈如水 |
| **身材** | 纤细娇小，学者体型 |
| **服装** | 白紫相间的研究员服装配护盾装置 |
| **气质** | 温婉知性，如玉般温润 |
| **特征物品** | 手腕的护盾发生器、眼镜 |

### 画面描述

二十三四岁的年轻学者，浑身散发着书卷气息。淡紫色的长发编成精致的辫子垂在胸前，发丝在光线下像是透明的水晶。淡蓝色的眼眸清澈见底，透过眼镜看向世界时带着求知的热情。

身穿白紫相间的研究员服装，既优雅又实用。腰间和手腕上佩戴着她自己设计的护盾发生器——那是她最自豪的发明。身材纤细娇小，是典型的学者体型。

她的动作总是轻柔优雅，说话时声音也轻轻的，但讲到研究时会不自觉地激动起来。整个人如同一块温润的玉石，内敛、坚韧、价值非凡。

## 四、性格详解

### 表面性格
- 温婉内敛，说话轻声细语
- 聪慧好学，对知识有渴望
- 有点害羞，不善交际
- 做事认真，追求完美

### 内在性格
- 对自己的研究有执着的热爱
- 内心坚定，不会轻易妥协
- 被窃取研究后有阴影
- 渴望自己的成果被正确使用

### 成长弧线
```
初期：专注研究，不太与人交流
  ↓
中期：开始融入团队，学会分享
  ↓
后期：用研究保护同伴
  ↓
终章：理解科学的意义是守护
```

## 五、来历与背景

### 过去
帝国科学院的研究员，专攻护盾技术。她研发的晶华护盾系统革新了帝国的防御技术，本应获得极高的荣誉。然而，她的研究被上司窃取署名，她本人被降级为助理。

### 叛逃经过
当她发现自己的护盾技术被用于"净化"殖民地（屠杀平民）时，她彻底崩溃了。她销毁了所有研究数据，带走了唯一的原型机逃跑。

**叛逃理由**（良心型）：
> "我研发护盾是为了保护人，不是为了杀人。那些数据...我宁愿它们消失。"

### 被收留
逃跑途中被舰队救起。她的护盾技术让舰队的防御能力大幅提升，而舰队给了她一个"正确使用科技"的地方。

## 十、设计备注

**设计意图**：
- 温婉学者类型
- "晶华"代表水晶般的纯净
- 被窃取研究的受害者
- 科技应该保护人的主题

**参考来源**：
- 研究员类角色
- 护盾/防御意象
- 温婉知性女性

**设计要点**：
- 立绘体现知性和温婉
- 白紫蓝配色为主
- 眼镜是重要标识
- 护盾发生器要有设计感
