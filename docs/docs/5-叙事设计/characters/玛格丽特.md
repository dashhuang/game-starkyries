### 画面描述

二十八九岁的女军官，整个人如同一座钢铁堡垒。深铁灰色的短发干练利落，没有一丝多余。钢蓝色的眼眸坚定得像是能穿透一切谎言，让人不敢直视。

身穿深灰色军法官制服，笔挺得如同刀削。胸口别着天平造型的胸针——那是军事法庭的标志，也是她信仰的象征。身上还有几枚勋章，见证着她曾经的荣耀。

身材高大健壮，是标准的军人体魄，但每一个动作都带着法官特有的庄重。口袋里装着父亲留下的怀表——她的父亲也是一位法官，教会她"正义"二字的分量。

她很少笑，但她的眼神比任何微笑都温暖——如果你是无辜的话。

## 四、性格详解

### 表面性格
- 刚毅坚定，从不妥协
- 正直不阿，绝不徇私
- 寡言少语，但句句在理
- 原则性强，黑白分明

### 内在性格
- 内心柔软，对无辜者有保护欲
- 曾经因为遵守命令而做错事
- 对"真正的正义"有自己的理解
- 渴望找到值得守护的东西

### 成长弧线
```
初期：坚持自己的原则，显得死板
  ↓
中期：学会原则也需要灵活运用
  ↓
后期：找到"正义"的新定义
  ↓
终章：从"帝国的法官"变成"正义的守护者"
```

## 五、来历与背景

### 过去
帝国军事法庭的法官，以铁面无私著称，被称为"铁心法官"。她判决的案件从不翻案，因为她会彻查每一个细节。然而，她渐渐发现很多案件根本不允许她彻查——结论早已内定。

### 叛逃经过
某次审判中，她被要求做伪证陷害一位无辜的同僚——那是她的老师，一位正直的将军。上面已经内定了有罪判决，只需要她配合演戏。她拒绝了，并公开揭露了真相。

**叛逃理由**（良心型）：
> "法律是用来守护正义的，不是权贵的玩具。如果法庭不能公正，那我宁愿背叛法庭。"

### 被收留
被帝国追杀时遇到舰队。在确认舰队"是正义的一方"后，她选择加入，用自己的方式继续守护正义。

## 六、与指挥官的关系

**关系定位**：正义的守护者、刚柔并济的伴侣

### 关系发展

| 阶段 | 好感度 | 互动特点 |
|------|--------|----------|
| **初识** | Lv.1 | 审视观察，判断是否值得追随 |
| **熟悉** | Lv.2 | 认可指挥官的正义，开始信任 |
| **信任** | Lv.3 | 讲述法庭的黑暗和自己的挣扎 |
| **暧昧** | Lv.4 | 钢铁外表下开始流露柔软 |
| **誓约** | Lv.5 | "我的天平，从此为你倾斜。" |

### 特殊互动
- 会帮舰队处理纠纷
- 判断事情总是公正客观
- 难得露出柔软时会有点不好意思

## 七、关键台词

### 战斗语音
- （开始）「穿甲号，执行正义。」
- （攻击）「判决。」
- （受伤）「...不影响判断。」
- （暴击）「无处遁形。」
- （波次结束）「审判完毕。」
- （胜利）「正义...终将胜利。」
- （失败）「...是我的错误。」

### 日常语音（好感度相关）
- （Lv.1）「你就是指挥官。我在观察你。」
- （Lv.2）「我认可你的行事风格。请继续保持。」
- （Lv.3）「那个时候...我以为自己在守护正义。谢谢你愿意听。」
- （Lv.4）「和你在一起时...我好像不用那么紧绷了。」
- （Lv.5）「我不再是帝国的法官。我是你的...你的人。」

### 特殊语音
- （长时间未登录）「你去哪了？...我等了你很久。」
- （生日）「这是父亲的怀表。现在...它是你的了。」
