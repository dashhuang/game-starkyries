## 三、外形与气质

### 外形要素

| 要素 | 描述 |
|------|------|
| **年龄感** | 外表25岁左右，实际年龄成谜 |
| **发型** | 波浪长卷发，及腰 |
| **发色** | 银白渐变淡紫，如月光 |
| **瞳色** | 异色瞳：左眼金色，右眼紫色 |
| **身材** | 高挑妖娆，曲线玲珑 |
| **服装** | 黑紫色华丽礼服，露背设计，暗金纹饰 |
| **气质** | 妖艳魅惑，危险又迷人 |
| **特征物品** | 颈间的红宝石项链（据说是诅咒之物） |

### 画面描述

像是从禁忌童话中走出的堕天使。银白渐变淡紫的波浪长发倾泻而下，在腰间轻轻摇曳。最引人注目的是那双异色瞳——左眼流金，右眼曜紫，仿佛能看穿一切谎言与伪装。
身着黑紫色华丽礼服，大胆的露背设计露出光洁的脊背，暗金纹饰在烛光下若隐若现。她的一举一动都带着贵族的优雅，却又处处透着危险的气息。
颈间那枚红宝石项链是她唯一不离身的物品，据说是被诅咒之物，也有人说那是她自己施下的诅咒。
她总是带着似笑非笑的表情，让人猜不透她在想什么——而这正是她想要的。

## 四、性格详解

### 表面性格
- 玩世不恭，喜欢用话语撩拨他人
- 看透一切却装作漫不经心
- 享乐主义，追求刺激

### 内在性格
- 极度聪慧，洞察力惊人
- 对虚伪和欺骗深恶痛绝
- 有自己坚守的底线和正义

### 成长弧线
```
初期：以旁观者姿态审视一切，随时准备抽身
  ↓
中期：发现指挥官与众不同，产生兴趣
  ↓
后期：逐渐卸下伪装，展露真实的自己
  ↓
终章：找到值得认真对待的人和事
```

## 五、来历与背景

### 过去
维尔家族是帝国的古老贵族，莉莉丝从小在尔虞我诈中长大，练就了看穿人心的本领。她厌恶贵族的虚伪，却不得不戴着面具周旋其中。

### 叛逃经过
某次偶然，她截获了皇帝的一道"圣谕"，却发现这道关乎数十万人命运的命令——竟是由AI自动生成的。皇帝根本没有看过。她将此事公开，随即遭到全面追杀。

**叛逃理由**（良心型 + 荒诞型）：
> "我不在乎谁当皇帝，但用AI来决定人的生死？这个笑话一点都不好笑。"

### 被收留
逃亡途中被多方势力追杀，包括夜凛。在一次遭遇战中被指挥官的舰队意外搭救，从此留下。

## 六、与指挥官的关系

**关系定位**：从玩物到挚爱，唯一让她认真的人

### 关系发展

| 阶段 | 好感度 | 互动特点 |
|------|--------|----------|
| **初识** | Lv.1 | 把指挥官当有趣的玩具，不断试探 |
| **熟悉** | Lv.2 | 发现指挥官不上当，兴趣大增 |
| **信任** | Lv.3 | 偶尔露出真实的一面，但很快收起 |
| **暧昧** | Lv.4 | 撩拨变得认真，开始吃醋 |
| **誓约** | Lv.5 | "在你面前，我不需要演戏。" |

### 特殊互动
- 喜欢突然凑近说些暧昧的话
- 被认真对待时反而会害羞（但绝不承认）
- 会用"恶魔契约"开玩笑
