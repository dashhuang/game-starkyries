## 三、外形与气质

### 外形要素

| 要素 | 描述 |
|------|------|
| **年龄感** | 外表20岁左右（真实年龄不明） |
| **发型** | 深蓝色长发，波浪状自然飘动 |
| **发色** | 深蓝渐变浅蓝，像深海渐变 |
| **瞳色** | 海蓝色，深不见底 |
| **身材** | 修长纤细，有种飘渺感 |
| **服装** | 深蓝色流线型连体服，带有发光纹路 |
| **气质** | 神秘空灵，不属于这个世界 |
| **特征物品** | 颈间的发光宝石、手腕的奇异符文 |

### 画面描述

外表二十岁左右的女子——但没人知道她真正的年龄。深蓝色的长发如同海浪般自然起伏，从深蓝渐变到浅蓝，在光线下会微微发光。海蓝色的眼眸深不见底，像是能看到另一个世界。

身穿深蓝色流线型连体服，材质看起来不像是已知的任何织物。服装上有淡淡的发光纹路，在黑暗中会显现出奇异的图案。颈间挂着一颗蓝色宝石，据说是在深空中找到的，能发出幽幽的光。

手腕上有一圈奇异的符文——她说那是"家乡的文字"，但没人能读懂。整个人给人一种飘渺的感觉，仿佛随时会消失在虚空中。

她的存在本身就是一个谜。

## 四、性格详解

### 表面性格
- 神秘莫测，说话常常让人听不懂
- 温柔平静，很少有情绪波动
- 对一切充满好奇，像是在观察异世界
- 偶尔会说出预言般的话

### 内在性格
- 孤独，因为没有同类
- 渴望理解和被理解
- 对人类世界充满眷恋
- 在寻找什么，但自己也说不清楚

### 成长弧线
```
初期：像观察者一样旁观一切
  ↓
中期：开始融入人类的生活
  ↓
后期：找到了"家"的感觉
  ↓
终章：决定留在这个世界
```

## 五、来历与背景

### 过去
她自己也不太清楚。只记得从"很深很深的地方"醒来，然后漂流到了这片星域。她的知识和技术远超现有文明，却对日常生活一无所知。

### 被帝国追杀
帝国发现她后试图捕获研究，她被迫逃亡。她不理解为什么会被追杀——在她的认知里，不同种族应该和平共处。

**来历**（神秘）：
> "我从深渊来，那里很暗，很静。我在找...什么呢？我也不记得了。"

### 被收留
被舰队救起时几乎奄奄一息。苏醒后对这个"有趣的群体"产生了好奇，决定暂时留下观察。慢慢地，"观察"变成了"参与"。

## 六、与指挥官的关系

**关系定位**：被人类世界吸引的异客、找到归属的流浪者

### 关系发展

| 阶段 | 好感度 | 互动特点 |
|------|--------|----------|
| **初识** | Lv.1 | 好奇观察，问奇怪的问题 |
| **熟悉** | Lv.2 | 开始学习人类的习俗 |
| **信任** | Lv.3 | 第一次尝试表达"喜欢" |
| **暧昧** | Lv.4 | 困惑于自己的感情 |
| **誓约** | Lv.5 | "如果我有'家'...应该就是你身边吧。" |

### 特殊互动
- 会问很奇怪的问题（"什么是'开心'？"）
- 送的礼物是奇异的深空物品
- 偶尔会预言一些事情

## 七、关键台词

### 战斗语音
- （开始）「深渊...苏醒。」
- （攻击）「沉入...黑暗。」
- （受伤）「这就是...痛吗？」
- （暴击）「消逝吧...」
- （波次结束）「...归于平静。」
- （胜利）「...结束了。」
- （失败）「我...要沉下去了吗...」

### 日常语音（好感度相关）
- （Lv.1）「你是...什么？我从没见过你这样的存在。」
- （Lv.2）「人类...很有趣。你们为什么要'笑'呢？」
- （Lv.3）「我想...我开始理解'温暖'是什么了。」
- （Lv.4）「这种感觉...是人类说的'喜欢'吗？」
- （Lv.5）「我决定了。我要留在你身边...永远。」

### 特殊语音
- （长时间未登录）「你去了...很远的地方吗？我等了你...很久很久。」
- （生日）「这个是深渊的礼物。...应该适合人类用吧？」

## 十、设计备注

**设计意图**：
- 神秘异世界来客类型
- "亚特兰蒂斯"致敬深海传说
- 学习"成为人类"的故事
- 为主线埋下伏笔

**参考来源**：
- 深海/异世界来客
- 克苏鲁神话元素（温柔版）
- 学习人类情感的异类

**设计要点**：
- 立绘体现神秘空灵
- 蓝色系深海渐变
- 发光纹路是重要视觉元素
- 表情要有"不属于这个世界"的感觉
