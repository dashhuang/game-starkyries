## 三、外形与气质

### 外形要素

| 要素 | 描述 |
|------|------|
| **年龄感** | 外表25岁左右（真实年龄不明） |
| **发型** | 深红色长直发，垂至腰际 |
| **发色** | 深红色，如同凝固的鲜血 |
| **瞳色** | 猩红色，在暗处会微微发光 |
| **身材** | 高挑曼妙，完美比例 |
| **服装** | 黑红相间的哥特式礼服 |
| **气质** | 妖艳神秘，危险而迷人 |
| **特征物品** | 蝙蝠造型发饰、颈间的红宝石项链 |

### 画面描述

外表二十五岁左右的女子——但她的眼神里藏着远超年龄的古老智慧。深红色的长直发如同凝固的鲜血般垂至腰际，在光线下泛着妖异的光泽。猩红色的眼眸深邃神秘，在黑暗中会微微发出红光。

身穿黑红相间的哥特式礼服，裙摆繁复华丽却不妨碍行动。身材高挑曼妙，每一个比例都完美得不像凡人。苍白的肌肤透着不正常的光泽，嘴唇是艳丽的红色。

发间别着一枚蝙蝠造型的发饰，颈间挂着一枚红宝石项链——据说那是某个古老家族的传承。她的动作优雅得像是在跳舞，每一步都带着致命的诱惑。

整个人就像是从古老传说中走出的永夜贵族，美丽、危险、永恒。

## 四、性格详解

### 表面性格
- 妖艳冷漠，对大多数事不感兴趣
- 优雅从容，永远不疾不徐
- 说话带着古老的腔调
- 偶尔会调戏有趣的人

### 内在性格
- 活了太久，对生死看得很淡
- 内心孤独，没有真正的同类
- 对有趣的灵魂会产生兴趣
- 渴望找到让她"心动"的存在

### 成长弧线
```
初期：冷眼旁观，把一切当做消遣
  ↓
中期：被指挥官和同伴们吸引
  ↓
后期：开始认真对待这份关系
  ↓
终章：找到了值得陪伴的存在
```

## 五、来历与背景

### 过去
没人知道她从哪里来，活了多久。她自己也不说——或者说，她自己也记不清了。有人说她是某个被遗忘文明的后裔，有人说她是实验体，也有人说...她真的是传说中的存在。

### 被帝国追杀
帝国发现了她"不老"的秘密，想要捕获研究。对她来说这只是漫长岁月中又一个无聊的麻烦，但这次她决定找个有趣的地方躲一躲。

**来历**（神秘）：
> "我从黑暗中来，在黑暗中行走了...很久。你问我多久？呵，我已经忘了怎么数年份了。"

### 被收留
被舰队"救起"——其实是她主动出现的，因为觉得这群人"有点意思"。现在把舰队当做消遣，但不知不觉间已经把这里当做了归属。

## 六、与指挥官的关系

**关系定位**：永夜的陪伴者、找到心动的古老存在

### 关系发展

| 阶段 | 好感度 | 互动特点 |
|------|--------|----------|
| **初识** | Lv.1 | 玩味观察，偶尔调戏 |
| **熟悉** | Lv.2 | 开始感兴趣，主动接近 |
| **信任** | Lv.3 | 讲述一些古老的往事 |
| **暧昧** | Lv.4 | "很久没有...心动的感觉了。" |
| **誓约** | Lv.5 | "愿意与我共度永夜吗？" |

### 特殊互动
- 只在夜晚出现
- 会调戏指挥官
- 讲故事时会透露惊人的历史知识

## 七、关键台词

### 战斗语音
- （开始）「呵...让我尝尝血的味道。」
- （攻击）「安息吧。」
- （受伤）「有点...疼呢。」
- （暴击）「鲜血的芬芳...」
- （波次结束）「...无聊。」
- （胜利）「这点程度，不过是消遣。」
- （失败）「看来...我太大意了。」

### 日常语音（好感度相关）
- （Lv.1）「哦？你就是指挥官？看起来...有点意思。」
- （Lv.2）「你让我想起了...很久以前的某个人。」
- （Lv.3）「活了这么久，有趣的事越来越少了。但你...是个例外。」
- （Lv.4）「这种感觉...是心动吗？我以为我早就忘记了。」
- （Lv.5）「我愿意用永恒来陪伴你。这是...第一次这么想。」

### 特殊语音
- （长时间未登录）「你去哪了？...时间对我来说不重要，但等你...很漫长。」
- （生日）「生日快乐。这一天对你很重要吧？那对我也是。」
