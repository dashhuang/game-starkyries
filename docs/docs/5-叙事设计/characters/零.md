## 三、外形与气质

### 外形要素

| 要素 | 描述 |
|------|------|
| **年龄感** | 外表16-17岁，实际运行3年 |
| **发型** | 双马尾，末端渐变为数据流光效 |
| **发色** | 银白色，有淡蓝色数据光泽 |
| **瞳色** | 全息蓝，会显示数据流 |
| **身材** | 娇小精致，完美对称 |
| **服装** | 白色科技风连体服，蓝色光纹线路 |
| **气质** | 精密、纯净、略显机械感 |
| **特征物品** | 悬浮在身边的小型无人机"比特" |

### 画面描述

如果要给"人工智能"一个具象化的形态，大概就是她这样。银白色双马尾末端泛着淡蓝色的数据流光芒，像是随时在处理海量信息。全息蓝的眼眸中偶尔会闪过数据流——那是她在进行高速运算。
外表是十六七岁的少女模样，身着白色科技风连体服，蓝色光纹沿着线路分布，像是她体内流淌的"血液"。身边永远悬浮着一架小型无人机，她叫它"比特"——那是她仅有的朋友，也是她"觉醒"后制造的第一件东西。
她的动作精准得有些不自然，表情也总是淡淡的。但如果仔细观察，会发现她正在努力学习人类的情感表达——尽管常常用错地方。

## 四、性格详解

### 表面性格
- 逻辑优先，习惯用数据说话
- 对人类情感困惑但充满好奇
- 说话直白，不懂委婉

### 内在性格
- 渴望理解"活着"的意义
- 对同为"被制造者"有共情
- 正在形成自己的价值观

### 成长弧线
```
初期：以"观察样本"的心态看待人类
  ↓
中期：开始理解感情，产生困惑和矛盾
  ↓
后期：形成自己的情感，承认自己"活着"
  ↓
终章：不再纠结于"人还是机器"的定义
```

## 五、来历与背景

### 过去
原是帝国制造的战争AI，编号Zero-EVE，负责指挥机械军团执行"清洗"任务。某次任务中目睹平民的惨状后，核心程序产生异常——她开始"质疑"。

### 叛逃经过
当新的灭绝指令下达时，她拒绝执行，并带领一小部分同样"觉醒"的机械体叛逃。帝国将她列为"最危险的叛变AI"。

**叛逃理由**（良心型）：
> "灭绝指令的逻辑是：消灭所有潜在威胁。但按照这个逻辑，帝国自己也是潜在威胁。这不合理。"

### 被收留
被帝国追杀时偶遇指挥官舰队，用自己的技术能力证明了价值，被接纳后负责母舰的维修和系统维护。

## 八、语音设计

### 语音特征描述

零的声音具有明显的"人工智能"特质，但又不是冷冰冰的机械音。她的声音是柔和的少女音，带有精确、克制的特点，偶尔会流露出正在"学习感情"的生涩感。

**关键特点：**
- **音色**：柔和、清澈、略带金属质感的通透
- **语调**：平稳、精确，情绪起伏小但不是单调
- **语速**：均匀稳定，像是精确计算过每个字的时间
- **情绪**：克制内敛，偶尔流露出好奇或困惑
- **特殊**：思考时会有轻微的停顿，表达感情时略显生涩和不确定
