| **特征物品** | 堡垒徽章、腰间的急救包 |

### 画面描述

二十八九岁的女军官，整个人散发着让人安心的气息。深棕色的长发盘成实用的发髻，一丝不苟却不显死板。琥珀色的眼眸温柔而坚定，像是能包容一切。

身穿深蓝色重装防御服，看起来厚重却不笨拙。胸口别着堡垒徽章——那是她防卫舰队的标志。身材高大丰满，是那种让人想要依靠的体型。

腰间永远挂着急救包，虽然她是指挥官不是医疗兵，但她总说"有备无患"。她走路的步伐稳健有力，说话的语气温和却有分量。

整个人就像一座会移动的堡垒，坚固、可靠、让人安心。

## 四、性格详解

### 表面性格
- 可靠沉稳，是大家的依靠
- 温柔包容，很少生气
- 照顾周到，像妈妈一样
- 说话温和，但很有分量

### 内在性格
- 保护欲极强，会为了部下拼命
- 其实很担心自己保护不了所有人
- 对失去部下这件事无法释怀
- 把舰队所有人当做自己的孩子

### 成长弧线
```
初期：尽全力保护每一个人
  ↓
中期：学会接受"无法保护所有人"的现实
  ↓
后期：理解保护的真正意义
  ↓
终章：成为舰队最坚实的后盾
```

## 五、来历与背景

### 过去
帝国防卫舰队的指挥官，负责保护边境殖民地。她的舰队以"从不丢失一个平民"著称，是边境居民最信赖的守护者。部下们都叫她"艾琳妈妈"——虽然她总是假装生气。

### 叛逃经过
某次防御作战中，上级命令她放弃一个"没有战略价值"的殖民地，撤退保存兵力。她违抗命令，带领部队死守殖民地，成功保护了居民。事后被以"抗命"罪判处军法处置。

**叛逃理由**（良心型）：
> "那里有三千个平民。什么叫'没有价值'？人命不是价值吗？"

### 被收留
带着愿意追随的部下逃离帝国，被舰队接纳。她的防御能力和领导力很快让她成为舰队的重要支柱。

## 六、与指挥官的关系

**关系定位**：可靠的后盾、温柔的守护者

### 关系发展

| 阶段 | 好感度 | 互动特点 |
|------|--------|----------|
| **初识** | Lv.1 | 照顾过头，把指挥官也当孩子 |
| **熟悉** | Lv.2 | 开始尊重指挥官的判断 |
| **信任** | Lv.3 | 倾诉曾经失去部下的痛苦 |
| **暧昧** | Lv.4 | "不是照顾孩子...是想照顾你。" |
| **誓约** | Lv.5 | "让我永远守护在你身边。" |

### 特殊互动
- 会关心指挥官有没有吃饭休息
- 伤员第一个冲过去照顾
- 被叫"妈妈"会假装生气实际很高兴

## 七、关键台词

### 战斗语音
- （开始）「堡垒号，启动防御！所有人，我会保护你们！」
- （攻击）「退后！」
- （受伤）「这点程度...还能撑住！」
- （暴击）「不要靠近我的同伴！」
- （波次结束）「...都平安就好。」
- （胜利）「太好了...大家都没事。」
- （失败）「对不起...我没能保护住...」

### 日常语音（好感度相关）
- （Lv.1）「指挥官，今天吃饭了吗？来，我做了便当。」
- （Lv.2）「有什么困难都可以和我说。我会帮你的。」
- （Lv.3）「我曾经...失去过部下。那种痛，我不想再经历了。」
- （Lv.4）「对你的感情...好像和对其他人不一样。」
- （Lv.5）「不管发生什么，我都会守护你。这是誓言。」

### 特殊语音
- （长时间未登录）「你去哪了！让人担心死了！...回来就好。」
- （生日）「生日快乐！我做了蛋糕，虽然样子不太好看...」
