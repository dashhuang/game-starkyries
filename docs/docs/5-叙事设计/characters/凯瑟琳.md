## 三、外形与气质

### 外形要素

| 要素 | 描述 |
|------|------|
| **年龄感** | 23-25岁，成熟干练 |
| **发型** | 棕红色短发，蓬松随意 |
| **发色** | 棕红色，充满活力 |
| **瞳色** | 琥珀色，热情明亮 |
| **身材** | 高大健壮，力量型体格 |
| **服装** | 军绿色炮兵制服，卷起袖子 |
| **气质** | 豪爽开朗，大姐头气场 |
| **特征物品** | 永远叼着的棒棒糖（戒烟用）、手臂上的火炮纹身 |

### 画面描述

二十四五岁的女炮兵，整个人像一门随时准备开火的大炮。棕红色短发蓬松随意，像是刚从炮火中走出来。琥珀色的眼眸热情明亮，笑起来格外爽朗。

身材高大健壮，是力量型的体格。军绿色炮兵制服的袖子卷到手肘，露出结实的手臂，右臂上有一个火炮纹身——那是她的炮兵部队的标志。嘴里永远叼着一根棒棒糖——是她戒烟的替代品。

她的存在感很强，笑声能传很远，走路带风。看起来大大咧咧，但处理起事情来意外地靠谱。是那种让人想叫一声"大姐"的类型。

## 四、性格详解

### 表面性格
- 豪爽大气，说话直来直去
- 大大咧咧，不拘小节
- 乐观开朗，什么事都能笑着面对
- 重情重义，把同伴当家人

### 内在性格
- 其实很细心，只是不表现
- 会默默照顾后辈
- 对自己闯的祸有点愧疚
- 害怕连累别人

### 成长弧线
```
初期：大大咧咧，经常闯祸
  ↓
中期：开始注意控制火力（字面意思）
  ↓
后期：学会"该炸的时候炸"
  ↓
终章：成为舰队最可靠的火力支援
```

## 五、来历与背景

### 过去
帝国炮兵部队的王牌炮手，以火力猛著称。她的口号是"没有什么是一炮解决不了的，如果有，那就两炮"。战绩辉煌，但也因为火力控制问题闯过不少祸。

### 叛逃经过
在一次演习中，因为坐标计算失误（其实是上面给错了坐标），她把炮弹打进了某位高官的度假别墅。更倒霉的是，那位高官当时正好在别墅里。虽然没死人，但她被判"蓄意谋杀上级"。

**叛逃理由**（荒诞型）：
> "坐标是他们给的！我怎么知道那是长官的别墅！...好吧确实炸爽了，但那不是重点！"

### 被收留
逃跑时顺手炸了追兵的补给站，被舰队发现后一拍即合。她的火力正是舰队需要的，舰队的自由也正是她想要的。

## 六、与指挥官的关系

**关系定位**：靠谱的大姐、豪爽的守护者

### 关系发展

| 阶段 | 好感度 | 互动特点 |
|------|--------|----------|
| **初识** | Lv.1 | 热情过头，大力拍肩 |
| **熟悉** | Lv.2 | 把指挥官当小兄弟/小妹妹照顾 |
| **信任** | Lv.3 | 倾诉闯祸的愧疚 |
| **暧昧** | Lv.4 | 变得有点不自在，不知道怎么表达 |
| **誓约** | Lv.5 | "以后我的炮火只为你开路！" |

### 特殊互动
- 经常拉指挥官喝酒（非酒精饮料）
- 闯祸后会来找指挥官诉苦
- 会默默帮指挥官解决麻烦

## 七、关键台词

- （开始）「炮击号，火力全开！」

- （Lv.4）「最近...不知道为啥看到你就紧张...」
- （Lv.5）「以后你往哪指，姐的炮就往哪轰！」

- （生日）「姐给你弄了个大惊喜！放心不是炮弹！...应该不是。」

## 十、设计备注

**设计意图**：
- 豪爽大姐类型
- "炸了长官别墅"的荒诞叛逃理由
- 大大咧咧但细心可靠
- 火力狂但不是无脑

**参考来源**：
- 女性大姐头角色
- 炮兵/火力担当
- 喜剧类军人

**设计要点**：
- 立绘体现豪爽和力量感
- 棕红色系为主
- 卷袖子和棒棒糖是重要细节
- 要有"大姐头"的气场
