## 三、外形与气质

### 外形要素

| 要素 | 描述 |
|------|------|
| **年龄感** | 17-19岁，少女感 |
| **发型** | 双马尾，发尾微微上翘 |
| **发色** | 樱粉色渐变白色 |
| **瞳色** | 天蓝色，明亮清澈 |
| **身材** | 娇小纤细，轻盈体型 |
| **服装** | 白色机车夹克配短裙，红色护目镜架在额头 |
| **气质** | 活力四射，像风一样自由 |
| **特征物品** | 红色护目镜、左耳的羽毛耳饰 |

### 画面描述

十七八岁的少女，浑身散发着不安分的活力。樱粉色的双马尾在身后飘扬，发尾渐变成淡淡的白色，像是划过天际的流星尾迹。天蓝色的大眼睛闪烁着兴奋的光芒，总是东张西望，仿佛随时准备冲向下一个目的地。

身穿白色机车夹克，下摆短得刚好露出细腰，搭配同色系百褶短裙。红色的飞行护目镜随意架在额头，左耳戴着一枚白色羽毛耳饰——据说是她第一次突破音速时从云层里捡到的（虽然没人相信）。

整个人就像一阵旋风，从不在一个地方停留太久。站着的时候也会不自觉地踮脚，仿佛随时要起飞。

## 四、性格详解

### 表面性格
- 急性子，说话快走路快做事快
- 闲不下来，总在找事做
- 直率坦诚，想到什么说什么
- 乐观开朗，很少沮丧

### 内在性格
- 害怕停下来，不知道静止时该做什么
- 追求速度是因为想追上某个人（已故的姐姐）
- 其实很害怕失去，所以用忙碌麻痹自己
- 对被信任的人会展现依赖的一面

### 成长弧线
```
初期：风风火火，总是冲在最前面
  ↓
中期：因为太快导致失误，开始反思
  ↓
后期：学会"快"与"稳"的平衡
  ↓
终章：明白真正的速度是为了保护重要的人
```

## 五、来历与背景

### 过去
帝国舰队最年轻的王牌飞行员，16岁就打破了多项速度记录。被称为"帝国之风"，是军方重点培养的对象。她的姐姐曾是更优秀的飞行员，在一次任务中牺牲，美月一直在追逐姐姐的背影。

### 叛逃经过
在一次紧急任务中，她以三倍标准速度穿越封锁区，成功完成救援。然而帝国交通管理局以"危害公共安全罪"起诉她——超速驾驶，罚款三百万星币，吊销执照，判处五年监禁。

**叛逃理由**（荒诞型）：
> "超速？我救了二十七个人！他们说我超速危害安全？那些人不救才危害安全吧！"

### 被收留
从监狱转运途中被舰队劫走。起初只是借用她的驾驶技术，后来成为舰队不可或缺的"风一样的存在"。

## 六、与指挥官的关系

**关系定位**：元气后辈、需要被拉住的小跑车

### 关系发展

| 阶段 | 好感度 | 互动特点 |
|------|--------|----------|
| **初识** | Lv.1 | 话多热情，问东问西 |
| **熟悉** | Lv.2 | 开始黏着指挥官，什么事都要报告 |
| **信任** | Lv.3 | 倾诉关于姐姐的事，罕见地安静 |
| **暧昧** | Lv.4 | 会不好意思地放慢脚步等指挥官 |
| **誓约** | Lv.5 | "和你在一起的时候...我不想那么快。" |

### 特殊互动
- 汇报工作永远超快语速，需要让她重说
- 会拉着指挥官去兜风
- 罕见安静时说明心事重重

## 十、设计备注

**设计意图**：
- 元气少女类型，给舰队带来活力
- 急性子设定提供喜剧效果
- 姐姐的故事线为角色增加深度
- "超速驾驶"的叛逃理由体现帝国荒诞

**参考来源**：
- 赛车娘系角色
- 头文字D的速度追求
- 元气少女类型

**设计要点**：
- 立绘需体现动感和速度感
- 头发、衣角要有风吹拂的感觉
- 表情以兴奋、期待为主
- 护目镜是重要标识物
