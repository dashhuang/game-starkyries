## 三、外形与气质

### 外形要素

| 要素 | 描述 |
|------|------|
| **年龄感** | 19-21岁，略显沧桑 |
| **发型** | 亚麻色短发，略显凌乱 |
| **发色** | 亚麻色，沾着灰尘的感觉 |
| **瞳色** | 淡绿色，藏着坚定 |
| **身材** | 瘦削但结实，幸存者体型 |
| **服装** | 旧式军用背心，实用主义风格 |
| **气质** | 沉稳内敛，像是经历过太多 |
| **特征物品** | 脖子上的子弹吊坠、手腕上的旧表 |

### 画面描述

十九二十岁的少女，却有着不符合年龄的沉稳。亚麻色的短发略显凌乱，像是没空打理。淡绿色的眼眸看起来平静，但深处藏着不灭的火焰。

身穿旧式军用背心，实用主义风格，没有任何多余的装饰。衣服上有缝补的痕迹——那是在废墟中学会的技能。身材瘦削但结实，是在艰苦环境中锻炼出来的体型。

脖子上挂着一颗子弹做成的吊坠——那是她父亲留下的，也是她复仇的象征。手腕上戴着一块老旧的手表，时间永远停在殖民地被摧毁的那一刻。

她说话不多，但每一句都经过深思熟虑。看起来温和，但握枪时眼神会变得锐利如刀。

## 四、性格详解

### 表面性格
- 耐心沉稳，从不冲动
- 低调谨慎，不爱出风头
- 温和有礼，好相处
- 做事靠谱，值得信赖

### 内在性格
- 心中藏着对帝国的刻骨仇恨
- 活着就是为了复仇
- 害怕失去更多人
- 在复仇和放下之间挣扎

### 成长弧线
```
初期：为复仇而活，心无旁骛
  ↓
中期：开始在意同伴，动摇复仇决心
  ↓
后期：找到除了复仇以外的活着的理由
  ↓
终章：从"幸存者"变成"守护者"
```

## 五、来历与背景

### 过去
出生在边境殖民地"绿洲星"，是个平凡幸福的女孩。父亲是殖民地的民兵队长，教她射击只是为了自卫。然而帝国为了开采资源，将整个殖民地定性为"非法定居点"并摧毁。

### 叛逃经过
她不是叛逃——她从来就不属于帝国。那一天，她躲在地下室，听着外面的惨叫和爆炸声，等到一切安静后爬出来，发现家园变成废墟，家人全都死了。她是整个殖民地唯一的幸存者。

**来历**（幸存者）：
> "我不是叛逃者。我是幸存者。帝国欠我的，我会一发一发地讨回来。"

### 被收留
在废墟中自学狙击技术，靠猎杀落单的帝国士兵生存。被舰队发现时已经独自杀了十几个帝国人。舰队给了她一个更大的舞台——和一个新的家。

## 六、与指挥官的关系

**关系定位**：找到新家的人、放下仇恨学会爱的人

### 关系发展

| 阶段 | 好感度 | 互动特点 |
|------|--------|----------|
| **初识** | Lv.1 | 保持距离，只谈任务 |
| **熟悉** | Lv.2 | 开始放松警惕，偶尔聊天 |
| **信任** | Lv.3 | 讲述殖民地的事，第一次流泪 |
| **暧昧** | Lv.4 | "你...让我想起了家的感觉。" |
| **誓约** | Lv.5 | "我不只是为复仇而活了。我想为你活着。" |

### 特殊互动
- 会默默照顾指挥官的起居
- 看到类似家乡的风景会发呆
- 会为指挥官做殖民地的家乡菜

## 十、设计备注

**设计意图**：
- 幸存者类型，复仇与放下的故事
- 年轻却沧桑的反差
- "子弹吊坠"象征复仇
- 最终学会放下仇恨

**参考来源**：
- 幸存者题材
- 复仇故事
- 成长与和解

**设计要点**：
- 立绘体现年轻但沧桑的感觉
- 朴实无华的实用风格
- 眼神要有"藏着什么"的感觉
- 子弹吊坠是重要标识
