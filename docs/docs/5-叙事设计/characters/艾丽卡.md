## 三、外形与气质

### 外形要素

| 要素 | 描述 |
|------|------|
| **年龄感** | 20-22岁，年轻气盛 |
| **发型** | 金色波浪长发，随意披散 |
| **发色** | 璀璨金色，像燃烧的火焰 |
| **瞳色** | 深红色，充满战意 |
| **身材** | 高挑健美，肌肉匀称的战士体型 |
| **服装** | 华丽的红黑贵族军装，金色纹章装饰 |
| **气质** | 骄傲张扬，战场女武神 |
| **特征物品** | 腰间的家族佩剑、手背的决斗伤疤 |

### 画面描述

二十出头的贵族少女，浑身散发着高傲与战意交织的气场。金色波浪长发如同燃烧的火焰般披散在肩后，深红色的眼眸里燃烧着永不熄灭的战斗欲望。

身穿红黑相间的华丽军装，金色纹章装饰在肩章和袖口，彰显着布伦家族的尊贵。然而军装上的几处破损和修补痕迹说明这不是摆设——这是真正上过战场的战衣。

身材高挑健美，不是贵族小姐的柔弱，而是久经沙场的战士体型。腰间挂着家族传承的佩剑，虽然舰战中用不上，但她从不取下。右手手背上有一道显眼的伤疤——那是和皇室成员决斗时留下的"战利品"。

整个人像是一头随时准备冲锋的战狮，骄傲、强大、无所畏惧。

## 四、性格详解

### 表面性格
- 热血好战，看到强者就想打
- 贵族傲气，但不是看不起人
- 直爽豪迈，不喜欢弯弯绕绕
- 输赢分明，愿赌服输

### 内在性格
- 追求纯粹的战斗，厌恶阴谋诡计
- 对弱者有保护欲但嘴上不承认
- 希望证明自己不只是"贵族小姐"
- 对被认可这件事很在意

### 成长弧线
```
初期：莽撞冲动，只想证明自己
  ↓
中期：学会团队配合，不只靠蛮力
  ↓
后期：理解真正的强大不只是战斗力
  ↓
终章：成为能保护同伴的"骑士"
```

## 五、来历与背景

### 过去
布伦家族是帝国军事贵族，世代出名将。艾丽卡从小接受严格的战斗训练，立志成为家族荣耀的延续。然而她更喜欢用拳头而不是政治手段解决问题，这在贵族圈子里显得格格不入。

### 叛逃经过
在一次宫廷舞会上，皇室旁系成员当众羞辱了她的战友（一个平民出身的士兵）。艾丽卡当场提出决斗，并在光明正大的对决中打伤了对方。结果她被判"暴力伤害皇室"，家族也被连累。

**叛逃理由**（荒诞型+良心型）：
> "他先侮辱我的战友，我堂堂正正决斗赢了，凭什么我有罪？贵族的荣耀呢？哼，这种贵族不要也罢！"

### 被收留
独自逃亡时被舰队发现，一开始因为骄傲不愿接受帮助，被打了一顿后（字面意思）才愿意加入。现在是舰队最热衷于战斗的成员。

## 六、与指挥官的关系

**关系定位**：忠诚的骑士、崇拜强者的傲娇

### 关系发展

| 阶段 | 好感度 | 互动特点 |
|------|--------|----------|
| **初识** | Lv.1 | 不服气，总想挑战 |
| **熟悉** | Lv.2 | 被战术能力折服，开始尊重 |
| **信任** | Lv.3 | 承认指挥官是"值得效忠的人" |
| **暧昧** | Lv.4 | 傲娇发作，明明在意却嘴硬 |
| **誓约** | Lv.5 | "我的剑为你出鞘，我的心...也是。" |

### 特殊互动
- 打赢了会来炫耀，打输了会闷闷不乐
- 送的礼物都是武器或战斗相关
- 受伤时会逞强说没事

## 七、关键台词

### 战斗语音
- （开始）「哈哈！来吧！让我好好享受战斗！」
- （攻击）「受死吧！」
- （受伤）「这点程度...正好热身！」
- （暴击）「这才像样！」
- （波次结束）「不够！再来更多！」
- （胜利）「哈哈哈！痛快！」
- （失败）「可恶...这次算我输！」

### 日常语音（好感度相关）
- （Lv.1）「你就是指挥官？看起来不怎么能打嘛。」
- （Lv.2）「哼，战术方面...勉强承认你有两下子。」
- （Lv.3）「我艾丽卡·冯·布伦愿为你效力！这是骑士的誓言！」
- （Lv.4）「才、才不是特意来找你的！只是...路过！」
- （Lv.5）「能遇到你...是我最大的幸运。这是真心话。」

### 特殊语音
- （长时间未登录）「你跑哪去了！我都没对手了！」
- （生日）「这把剑送你！别误会，只是家族规矩要给认可的人送剑！」

## 十、设计备注

**设计意图**：
- 热血战士类型，带来战斗的激情
- 贵族+武斗派的反差萌
- 傲娇的忠诚骑士定位
- 决斗叛逃理由体现"荣耀与虚伪"的冲突

**参考来源**：
- 骑士类角色
- 热血战斗系角色
- 贵族武斗家

**设计要点**：
- 立绘需体现贵族气质与战士力量的结合
- 金发红眼的经典强者配色
- 表情以自信、战意为主
- 佩剑和手背伤疤是重要细节
