### 画面描述

二十七八岁的女战士，整个人就像一座不可撼动的山岳。银白色的短发凌乱地披散，像是北境的雪原。冰蓝色的眼眸坚定得如同钢铁，里面燃烧着不屈的火焰。

身穿厚重的北欧风格重甲，银色的金属在光线下闪闪发光。这套装甲陪伴她守护边境多年，每一道划痕都是荣耀的勋章。她的身材高大健壮，是标准的战士体型。

手持一面巨大的盾牌，上面刻着她家族的纹章——那是一只展翅的苍鹰。颈间挂着一个朴素的护身符，是她母亲留下的遗物。

她很少说话，但每一次开口都掷地有声。站在那里就像一道不可逾越的城墙，让人感到无比安心。

## 四、性格详解

### 表面性格
- 刚强不屈，绝不后退
- 沉默寡言，行动多于言语
- 严肃认真，从不开玩笑
- 对敌人毫不留情

### 内在性格
- 对失去故乡的痛苦深埋心底
- 把保护当做唯一的活着意义
- 内心柔软，会默默照顾弱小
- 渴望有朝一日能重建家园

### 成长弧线
```
初期：沉默守护，不与人交流
  ↓
中期：开始接受舰队是新的家
  ↓
后期：学会为自己而活
  ↓
终章：从"守护废墟"变成"守护希望"
```

## 五、来历与背景

### 过去
北境殖民地"霜落"的守护者，家族世代守护这片土地。她是最后一任守护者，也是唯一的幸存者。帝国为了开采资源摧毁了霜落，她在废墟中守护了三年，直到再也没有需要守护的人。

### 被收留
被舰队在废墟中发现时，她几乎饿死冻死。起初不愿离开——那是她的故乡。后来被说服：与其守护废墟，不如守护还活着的人。

**来历**（幸存者）：
> "霜落没了。但我还在。我会继续守护...直到最后一口气。"

### 加入舰队
加入舰队后成为最可靠的防线。她的盾从未让敌人越过一步，就像她曾经守护霜落一样。

## 六、与指挥官的关系

**关系定位**：沉默的守护者、找到新家的战士

### 关系发展

| 阶段 | 好感度 | 互动特点 |
|------|--------|----------|
| **初识** | Lv.1 | 沉默寡言，只谈任务 |
| **熟悉** | Lv.2 | 开始说一些简短的话 |
| **信任** | Lv.3 | 讲述霜落的故事，第一次流泪 |
| **暧昧** | Lv.4 | "你...是我新的守护对象。" |
| **誓约** | Lv.5 | "我的盾，从今以后只为你而立。" |

### 特殊互动
- 会默默站在指挥官身边
- 受伤后不说，需要别人发现
- 偶尔会做北境的传统食物
