## 二、战舰属性

| 设计理念 | 说明 |
|---------|------|
| **无负面属性** | 新手友好，没有任何惩罚机制 |
| **加成较弱但全面** | 引擎+5%、护盾+5、资源回收+8 |
| **高容错率** | 适合学习游戏机制，任何Build都能玩 |
| **难度评级** | ★☆☆☆☆（最简单） |

```
先锋号属性：
├─ 引擎推力 +5%（略快，容易躲避）
├─ 最大护盾 +5（多挨几下）
├─ 资源回收 +8（前期经济更宽裕）
├─ 无负面属性 ✓
└─ 无特殊能力（纯粹靠玩家操作和Build）
```

**玩法说明**：
作为初始角色，先锋号的设计理念是"没有短板"。虽然没有突出的强项，但也没有任何需要规避的弱点，让新手可以专注于学习游戏基础机制，而不用担心被角色特性坑到。

## 三、外形与气质

### 外形要素

| 要素 | 描述 |
|------|------|
| **年龄感** | 18-20岁，青春活力 |
| **发型** | 马尾，发尾挑染一缕星蓝色 |
| **发色** | 黑色，普通但亲切 |
| **瞳色** | 琥珀色，明亮有神 |
| **身材** | 标准身材，不高不矮 |
| **服装** | 白蓝相间标准舰长制服，金色肩章 |
| **气质** | 邻家少女感，亲和力强 |
| **特征物品** | 无特殊配饰，强调"普通感" |

### 画面描述

十八九岁的少女，有着邻家女孩般的亲切感。黑色长发扎成利落的马尾，发尾挑染了一缕淡淡的星蓝色——那是她给自己的小小任性。琥珀色的眼睛明亮有神，笑起来会弯成月牙。
身穿白蓝相间的标准舰长制服，整洁得一丝不苟，金色肩章在灯光下微微反光。身材匀称，不高不矮，站在人群里不会特别显眼，却总能让人感到莫名的安心。
没有夸张的配饰，没有华丽的装扮。她的"普通"本身就是一种设计——在这个充斥着传奇和异类的舰队里，星遥是最接近"正常人"的存在，也因此成为了所有人的情感锚点。
整个人给人干净、温暖、充满朝气的感觉，像是清晨的第一缕阳光。

## 四、性格详解

### 表面性格
- 开朗活泼，爱笑
- 话多，喜欢和人聊天
- 乐观向上，遇到困难也不放弃
- 有点冒失，偶尔闯祸

### 内在性格
- 责任感极强，把舰队的事当成自己的事
- 会逞强，不想让别人担心
- 对过去有遗憾，偶尔流露出怀念
- 害怕失去同伴，所以拼命保护大家

### 成长弧线
```
初期：努力表现得开朗，掩盖不安
  ↓
中期：在战斗中逐渐展现可靠的一面
  ↓
后期：学会依靠别人，接受指挥官的支持
  ↓
终章：从"想被保护的女孩"成长为"守护同伴的战士"
```

## 五、来历与背景

### 帝国时期
- 军校全科成绩第三名（第一名是亚历珊德拉将军的女儿）
- 被称为"军校之星"，前途光明
- 性格好，朋友很多，在军校很受欢迎

### 叛逃经过
在一次边境巡逻任务中，发现了一艘被帝国追杀的民用飞船。船上是一个被判"崇拜不绝对"的家庭——父亲只是在网上问了一句"皇帝陛下真的还活着吗？"

星遥违抗命令，帮助他们逃跑。结果被同行的监察官举报，自己也被打成"叛逆同党"。

**叛逃理由**（良心型）：
> "我只是...觉得不能看着一家人去死啊。他们的小女儿才5岁...谁能想到问一句话就是死罪呢？"

### 被收留
- 逃亡途中被母舰救起
- 是最早加入舰队的成员之一
- 自然而然地成为了舰队的"门面担当"

## 八、语音设计

### 语音特征描述

星遥的声音是典型的"邻家少女"类型，年龄感在18-20岁之间。音色清亮、温暖，带有自然的青春活力感。语速偏快，尤其是兴奋或紧张时会不自觉加速。情绪表达非常直接，喜怒哀乐都能从声音中清晰感受到。

**关键特点：**
- **音色**：清亮、温暖、有少许气声（尤其是兴奋时）
- **语调**：活泼上扬，句尾常带轻快的上挑
- **语速**：中偏快，情绪激动时会加速
- **情绪**：表达直接，容易感染他人
- **特殊**：紧张时会有轻微的颤音，开心时会不自觉拉长尾音

## 十、新手引导设计

### 第一次进入游戏
```
[画面：母舰内部，警报响起]

星遥：「指挥官！敌舰接近中！」
星遥：「没时间解释了，我们得立刻出发寻找新的跳跃坐标！」
星遥：「我来驾驶先锋号，你负责指挥！」

→ 进入战斗（基础操作教学）
```

### 第一次战斗中（适时提示）
```
星遥：「用方向键移动！敌人太多就先躲开！」
星遥：「武器会自动攻击的，我们专心走位就好！」
星遥：「那个闪光的是能量晶体，捡起来！」
星遥：「撑过去！跳跃坐标快计算完了！」
```

### 第一次胜利
```
星遥：「成功了！我们找到新的坐标了！」
星遥：「指挥官，你的指挥真厉害！」
星遥：「接下来...让我们继续前进吧！」

→ 解锁补给站系统
```

### 第一次失败
```
星遥：「呜...果然还是不行呢...」
星遥：「没事的，零会帮忙修复先锋号的。」
星遥：「我们还有时间，重新调整一下装备再出发吧！」

[零登场]
零：「战损分析完毕。主推进器损伤17%，已开始修复。」
零：「预计修复时间：3分钟。建议利用这段时间优化装备配置。」

星遥：「指挥官，这是零。她负责母舰的维修工作。」
星遥：「有她在，不管战舰坏成什么样都能修好哦！」

→ 认识零、解锁装备系统
```

### 补给站教学
```
星遥：「这是补给站！可以用晶体购买武器和模块。」
星遥：「每次只能选一个，要慎重考虑哦。」
```

## 十二、设计备注

**设计意图**：
- 作为女一号，需要足够的亲和力让玩家喜欢
- 作为新手引导，台词要自然不说教
- 性格设计参考了"正统女主"但增加了一些小缺点避免完美化
- 战舰属性设计参考Brotato的Well Rounded，强调新手友好

**参考来源**：
- 明日方舟·阿米娅（亲和力、成长线）
- 碧蓝航线·企业（责任感、可靠）
- Brotato·Well Rounded（属性设计）
