## 三、外形与气质

### 外形要素

| 要素 | 描述 |
|------|------|
| **年龄感** | 看起来18-19岁，实际未知 |
| **发型** | 长直发，及膝，轻盈飘逸 |
| **发色** | 银灰渐变透明，发梢似乎在消散 |
| **瞳色** | 淡紫色，瞳孔深处有迷雾感 |
| **身材** | 纤细飘渺，存在感很低 |
| **服装** | 层叠薄纱长裙，白灰渐变，似雾似烟 |
| **气质** | 空灵虚幻，仿佛随时会消失 |
| **特征物品** | 无（她本身就像个特征） |

### 画面描述

她就像一缕即将散去的晨雾，美丽而不真实。银灰色的长发垂落及膝，发梢渐变成透明，仿佛正在慢慢消融于空气之中。淡紫色的眼眸深处像是藏着一片迷雾，让人看不清她在想什么。
身着层叠的薄纱长裙，白灰渐变的色调让她整个人都像是笼罩在雾气之中。她的动作轻柔得没有声音，走路也不会留下脚印——有人说这是技能，也有人说她根本就不是"人"。
她总是带着温柔但疏离的微笑，像是在告诉所有人：不要靠太近，因为我随时可能消失。
没有人知道她从哪里来，她自己似乎也不太确定。

## 四、性格详解

### 表面性格
- 温柔有礼，但保持距离
- 说话轻声细语，总带着一丝忧郁
- 存在感极低，常常被遗忘

### 内在性格
- 对"存在"感到不安
- 渴望被记住，却又害怕
- 可能知道一些关于自己来历的事

### 成长弧线
```
初期：飘忽不定，随时准备消失
  ↓
中期：被指挥官"看见"，产生留下的想法
  ↓
后期：开始探索自己的来历
  ↓
终章：不管是什么，都选择"存在"下去
```

## 五、来历与背景

### 过去
不明。她最早的记忆是在一片迷雾中醒来，不知道自己是谁，从哪来，要去哪。她只知道自己叫"希尔薇·米斯特"，以及——不要让任何人太靠近。

### 叛逃经过
没有明确的叛逃经历。她像幽灵一样在宇宙中漂流，直到被指挥官的舰队"捡到"。

**叛逃理由**（神秘型）：
> "我也想知道我是谁，我从哪里来。也许……跟着你们能找到答案。"

### 被收留
在一片废弃空间站被发现，她说不清自己怎么到那里的。指挥官没有追问，只是邀请她上船——这让她第一次感到被"接纳"。

---

## 六、与指挥官的关系

**关系定位**：让她"存在"的锚点

### 关系发展

| 阶段 | 好感度 | 互动特点 |
|------|--------|----------|
| **初识** | Lv.1 | 经常被忽略，但指挥官总能找到她 |
| **熟悉** | Lv.2 | 惊讶于指挥官总是记得她 |
| **信任** | Lv.3 | 开始主动出现在指挥官身边 |
| **暧昧** | Lv.4 | "你能……一直看着我吗？" |
| **誓约** | Lv.5 | "只要你还记得我，我就不会消失。" |

### 特殊互动
- 会在指挥官看不到的地方默默注视
- 被忽略时会有点难过，但不会说出来
- 每次出现都像第一次见面一样问好

## 七、关键台词

### 战斗语音
- （开始）「我……会努力的。」
- （攻击）「请消散吧。」
- （受伤）「没关系……习惯了。」
- （暴击）「……对不起。」
- （波次结束）「结束了吗……？」
- （胜利）「我还在呢……」
- （失败）「果然……我不行吗……」

### 日常语音（好感度相关）
- （Lv.1）「啊……你看得见我吗？」
- （Lv.2）「你总是能找到我……真厉害。」
- （Lv.3）「能和你说话的时候，我感觉自己真的存在。」
- （Lv.4）「不要忘记我，好吗……？」
- （Lv.5）「我想一直留在你身边……可以吗？」

### 特殊语音
- （长时间未登录）「你……还记得我吗？」
- （生日）「生日快乐。我不会忘记的，永远都不会。」
