## 三、外形与气质

### 外形要素

| 要素 | 描述 |
|------|------|
| **年龄感** | 23-25岁，成熟端庄 |
| **发型** | 银白色长卷发，精心打理 |

| **瞳色** | 紫罗兰色，高贵神秘 |
| **身材** | 高挑丰满，贵族体态 |
| **服装** | 紫白相间的皇家礼服改战装 |
| **气质** | 高贵优雅，皇族风范 |
| **特征物品** | 皇室戒指、紫罗兰胸针 |

### 画面描述

二十四五岁的贵族女性，浑身散发着与生俱来的高贵气息。银白色的长卷发精心打理，每一缕都恰到好处。紫罗兰色的眼眸深邃而高贵，是帝国皇室旁系的标志。

身穿紫白相间的服装，是皇家礼服改制的战斗装——保留了优雅的裁剪，却更适合战斗。衣料是上等的丝绸与防护材料的结合，既华贵又实用。

身材高挑丰满，是典型的贵族体态。无名指上戴着皇室戒指——虽然她已被除籍，但这是她母亲留下的，她从不摘下。胸口别着一枚紫罗兰胸针，是她名字的象征。

举手投足都是皇族的教养，即使流亡也不曾丢失。但偶尔，当她以为没人看见时，那优雅的面具下会露出一丝疲惫。

## 四、性格详解

### 表面性格
- 高贵优雅，礼仪完美
- 端庄自持，从不失态
- 待人有礼但保持距离
- 说话文雅，措辞讲究

### 内在性格
- 内心柔软，很容易被感动
- 坚强隐忍，独自承受一切
- 对背叛她的家族又恨又爱
- 渴望真诚的感情

### 成长弧线
```
初期：保持皇族距离，不愿放下身段
  ↓
中期：被同伴的真诚打动，开始融入
  ↓
后期：学会做一个"普通人"
  ↓
终章：放下皇族的包袱，做真正的自己
```

## 五、来历与背景

### 过去
帝国皇室旁系公主，从小接受最精英的教育。在宫廷中学会了优雅、社交、政治，也学会了勾心斗角。她曾经有机会成为皇后，但一场宫斗让她失去了一切。

### 叛逃经过
在皇位继承的争斗中，她被自己的姐姐设计陷害，以"通敌叛国"的罪名被剥夺身份，流放边境。流放途中，押送她的船只"意外"发生故障，所有人都要"死于事故"。她是唯一的幸存者。

**叛逃理由**（政治型）：
> "我的姐姐想要我的命。皇室...不，那个家族，不需要我了。"

### 被收留
流亡途中被舰队救起。虽然失去了地位，但她的教养和见识使她很快适应了新环境。只是再也不说自己曾是公主。
