## 三、外形与气质

### 外形要素

| 要素 | 描述 |
|------|------|
| **年龄感** | 24-26岁，病态美感 |
| **发型** | 深红色长波浪发，凌乱中带着美感 |
| **发色** | 深红色，如同干涸的血 |
| **瞳色** | 紫红色，带着扭曲的光 |
| **身材** | 纤细苍白，有很多旧伤疤 |
| **服装** | 黑红色束缚风格服装，带有荆棘装饰 |
| **气质** | 病态艳丽，危险的美 |
| **特征物品** | 荆棘纹身、颈间的玫瑰吊坠 |

### 画面描述

二十五六岁的女子，浑身散发着病态的美感。深红色的长波浪发凌乱地披散，像是被风暴蹂躏过的玫瑰。紫红色的眼眸里有着扭曲的光芒，看向人时像是在审视猎物。

身穿黑红色束缚风格服装，带有荆棘图案的装饰。衣服露出的皮肤苍白得不正常，上面有很多旧伤疤——那些是她自己造成的，也是她"审讯"时留下的。

左臂上有一圈荆棘纹身，从手腕蔓延到肩膀。颈间挂着一枚枯萎的玫瑰吊坠——据说是她最后一个"清白"受害者送的。

她笑起来的时候很美，但那种美让人毛骨悚然。整个人就像一朵长在荆棘丛中的玫瑰，美丽、危险、带着刺。

## 四、性格详解

### 表面性格
- 病态美感，对痛苦有奇怪的迷恋
- 说话轻柔，但内容往往很可怕
- 喜欢观察别人的反应
- 偶尔会自虐

### 内在性格
- 其实极度自我厌恶
- 用痛苦惩罚自己曾经的罪行
- 渴望被原谅但觉得自己不配
- 在寻找救赎的方式

### 成长弧线
```
初期：自我惩罚，沉浸在痛苦中
  ↓
中期：被同伴接受，开始动摇
  ↓
后期：学会"被原谅"
  ↓
终章：从"用痛苦赎罪"变成"用守护赎罪"
```

## 五、来历与背景

### 过去
帝国的审讯官，专门"审问"叛乱分子。她的审讯技术堪称艺术，从未有人能在她面前保守秘密。她曾以此为傲——直到发现自己审问的很多人根本是无辜的。

### 叛逃经过
某次审讯后，那个被她折磨得奄奄一息的"叛徒"临死前送了她一朵玫瑰，说："不恨你...你只是...被欺骗了..."她崩溃了。调查后发现，她审问过的人有一半以上是被栽赃的无辜者。

**叛逃理由**（良心型）：
> "我以为我在守护帝国。原来我只是...刽子手。那些人的惨叫...每晚都在我耳边回响。"

### 被收留
企图自杀时被舰队救起。她认为自己不配活着，但被指挥官说服"用活着来赎罪"。现在用自己承受伤害的能力保护同伴——把痛苦当作赎罪的方式。

## 六、与指挥官的关系

**关系定位**：等待救赎的罪人、学会被爱的人

### 关系发展

| 阶段 | 好感度 | 互动特点 |
|------|--------|----------|
| **初识** | Lv.1 | 自我厌恶，觉得不配被善待 |
| **熟悉** | Lv.2 | 对指挥官的善意感到困惑 |
| **信任** | Lv.3 | 第一次讲述过去，请求被惩罚 |
| **暧昧** | Lv.4 | "为什么...你不恨我？" |
| **誓约** | Lv.5 | "如果你愿意原谅我...我想为你活着。" |

### 特殊互动
- 会试探性地请求被惩罚
- 被温柔对待时会不知所措
- 保护指挥官时会格外拼命

## 七、关键台词

### 战斗语音
- （开始）「让我...承受痛苦。」
- （攻击）「还给你...」
- （受伤）「啊...这样...才对...」
- （暴击）「痛苦...是我应得的。」
- （波次结束）「...还不够。」
- （胜利）「...我还活着。」
- （失败）「终于...可以结束了...？」

### 日常语音（好感度相关）
- （Lv.1）「...你为什么要对我好？」
- （Lv.2）「我不值得...你的善意。」
- （Lv.3）「我杀过很多人...很多无辜的人。你还愿意看着我吗？」
- （Lv.4）「你的温柔...让我不知道该怎么办。」
- （Lv.5）「谢谢你...让我觉得...也许能被原谅。」

### 特殊语音
- （长时间未登录）「你...终于回来了。我以为...你也放弃我了。」
- （生日）「这是...我能送的最好的东西。一朵没有刺的玫瑰。」

## 十、设计备注

**设计意图**：
- 病态美感角色
- 自我惩罚与救赎的主题
- "荆棘中的玫瑰"意象
- 沉重但最终走向光明的故事

**参考来源**：
- 审讯官/拷问类角色
- 病态美学
- 救赎故事

**设计要点**：
- 立绘体现病态的美感
- 黑红配色为主
- 伤疤和荆棘是重要视觉元素
- 表情要有扭曲与脆弱的交织

**警告**：这是一个较为沉重的角色，设计时注意把握度，最终走向应是救赎与希望。
