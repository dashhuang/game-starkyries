## 三、外形与气质

### 外形要素

| 要素 | 描述 |
|------|------|
| **年龄感** | 19-20岁，与朱璃同龄 |
| **发型** | 长直发，及腰，整齐如瀑 |
| **发色** | 冰蓝渐变银白，像被霜雪覆盖 |
| **瞳色** | 浅蓝近乎透明，像冰湖 |
| **身材** | 纤细清瘦，如冰雕 |
| **服装** | 白蓝配色长裙，雪花暗纹，高领设计 |
| **气质** | 冷若冰霜，不易接近 |
| **特征物品** | 手腕上与朱璃配对的蓝色手环 |

### 画面描述

如果朱璃是燃烧的火焰，玄霜就是凝固的冰晶。冰蓝渐变银白的长发如瀑布般垂落及腰，每一缕都整齐得像是精心雕琢过。浅蓝近乎透明的眼眸像是一潭冰湖，平静无波，深不见底。
白蓝配色的长裙高贵典雅，高领的设计把她裹得严严实实，雪花暗纹在光线下若隐若现。她的皮肤像是从未见过阳光，白皙得近乎透明。手腕上蓝色的手环是她唯一与姐姐朱璃相同的地方——那是姐妹俩的信物。
她很少说话，也很少有表情变化。但如果仔细观察，会发现她的视线总是追随着姐姐，眼底会有转瞬即逝的温柔。

## 四、性格详解

### 表面性格
- 沉默寡言，惜字如金
- 面无表情，难以接近
- 做事有条理，不喜混乱

### 内在性格
- 对姐姐朱璃极度依恋
- 其实很关心别人，只是不知道怎么表达
- 内心细腻，记得每个人的喜好

### 成长弧线
```
初期：只跟着姐姐，对其他人不感兴趣
  ↓
中期：被指挥官的善意触动，开始观察
  ↓
后期：学会用自己的方式表达关心
  ↓
终章：依然话少，但每句话都是真心
```

## 五、来历与背景

### 过去
和双胞胎姐姐朱璃一起在帝国军校长大。与外向的姐姐不同，玄霜从小就沉默寡言，只有在姐姐身边才会稍微放松。她的成绩一直很优秀，但存在感却很低。

### 叛逃经过
姐姐朱璃因为"侮辱皇室"被通缉时，玄霜毫不犹豫地选择一起叛逃。她没有发表任何看法，只是默默收拾好两人的行李。

**叛逃理由**（良心型/亲情型）：
> "姐姐在哪，我就在哪。"

### 被收留
跟着姐姐一起加入指挥官的舰队。全程没说几句话，但行动比任何人都迅速。

## 六、与指挥官的关系

**关系定位**：冰山融化的过程

### 关系发展

| 阶段 | 好感度 | 互动特点 |
|------|--------|----------|
| **初识** | Lv.1 | 无视，只看姐姐的面子才留下 |
| **熟悉** | Lv.2 | 开始用眼神回应，偶尔点头 |
| **信任** | Lv.3 | 会主动递东西，说一两个字 |
| **暧昧** | Lv.4 | 盯着指挥官看的时间变长了 |
| **誓约** | Lv.5 | "我话不多……但你说的，我都记得。" |

### 特殊互动
- 会默默记住指挥官的喜好，然后"碰巧"准备好
- 被夸奖时耳朵会微微泛红
- 姐姐提到指挥官时会格外注意听

## 七、关键台词
