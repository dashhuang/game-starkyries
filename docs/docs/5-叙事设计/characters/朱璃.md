## 三、外形与气质

### 外形要素

| 要素 | 描述 |
|------|------|
| **年龄感** | 19-20岁，青春热血 |
| **发型** | 高马尾，发尾自然卷翘 |
| **发色** | 火焰渐变（根部深红→发尾橙金） |
| **瞳色** | 琥珀金，眼中像有火焰跳动 |
| **身材** | 健康匀称，充满活力 |
| **服装** | 红黑配色战斗服，露出腹部，绑带设计 |
| **气质** | 热情似火，感染力强 |
| **特征物品** | 手腕上与玄霜配对的红色手环 |

### 画面描述

像一团行走的火焰，所到之处都会被她的热情点燃。火焰渐变色的高马尾随着她的动作不停跳跃，从根部的深红渐变到发尾的橙金，像是真正的火舌在舞动。琥珀金的眼眸中似乎真的有火焰在跳动，永远闪烁着热烈的光芒。
一身红黑配色的战斗服利落又张扬，露出结实的腹部——"战斗的时候要凉快点！"她是这么解释的。手腕上那个红色手环是她唯一的装饰品，与妹妹玄霜的蓝色手环是一对。
她笑起来的时候特别灿烂，像是能融化一切的太阳。说话声音大、动作幅度大、情绪波动也大——总之，非常"大"就是了。

## 四、性格详解

### 表面性格
- 热情开朗，自来熟
- 说话直率，不会拐弯抹角
- 大大咧咧，不拘小节

### 内在性格
- 非常重视妹妹玄霜
- 嘴上说不在乎，其实很护短
- 直觉敏锐，大事不糊涂

### 成长弧线
```
初期：觉得舰队是暂时落脚地，随时准备走
  ↓
中期：和大家混熟了，开始护短
  ↓
后期：把舰队当成新的家
  ↓
终章：愿意为这个家付出一切
```

## 五、来历与背景

### 过去
和双胞胎妹妹玄霜一起在帝国军校长大，是校内有名的"冰火姐妹"。姐妹俩性格截然相反，但感情极好。朱璃是那种交友广泛、到哪都能混得开的类型。

### 叛逃经过
一次庆功宴上喝多了，吐槽了一句"皇帝讲话真啰嗦"，被人录了下来。第二天就被以"侮辱皇室"的罪名通缉。她决定跑路时，玄霜二话不说跟着一起走了。

**叛逃理由**（荒诞型）：
> "就说了一句皇帝啰嗦，这也要砍头？行吧，你们开心就好，老娘不伺候了！"

### 被收留
跑路途中遇到指挥官的舰队，见猎心喜要比试一番，打完觉得"还不错"，于是带着妹妹一起加入了。

## 六、与指挥官的关系

**关系定位**：从称兄道弟到脸红心跳

### 关系发展

| 阶段 | 好感度 | 互动特点 |
|------|--------|----------|
| **初识** | Lv.1 | 把指挥官当哥们儿，勾肩搭背 |
| **熟悉** | Lv.2 | 什么话都说，毫无距离感 |
| **信任** | Lv.3 | 发现自己好像有点不一样了 |
| **暧昧** | Lv.4 | 容易脸红，说话开始结巴 |
| **誓约** | Lv.5 | "从今往后，我的火只为你燃烧！" |

### 特殊互动
- 经常拉指挥官去喝酒
- 会炫耀"我妹妹超厉害的"
- 吃醋时会假装若无其事

## 七、关键台词

### 战斗语音
- （开始）「燃起来吧！」
- （攻击）「吃我一发！」
- （受伤）「嘶——有点疼！」
- （暴击）「这火辣不辣？」
- （波次结束）「热身完毕！」
- （胜利）「哈哈哈！痛快！」
- （失败）「可恶……下次一定！」

### 日常语音（好感度相关）
- （Lv.1）「哟！今天也要努力啊！」
- （Lv.2）「晚上一起喝两杯怎么样？」
- （Lv.3）「你知道吗，我妹妹小时候超可爱的……」
- （Lv.4）「那个……你今天、好像特别帅……才没有！」
- （Lv.5）「我这把火啊，以后只许你来点。」

### 特殊语音
- （长时间未登录）「你去哪了！我还以为你跑了呢！」
- （生日）「生日快乐！来来来，姐请你喝酒！」
