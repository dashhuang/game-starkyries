### 画面描述

她站在那里，就像一把出鞘的剑——锋利、冷冽、不可侵犯。纯黑的长发用金属环束成低马尾，利落干练，没有一丝多余。深金色的眼眸锐利如鹰，仿佛能穿透星辰，看到光年之外。
高挑健美的身材被深蓝色紧身飞行服完美包裹，金色的镶边在暗处微微发光。身后的披风被引擎风吹得猎猎作响，平添几分王者气势。腰间挂着一枚小小的家徽——那是她曾经拥有一切的证明，如今只剩下这个。
她很少看人，不是傲慢，而是习惯了仰望星空。在那片星海中，她曾是最耀眼的那颗星。

## 四、性格详解

### 表面性格
- 孤傲冷淡，不爱社交
- 对自己要求极高，对他人也是
- 独来独往，不依赖任何人

### 内在性格
- 背负着沉重的过去
- 对信任的人会付出一切
- 害怕再次失去，所以选择不靠近

### 成长弧线
```
初期：独来独往，拒绝融入
  ↓
中期：指挥官的不放弃让她产生动摇
  ↓
后期：承认自己需要同伴
  ↓
终章：不再是孤独的剑，而是守护的盾
```

## 五、来历与背景

### 过去
索尔家族是独立星系的名门，莱拉是家族最出色的王牌飞行员，被称为"索尔之星"。直到那场政变——她的家族被帝国扶持的傀儡势力屠杀殆尽，只有她一人逃出。

### 叛逃经过
不是叛逃，而是幸存。她被陷害成"叛徒"，在走投无路时坠入星海，被路过的指挥官舰队救起。那是她第一次欠下人情。

**叛逃理由**（政治型）：
> "我没有背叛任何人。是他们背叛了我的家族，背叛了整个星系。"

### 被收留
重伤昏迷时被指挥官救起，醒来后想要离开，却发现已经无处可去。指挥官只说了一句"你可以留下"，没有问任何问题。

## 六、与指挥官的关系

**关系定位**：救命恩人，后来成为唯一信任的人

### 关系发展

| 阶段 | 好感度 | 互动特点 |
|------|--------|----------|
| **初识** | Lv.1 | 欠了人情，但保持距离 |
| **熟悉** | Lv.2 | 还人情的过程中有了更多接触 |
| **信任** | Lv.3 | 终于愿意说出过去 |
| **暧昧** | Lv.4 | 发现自己想留在这个人身边 |
| **誓约** | Lv.5 | "我的剑，为你而战。我的心，只为你跳动。" |

### 特殊互动
- 嘴上说着"只是还人情"，其实早就还清了
- 会默默挡在指挥官身前
- 被夸奖时会别开视线

## 七、关键台词

### 战斗语音
- （开始）「出击。」
- （攻击）「星陨！」
- （受伤）「这种程度……！」
- （暴击）「尘归尘，土归土。」
- （波次结束）「下一个。」
- （胜利）「不值一提。」
- （失败）「可恶……我还不能倒下……」

### 日常语音（好感度相关）
- （Lv.1）「有什么事就说，别浪费时间。」
- （Lv.2）「你总是出现在奇怪的时候……算了。」
- （Lv.3）「我的过去……你真的想知道？」
- （Lv.4）「我不需要任何人……至少我曾经这么以为。」
- （Lv.5）「能遇见你，是星空给我最后的礼物。」

### 特殊语音
- （长时间未登录）「……你最好有个好理由解释你去了哪。」
- （生日）「这种日子……总之，生日快乐。只此一次。」
