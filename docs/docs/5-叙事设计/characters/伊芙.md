## 三、外形与气质

### 外形要素

| 要素 | 描述 |
|------|------|
| **年龄感** | 22-24岁（真实年龄不详） |
| **发型** | 不对称短发，左长右短，遮住左眼 |
| **发色** | 银灰色，带紫色挑染 |
| **瞳色** | 异色瞳：左眼紫色、右眼金色 |
| **身材** | 纤细灵活，间谍体型 |
| **服装** | 黑色紧身潜行服，可变形多功能外套 |
| **气质** | 神秘多变，让人捉摸不透 |
| **特征物品** | 左手腕的全息投影仪、颈部的假面项链 |

### 画面描述

二十三四岁的女子——或者说，这是她今天选择展示的年龄。银灰色的不对称短发左长右短，左侧刘海遮住了那只紫色的眼睛，只露出右眼的金色瞳孔。异色瞳是她唯一没法伪装的特征，也是她唯一的"真实"。

身穿黑色紧身潜行服，外面套着一件可变形的多功能外套——它可以在瞬间变成晚礼服、军装、或者平民衣物。左手腕上的全息投影仪能投射出任何她想要的面容。颈间挂着一条项链，吊坠是一个微型假面，据说是她第一次任务时的纪念品。

她的表情总是带着若有若无的微笑，让人分不清是真是假。站姿、语气、甚至走路的方式都可以随时切换，仿佛她体内住着无数个不同的人格。

整个人就像一面镜子，只会反射别人想看到的东西。但偶尔，在独处时，那双异色的眼睛里会流露出一丝迷茫——她自己都不记得真正的自己是什么样子了。

## 四、性格详解

### 表面性格
- 善变多端，每次出现都像不同的人
- 说话真真假假，让人分不清
- 喜欢观察和分析他人
- 对任何事都像在演戏

### 内在性格
- 已经忘记"真正的自己"是什么
- 内心深处渴望被真实地接受
- 对信任这种东西既渴望又恐惧
- 其实很孤独，但习惯了

### 成长弧线
```
初期：戴着面具与所有人相处
  ↓
中期：被指挥官看穿伪装，开始动摇
  ↓
后期：尝试展露真实的一面
  ↓
终章：找回"伊芙"这个身份
```

## 五、来历与背景

### 过去
帝国情报部的顶级特工，代号"千面"。从小被情报部培养，接受过无数次身份重塑训练，精通一百多种人格伪装。执行过无数高难度渗透任务，从未失败。是帝国最锋利的"暗刃"。

### 叛逃经过
在一次任务中，她无意间发现了情报部的"弃子计划"——所有特工在任务价值耗尽后都会被"处理"。她就在下一批名单上。更讽刺的是，处理她的命令是她自己最信任的上司下达的。

**叛逃理由**（政治型）：
> "我为他们演了一辈子的戏，结果连谢幕都不让我知道。呵...原来对他们来说，我也只是一个角色。"

### 被收留
假死逃脱后被舰队发现。起初没人信任她——毕竟她是间谍。但她用行动证明了自己，虽然大家还是不确定看到的是不是真正的她。

## 六、与指挥官的关系

**关系定位**：唯一能看穿她的人、找回自我的锚点

### 关系发展

| 阶段 | 好感度 | 互动特点 |
|------|--------|----------|
| **初识** | Lv.1 | 各种人格轮番登场，测试指挥官 |
| **熟悉** | Lv.2 | 被看穿伪装，开始有点慌 |
| **信任** | Lv.3 | 第一次用真实的声音说话 |
| **暧昧** | Lv.4 | 不知道怎么真诚地表达感情 |
| **誓约** | Lv.5 | "你是第一个...想要看到真正的我的人。" |

### 特殊互动
- 出场时人格随机
- 被看穿伪装会难得地不知所措
- 偶尔会透露真实的想法，然后装作在开玩笑

## 七、关键台词

### 战斗语音
- （开始）「今天想看哪一面的我？」
- （攻击）「——」（声音每次都略有不同）
- （受伤）「只是演技...」
- （暴击）「看穿你了。」
- （波次结束）「变换——」
- （胜利）「完美演出。」
- （失败）「...这次，不是在演戏。」

### 日常语音（好感度相关）
- （Lv.1）「你好呀~我是伊芙！...还是说，你想叫我别的名字？」
- （Lv.2）「你在试图分辨哪个是真的我？呵，我自己都不知道。」
- （Lv.3）「...这是我很久没用过的声音了。只给你听。」
- （Lv.4）「我不知道怎么真诚地喜欢一个人...你能教我吗？」
- （Lv.5）「不用面具的我，很无趣吧？......你还愿意看着我啊。」

### 特殊语音
- （长时间未登录）「你是去找别的"伊芙"了吗？...开玩笑的。」
- （生日）「这是真正的我准备的礼物。不是任何角色，是伊芙。」

## 十、设计备注

**设计意图**：
- 神秘间谍类型
- "寻找真实自我"的主题
- 异色瞳作为唯一"真实"的象征
- 与指挥官的关系是"被看穿"后的真诚

**参考来源**：
- 间谍/变装类角色
- 多重人格
- 身份认同议题

**设计要点**：
- 立绘体现神秘多变的气质
- 异色瞳是最重要的标识
- 不对称设计（发型、服装）
- 表情要有"真假难辨"的感觉
- 可以设计多个"人格"版本的表情
