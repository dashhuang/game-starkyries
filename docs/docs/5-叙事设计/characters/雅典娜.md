## 三、外形与气质

### 外形要素

| 要素 | 描述 |
|------|------|
| **年龄感** | 23-25岁，巅峰状态 |
| **发型** | 银白色短发，向后梳起 |
| **发色** | 银白色，如同月光 |
| **瞳色** | 金色，傲视一切 |
| **身材** | 健美有力，战士体型 |
| **服装** | 古希腊风格战甲，胸甲+短裙+护腿 |
| **气质** | 王者霸气，睥睨天下 |
| **特征物品** | 冠军腰带、手臂上的角斗纹身 |

### 画面描述

二十四五岁的女战士，浑身散发着不可一世的王者霸气。银白色短发向后梳起，露出光洁的额头和锐利的金色眼眸。那双眼睛里仿佛写着"我是最强的"，让人不敢直视。

身穿古希腊风格的战甲——金色胸甲贴合着健美的身躯，腰间是白色短裙，小腿包裹着同色护腿。不是华而不实的装饰，而是真正经历过无数战斗的铠甲。

身材健美有力，肌肉线条明显但不失女性柔美。腰间系着一条华丽的冠军腰带——那是她在星际角斗场一百连胜后获得的荣誉。右臂上有一圈角斗纹身，记录着她击败过的对手数量。

整个人如同从神话中走出的战神，高傲、强大、不可战胜。

## 四、性格详解

### 表面性格
- 骄傲自负，目中无人
- 好胜心极强，不接受失败
- 直来直去，讨厌废话
- 对弱者不屑一顾

### 内在性格
- 骄傲背后是童年被轻视的自卑
- 渴望被认可，所以不停战斗
- 对真正认可的人会展现温柔
- 其实很在意同伴，只是不说

### 成长弧线
```
初期：目中无人，只相信自己
  ↓
中期：被团队战打败，开始反思
  ↓
后期：学会团队配合的价值
  ↓
终章：从"孤独的王者"变成"战队的核心"
```

## 五、来历与背景

### 过去
出生在边境殖民地的贫民窟，从小被人欺负。偶然发现自己有战斗天赋后，进入地下角斗场，一路打到星际竞技场。百连胜的不败战绩让她成为传说，被称为"角斗女神"。

### 叛逃经过
帝国想要招安她成为宣传工具，被她拒绝。帝国于是在她的比赛中做手脚，想要打破她的不败记录。她识破阴谋，当场打倒裁判和幕后黑手，然后被全星系通缉。

**叛逃理由**（良心型+骄傲）：
> "想让我假输？开什么玩笑！我的胜利是靠自己打出来的，不是他们能给的！"

### 被收留
被通缉后一个人打退了三波追兵，但最终被舰队的火力覆盖逼到绝路。与其说是被收留，不如说是被"打服"后才愿意加入。

## 六、与指挥官的关系

**关系定位**：被征服的女王、傲娇的追随者

### 关系发展

| 阶段 | 好感度 | 互动特点 |
|------|--------|----------|
| **初识** | Lv.1 | 完全不服，想要单挑 |
| **熟悉** | Lv.2 | 被战术击败，开始不情愿地承认 |
| **信任** | Lv.3 | 承认指挥官的实力，愿意听从指挥 |
| **暧昧** | Lv.4 | 极度傲娇，明明在意却死不承认 |
| **誓约** | Lv.5 | "你是唯一...打败过我的人。所以，你要负责。" |

### 特殊互动
- 赢了会来炫耀，输了好几天不说话
- 叫指挥官一起训练时是在表达亲近
- 被夸奖时会傲娇地说"这不是当然的吗"

## 七、关键台词

### 战斗语音
- （开始）「让本王看看你们的实力！」
- （攻击）「就这？」
- （受伤）「……有点意思。」
- （暴击）「跪下！」
- （波次结束）「不过如此。」
- （胜利）「哼，又是本王的胜利！」
- （失败）「不可能...本王怎么可能...！」

### 日常语音（好感度相关）
- （Lv.1）「你就是指挥官？来，和本王单挑！」
- （Lv.2）「哼，本王暂时认可你的指挥能力。」
- （Lv.3）「有你在...打起来确实更顺手。」
- （Lv.4）「才、才不是在等你！本王只是...刚好路过！」
- （Lv.5）「只有你...让本王不想再战斗了。」

### 特殊语音
- （长时间未登录）「你去哪了！本王的对手不够！」
- （生日）「这个冠军腰带...借给你。不是送！是借！」

## 十、设计备注

**设计意图**：
- 高傲女王类型，"被攻略"的满足感
- 从自卑到自负的成长背景
- 百连胜的传说感
- 傲娇程度拉满

**参考来源**：
- 格斗游戏的女王角色
- 古希腊战神形象
- 傲娇女王类型

**设计要点**：
- 立绘体现王者霸气
- 银发金眼的强者配色
- 姿态要有居高临下的感觉
- 冠军腰带是重要标识
