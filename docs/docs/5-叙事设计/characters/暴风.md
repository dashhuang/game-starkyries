## 三、外形与气质

### 外形要素

| 要素 | 描述 |
|------|------|
| **年龄感** | 25-27岁，成熟野性 |
| **发型** | 狂野红发，自然卷曲蓬松 |
| **发色** | 火焰般的红色 |
| **瞳色** | 琥珀色，带着野性光芒 |
| **身材** | 健壮丰满，力量型体格 |
| **服装** | 海盗风皮甲，露肩短款，腰带挂满战利品 |
| **气质** | 狂野不羁，自由奔放 |
| **特征物品** | 左眼眼罩（装饰用，眼睛没事）、各种战利品 |

### 画面描述

二十五六岁的女海盗，浑身上下写满了"自由"二字。火焰般的红色卷发狂野地披散在肩头，像是永不熄灭的烈火。琥珀色的眼眸里闪烁着野性的光芒，嘴角总是挂着不羁的笑。

左眼戴着一个黑色眼罩——其实眼睛完好无损，只是觉得"这样比较有海盗范儿"。身穿棕色皮甲，露出一边结实的肩膀和手臂，上面有几道刀疤。腰带上挂满了各种小物件：敌人的徽章、奇怪的小饰品、不知道从哪抢来的金币。

身材健壮丰满，是力量型的体格，却不失女性的曲线。整个人散发着大海和烈酒的气息，让人想起狂风巨浪中的海盗旗。

她总是站得很随意，笑得很张扬，活得很自在。

## 四、性格详解

### 表面性格
- 野性不羁，不受任何规则约束
- 豪爽仗义，有福同享有难同当
- 大大咧咧，不拘小节
- 嗜酒如命，经常喝醉

### 内在性格
- 重情重义，为兄弟可以豁出命
- 厌倦了无意义的杀戮和抢劫
- 渴望找到值得为之战斗的东西
- 其实很孤独，失去过太多伙伴

### 成长弧线
```
初期：无法无天，把舰队当临时落脚点
  ↓
中期：开始把同伴当做新的"船员"
  ↓
后期：找到了新的"船长"（指挥官）
  ↓
终章：从海盗变成守护者
```

## 五、来历与背景

### 过去
暴风海盗团的首领，在星际航道上劫掠了十多年。手下有上百号人，是帝国通缉榜上有名有姓的大海盗。年轻时是为了生存，后来是为了自由，再后来...只是习惯了。

### 叛逃经过
某天醒来，发现自己已经厌倦了这种生活。抢劫商船时看到船员惊恐的眼神，突然觉得索然无味。她解散了海盗团，给每个手下分了金子，自己开着船离开了。

**叛逃理由**（良心型+厌倦）：
> "当海盗当腻了。以前觉得自由，现在觉得...空虚。该换个活法了。"

### 被收留
流浪时被舰队捡到。本来只是想搭个顺风船，结果发现这群人比海盗还有意思。就这么留下来了。

## 六、与指挥官的关系

**关系定位**：找到新船长的海盗、豪爽的大姐头

### 关系发展

| 阶段 | 好感度 | 互动特点 |
|------|--------|----------|
| **初识** | Lv.1 | 没大没小，称兄道弟 |
| **熟悉** | Lv.2 | 开始认可指挥能力，愿意听令 |
| **信任** | Lv.3 | 把指挥官当新的"船长" |
| **暧昧** | Lv.4 | 变得有点扭捏，不像平时 |
| **誓约** | Lv.5 | "以后我只为你一个人劫掠星辰！" |

### 特殊互动
- 经常拉指挥官喝酒
- 会分享"战利品"（奇奇怪怪的东西）
- 喝醉后会说心里话

## 七、关键台词

### 战斗语音
- （开始）「野蛮号出击！都给老娘抢光！」
- （攻击）「吃我一炮！」
- （受伤）「就这点程度？」
- （暴击）「宝贝们，都是老娘的了！」
- （波次结束）「战利品不错！」
- （胜利）「哈哈哈！今晚喝酒庆祝！」
- （失败）「可恶...下次定要抢回来！」

### 日常语音（好感度相关）
- （Lv.1）「哟，小指挥官！来喝一杯？」
- （Lv.2）「你这人不错，有当船长的潜质。」
- （Lv.3）「以前的事就不提了...现在我是你的人了。」
- （Lv.4）「那个...今晚月色不错...要不要...唉算了！」
- （Lv.5）「这辈子只跟你一个人了，不许反悔！」

### 特殊语音
- （长时间未登录）「船长不见了？不会是被抢走了吧！」
- （生日）「这个是我最珍贵的战利品，现在送你了！」
