## 三、外形与气质

### 外形要素

| 要素 | 描述 |
|------|------|
| **年龄感** | 22-24岁，成熟优雅 |
| **发型** | 黑色直长发，以白色发带束于脑后 |
| **发色** | 墨黑色，如同夜幕 |
| **瞳色** | 淡青色，清澈如水 |
| **身材** | 修长纤细，剑士身形 |
| **服装** | 改良中式剑服，白衣青纹 |
| **气质** | 如青竹般清雅出尘 |
| **特征物品** | 白色发带、背后的细长剑鞘 |

### 画面描述

二十三四岁的女子，周身萦绕着如水般平静的气息。墨黑色的长发以白色发带整齐地束在脑后，没有一丝杂乱。淡青色的眼眸像是山间清泉，平静中藏着锋利。

身穿改良的中式剑服——白色为底，青色纹路如同竹叶，衣袂飘飘却不妨碍动作。身形修长纤细，是标准的剑士体型，看似柔弱实则力量惊人。

背后斜挎着一柄细长的剑鞘，剑柄缠着与发带同色的白布。她的动作总是从容不迫，仿佛无论发生什么都乱不了她的节奏。

整个人如同一竿青竹，优雅、坚韧、不染尘埃。

## 四、性格详解

### 表面性格
- 优雅从容，说话轻声细语
- 追求完美，对自己和他人要求都高
- 淡泊名利，不争不抢
- 有礼有节，但保持距离

### 内在性格
- 对剑道有近乎执着的热爱
- 其实很不服输，只是不表现出来
- 曾经也是争强好胜的人，被挫折磨平
- 内心渴望找到真正的对手

### 成长弧线
```
初期：淡然处世，不太融入团队
  ↓
中期：被同伴们的热情感染，开始打开心扉
  ↓
后期：重新燃起对"最强"的追求
  ↓
终章：找到了剑道的真谛——为守护而战
```

## 五、来历与背景

### 过去
帝国军校剑道教官，被称为"帝国之刃"。在剑道大赛中从未败过，是当之无愧的第一。她的剑道被视为艺术，无数人慕名求教。

### 叛逃经过
在帝国剑道锦标赛决赛中，对手是某位高官的儿子。赛前被暗示"让一让"，她拒绝了，并以完美的技术获胜。赛后被以"比赛中使用禁止技术"为由取消成绩，还被革除教职。

**叛逃理由**（荒诞型）：
> "他们说我犯规。我用的是最基础的直刺，哪里犯规了？...哦，对他们来说，赢了就是犯规。"

### 被收留
独自隐居时被舰队找到。起初不愿加入，后来被指挥官"以棋论剑"的方式打动，决定加入看看这里能否让她找到新的剑道。

## 六、与指挥官的关系

**关系定位**：知音、以剑相交的伴侣

### 关系发展

| 阶段 | 好感度 | 互动特点 |
|------|--------|----------|
| **初识** | Lv.1 | 礼貌疏离，客套敷衍 |
| **熟悉** | Lv.2 | 发现指挥官懂"道"，开始交流 |
| **信任** | Lv.3 | 愿意展示真实的自己 |
| **暧昧** | Lv.4 | 会为指挥官泡茶，邀请赏月 |
| **誓约** | Lv.5 | "此剑为君而鸣，此心为君而动。" |

### 特殊互动
- 会邀请指挥官下棋或品茶
- 受伤后更在意剑有没有缺口
- 看到美丽的事物会难得地微笑

## 七、关键台词

### 战斗语音
- （开始）「利刃出鞘。」
- （攻击）「一。」「二。」「三。」（连击计数）
- （受伤）「...不影响。」

- （胜利）「承让。」

- （Lv.1）「有何贵干？」

- （Lv.3）「难得...遇到懂的人。」
- （Lv.4）「今夜月色不错...要一起赏月吗？」
- （Lv.5）「遇到你之后，我的剑有了新的意义。」

### 特殊语音
- （长时间未登录）「你终于来了...茶已经凉了三次。」
- （生日）「这柄剑送你。不要误会，只是...觉得适合你。」

## 十、设计备注

**设计意图**：
- 优雅剑客类型，带来东方美学
- "比赛没让"的叛逃理由荒诞又讽刺
- 追求完美但内心寂寞
- 与指挥官是"知音"的关系

**参考来源**：
- 剑道/武侠美学
- 追求完美的艺术家类型
- 竹子的意象（青竹、气节）

**设计要点**：
- 立绘体现东方剑客的优雅
- 白衣青纹的配色
- 动作要有"静中带动"的感觉
- 发带和剑是重要标识
