## 三、外形与气质

### 外形要素

| 要素 | 描述 |
|------|------|
| **年龄感** | 26-28岁，饱经沧桑 |
| **发型** | 灰白色长直发，随意披散 |
| **发色** | 灰白色，像是被抽干了生气 |
| **瞳色** | 冰蓝色，毫无感情 |
| **身材** | 高挑消瘦，有种病态的美 |
| **服装** | 黑色长款军装大衣，内搭高领 |
| **气质** | 死寂冰冷，如同行走的尸体 |
| **特征物品** | 从不摘下的黑色手套 |

### 画面描述

二十七八岁的女子，周身萦绕着死亡的气息。灰白色的长发随意披散，像是过早苍老的证明。冰蓝色的眼眸空洞得吓人，仿佛里面装着无数亡魂的影子。

身穿黑色长款军装大衣，遮住了大部分身形。高挑却消瘦，有一种病态的美感。手上永远戴着黑色手套——据说是为了遮住洗不掉的血。

她很少说话，说话时声音也很轻，像是怕惊醒什么。面无表情是常态，偶尔露出的情绪都是自嘲和厌恶。整个人就像一把用过太多次的刀，锋利、冰冷、充满杀意，却也伤痕累累。

## 四、性格详解

### 表面性格
- 冷酷无情，对一切都漠不关心
- 沉默寡言，能不说话就不说
- 执行任务时毫不手软
- 对生死看得很淡

### 内在性格
- 深深的自我厌恶
- 被噩梦缠绕，很少能安睡
- 认为自己不配被原谅
- 渴望赎罪，却不知道该怎么做

### 成长弧线
```
初期：自我放弃，只是在等死
  ↓
中期：被同伴的善意触动，开始动摇
  ↓
后期：决定用剩下的生命赎罪
  ↓
终章：学会与过去和解，重新活着
```

## 五、来历与背景

### 过去
帝国"清洗部队"的指挥官，专门负责"处理"不稳定因素。在她手下，无数"叛逆者"化为灰烬。她执行命令时毫不犹豫，被称为"帝国的毁灭者"。直到有一天，她发现自己清洗的很多人都是无辜的。

### 叛逃经过
执行完一次大规模清洗后，她知道了太多"不该知道的"。帝国决定把她也"清洗"掉——灭口。曾经最忠诚的执行者，转眼变成了被清洗的对象。

**叛逃理由**（政治型）：
> "我为帝国杀了那么多人，最后帝国要杀我。呵...也好。但我不会让他们得逞。"

### 被收留
逃亡途中被舰队救起。她不在乎被救，只是觉得"死在哪里都一样"。现在留在舰队，把每一次战斗都当作赎罪。

## 六、与指挥官的关系

**关系定位**：等待救赎的罪人、找回活着意义的人

### 关系发展

| 阶段 | 好感度 | 互动特点 |
|------|--------|----------|
| **初识** | Lv.1 | 漠然，觉得没必要亲近 |
| **熟悉** | Lv.2 | 对指挥官的善意感到困惑 |
| **信任** | Lv.3 | 第一次主动说起过去 |
| **暧昧** | Lv.4 | "你为什么...对我这么好？" |
| **誓约** | Lv.5 | "你让我觉得...也许还能活下去。" |

### 特殊互动
- 经常做噩梦，有时能听到她的呓语
- 会悄悄帮忙做事，不让别人发现
- 难得露出笑容时会很不自在

## 十、设计备注

**设计意图**：
- 黑暗过去的赎罪者类型
- 自我厌恶但被救赎的故事线
- "帝国也会清洗自己人"的讽刺
- 冰冷外表下的脆弱

**参考来源**：
- 刽子手类角色
- 赎罪者故事
- PTSD主题

**设计要点**：
- 立绘体现死寂和空洞
- 灰白色调为主
- 表情几乎没有起伏
- 黑色手套是重要细节
