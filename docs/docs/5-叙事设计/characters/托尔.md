## 三、外形与气质

### 外形要素

| 要素 | 描述 |
|------|------|
| **年龄感** | 22-24岁，英气勃发 |
| **发型** | 金色短发，根根竖起如雷电 |
| **发色** | 闪电金色，带电光效果 |
| **瞳色** | 电蓝色，激动时会闪烁 |
| **身材** | 高大健美，战士体型 |
| **服装** | 部落风格战甲配现代科技，雷纹装饰 |
| **气质** | 雷厉风行，如同风暴 |
| **特征物品** | 雷神锤造型的权杖、部落图腾项链 |

### 画面描述

二十三四岁的部落战士，浑身散发着风暴的气息。金色的短发根根竖起，像是被雷电劈过一般，在光线下闪烁着电光。电蓝色的眼眸充满力量，激动时会闪烁如同闪电。

身穿部落风格的战甲，融合了现代科技元素。皮革和金属的结合上刻满雷纹图案，是雷霆部落的标志。身材高大健美，是标准的战士体型，肌肉线条明显。

手持一柄雷神锤造型的权杖——那是她作为酋长之女的象征。颈间挂着部落图腾项链，上面刻着雷霆部落的神话。整个人就像是从风暴云中降临的雷神，威武、磅礴、势不可挡。

## 四、性格详解

### 表面性格
- 雷厉风行，做事果断
- 豪迈直爽，有话直说
- 重视荣誉，说到做到
- 热血好战，喜欢挑战

### 内在性格
- 对失去部落深感自责
- 把恢复部落荣耀当做使命
- 其实很重感情，把同伴当家人
- 对帝国有着深深的仇恨

### 成长弧线
```
初期：满心复仇，一心想要打回去
  ↓
中期：开始把舰队当做新的部落
  ↓
后期：理解"荣耀"不只是战斗
  ↓
终章：从复仇者变成守护者
```

## 五、来历与背景

### 过去
雷霆部落酋长之女，从小被当做下一任酋长培养。雷霆部落是边境最强大的独立势力之一，世代与自然共存，信仰雷神。托尔是部落最强的战士，被誉为"雷神的祝福"。

### 叛逃经过
帝国为了吞并边境，以"文明教化"为名强行吸收雷霆部落。托尔的父亲——老酋长拒绝投降，在抵抗中战死。部落被强制解散，族人被"重新安置"（实际上是奴役）。托尔是唯一逃出来的战士。

**来历**（独立势力）：
> "帝国毁了我的部落，杀了我的父亲。他们叫这'文明'？我会让他们知道什么是雷霆的愤怒！"

### 被收留
独自游击帝国军队时被舰队发现。起初只是借用舰队的力量复仇，后来把舰队当做了新的部落、新的家人。

### 关系发展

| 阶段 | 好感度 | 互动特点 |

| **誓约** | Lv.5 | "按雷霆部落的传统，我属于你了！" |

### 特殊互动

- 用部落方式庆祝胜利
- 送的礼物都是自己打猎/做的

## 十、设计备注

**设计意图**：
- 雷神战士类型
- "托尔"致敬北欧雷神
- 部落文化与现代的碰撞
- 复仇到守护的转变

**参考来源**：
- 北欧神话雷神
- 部落战士
- 雷电意象

**设计要点**：
- 立绘体现雷电的力量感
- 金色电蓝配色为主
- 头发要有"被电过"的感觉
- 雷神锤权杖是重要标识
