## 三、外形与气质

### 外形要素

| 要素 | 描述 |
|------|------|
| **年龄感** | 24-26岁，沉稳内敛 |
| **发型** | 深紫色长发，低马尾 |
| **发色** | 深紫色，夜色般深邃 |
| **瞳色** | 银色，如月光般清冷 |
| **身材** | 修长纤细，狙击手体型 |
| **服装** | 深灰色狙击手披风，内搭紧身战斗服 |
| **气质** | 安静如夜，致命如月 |
| **特征物品** | 从不离身的狙击镜、月牙形发饰 |

### 画面描述

二十五六岁的女子，整个人像是融入了夜色。深紫色的长发束成低马尾，发尾垂至腰际。银色的眼眸如同月光，清冷却不刺眼，总是带着审视的目光观察周围。

身穿深灰色的狙击手披风，风吹过时像是夜色的波浪。内搭黑色紧身战斗服，方便隐蔽和移动。身材修长纤细，是标准的狙击手体型——不需要力量，需要的是稳定和耐心。

发间别着一枚月牙形的银色发饰，是她母亲留下的遗物。腰间挂着她的狙击镜——虽然舰战用不上，但这是她最重要的"伙伴"。

她很少说话，但眼神会说话。安静得像是夜晚的月亮，让人觉得遥远却又无法忽视。

## 四、性格详解

### 表面性格
- 沉默寡言，话少但句句在点
- 冷静耐心，能等待很长时间
- 不善交际，但不是排斥
- 观察力极强，什么都看在眼里

### 内在性格
- 内心温柔，会暗中照顾同伴
- 对无辜者有保护欲
- 曾经因为任务杀错人而自责
- 渴望被理解但不知道怎么表达

### 成长弧线
```
初期：沉默独行，习惯一个人
  ↓
中期：开始融入团队，话变多了一点点
  ↓
后期：学会表达自己的想法
  ↓
终章：从"孤独的猎人"变成"同伴的守护者"
```

## 五、来历与背景

### 过去
帝国最顶级的狙击手，执行过无数暗杀任务，从未失手。被称为"帝国之月"，是令敌人闻风丧胆的存在。她的子弹只会在目标最放松的时候到来，就像月光一样无声无息。

### 叛逃经过
某次任务目标是一个"叛国者"的家属——一个老人和一个孩子。她的情报显示目标是两个成年男性。当瞄准镜里出现那张苍老的脸和孩子惊恐的眼睛时，她第一次扣不下扳机。

**叛逃理由**（良心型）：
> "我是狙击手，不是刽子手。目标错了。...或者说，从一开始就是要我杀这两个人。"

### 被收留
放弃任务后被追杀，在逃亡中被舰队救起。现在用她的技术保护同伴，再也不用瞄准无辜的人。

## 六、与指挥官的关系

**关系定位**：沉默的守护者、用行动表达感情的人

### 关系发展

| 阶段 | 好感度 | 互动特点 |

| **暧昧** | Lv.4 | 视线总是追随着指挥官 |
| **誓约** | Lv.5 | "从今以后...我的瞄准镜只为你而存在。" |

### 特殊互动
- 会默默出现在指挥官身边
- 送礼物时不说话直接放下就走
- 看星星时会难得地放松

## 十、设计备注

**设计意图**：
- 沉默狙击手类型
- "戴安娜"致敬月亮女神和猎手
- 沉默但温柔的反差
- 用行动而非语言表达感情

**参考来源**：
- 狙击手类角色
- 月亮和夜晚的意象
- 沉默寡言类型

**设计要点**：
- 立绘体现安静和致命
- 银紫色调为主
- 银色眼眸是重点
- 月牙发饰是重要标识
