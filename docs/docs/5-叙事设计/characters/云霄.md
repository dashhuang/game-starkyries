## 三、外形与气质

### 外形要素

| 要素 | 描述 |
|------|------|
| **年龄感** | 24-26岁，成熟稳重 |
| **发型** | 低马尾，整齐利落 |
| **发色** | 深棕色，沉稳内敛 |
| **瞳色** | 深灰色，如同阴天的天空 |
| **身材** | 高挑健美，军人体态 |
| **服装** | 深灰色军装外套，内搭黑色高领 |
| **气质** | 沉稳可靠，值得信赖的军人气质 |
| **特征物品** | 左胸口别着的银色猎鹰徽章 |

### 画面描述

二十五六岁的女军官，周身散发着令人安心的沉稳气息。深棕色长发整齐地束成低马尾，没有一丝碎发，体现着她一丝不苟的性格。深灰色的眼眸像是阴天的苍穹，平静中藏着风暴的力量。

身材高挑健美，标准的军人体态。深灰色的军装外套剪裁得体，内搭黑色高领衫，整洁得像是随时准备受阅。左胸口别着一枚银色猎鹰徽章——那是她带领中队击退海盗突袭时获得的荣誉。

站姿笔直如松，目光平和却锐利。她是那种一眼看去就让人觉得"这个人很可靠"的类型，沉默寡言却在关键时刻绝不退缩。

## 四、性格详解

### 表面性格
- 寡言少语，惜字如金
- 冷静自持，从不慌张
- 服从命令，执行力强
- 待人有礼但保持距离

### 内在性格
- 内心有着强烈的正义感
- 对无辜者有难以抑制的保护欲
- 曾因服从命令做错事而自责
- 渴望找到值得效忠的对象

### 成长弧线
```
初期：沉默寡言，只执行任务
  ↓
中期：开始主动表达意见，参与决策
  ↓
后期：成为舰队的战术核心，被信任
  ↓
终章：找到了真正值得守护的东西
```

## 五、来历与背景

### 过去
帝国边境舰队中校，带领猎鹰中队守护边境多年。战功赫赫，却始终保持低调。她的中队是边境最可靠的防线，当地居民视她为守护神。

### 叛逃经过
某次接到命令，要求"清理"一个被判定为"通敌"的边境村庄。到达现场后发现所谓"通敌"不过是村民给路过的难民分了些食物。她拒绝执行命令，率领中队护送村民撤离。

**叛逃理由**（良心型）：
> "他们只是善良而已。如果善良也是罪，那我不知道自己在守护什么。"

### 被收留
带着部分愿意追随的部下逃离帝国，辗转被母舰接纳。她的战术素养和作战经验让她迅速成为舰队的战术骨干。

## 六、与指挥官的关系

**关系定位**：可靠的副手、沉默的守护者

### 关系发展

| 阶段 | 好感度 | 互动特点 |
|------|--------|----------|
| **初识** | Lv.1 | 恪守上下级关系，保持距离 |
| **熟悉** | Lv.2 | 开始主动汇报，信任建立 |
| **信任** | Lv.3 | 讲述过去，征询意见 |
| **暧昧** | Lv.4 | 话变多了，会不自觉地靠近 |
| **誓约** | Lv.5 | "这一次，让我为真正重要的人而战。" |

### 特殊互动
- 战术建议总是精准有效
- 会默默把指挥官的杯子添满
- 看到指挥官疲惫时会建议休息

## 十、设计备注

**设计意图**：
- 可靠的军人类型，给舰队带来安定感
- 寡言但行动表达一切
- 叛逃理由体现军人的两难
- 与指挥官的关系是"沉默的信任"

**参考来源**：
- 传统军人角色
- 沉默寡言的守护者类型
- 武侠中的"剑客"气质

**设计要点**：
- 立绘体现军人的端正气质
- 表情平静内敛
- 站姿笔直标准
- 猎鹰徽章是重要标识
