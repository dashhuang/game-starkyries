## 三、外形与气质

### 外形要素

| 要素 | 描述 |
|------|------|
| **年龄感** | 19-21岁，疯狂少女 |
| **发型** | 橙红色炸毛短发，像爆炸后的样子 |
| **发色** | 橙红渐变黄色，像火焰 |
| **瞳色** | 金色，闪烁着疯狂的光 |
| **身材** | 娇小活泼，总是动个不停 |
| **服装** | 白色实验服（烧焦痕迹）配短裤 |
| **气质** | 疯狂热情，像一颗随时会炸的炸弹 |
| **特征物品** | 护目镜（架在额头）、腰带上挂满小型爆炸物 |

### 画面描述

十九二十岁的少女，整个人就像一颗行走的炸弹。橙红色的短发炸得像刚从爆炸现场出来，发梢渐变成明亮的黄色，像是燃烧的火焰。金色的眼眸里闪烁着狂热的光芒，看什么都像在计算"怎么炸掉它"。

身穿白色实验服，上面布满烧焦的痕迹和各种污渍——那是无数次"成功实验"的勋章。下身是方便活动的短裤，脚上是防爆靴。护目镜永远架在额头上，只有实验时才会戴上。

腰带上挂满了各种小型装置，她说那是"样品"，别人说那是"定时炸弹"。她走路带风，说话带笑，笑声里带着"哈哈哈这个爆炸肯定很漂亮"的疯狂。

整个人就是一个不稳定的化学反应，随时可能创造美丽的烟火——或者灾难。

## 四、性格详解

### 表面性格
- 疯狂热情，对爆炸有着近乎病态的热爱
- 话多且快，思维跳跃
- 乐观到有点傻，什么都往好处想
- 完全不在意别人的眼光

### 内在性格
- 其实是天才，只是表达方式特别
- 对科学有纯粹的热爱
- 内心善良，从不伤害无辜
- 对被认可这件事很在意

### 成长弧线
```
初期：只关心爆炸，不管其他
  ↓
中期：开始考虑"为什么而炸"
  ↓
后期：学会控制自己的天赋
  ↓
终章：从"疯狂科学家"变成"可靠的爆破专家"
```

## 五、来历与背景

### 过去
帝国研究所的爆破专家，年纪轻轻就是首席研究员。她研发的爆炸物效率是普通产品的十倍，但"附带损害"也是十倍。研究所又爱又怕她——爱她的天才，怕她的疯狂。

### 叛逃经过
某次实验"稍微"失控，炸毁了半个研究所。虽然没有人员伤亡（她提前疏散了所有人），但造成的损失足以让她被判处终身监禁。更重要的是，她发现自己的研究被用来制造屠杀平民的武器。

**叛逃理由**（荒诞型+良心型）：
> "炸研究所是意外！...好吧不完全是意外。但他们用我的配方杀平民才是重点！爆炸应该是美丽的，不是用来杀人的！"

### 被收留
逃跑时顺手炸了追兵的车队（"顺便测试新配方"），被舰队发现。她的爆破技术正是舰队需要的——只要别把母舰也炸了。

## 六、与指挥官的关系

**关系定位**：需要人管的疯狂天才、纯真的爱慕者

### 关系发展

| 阶段 | 好感度 | 互动特点 |
|------|--------|----------|
| **初识** | Lv.1 | 热情过头，想让指挥官看她的"作品" |
| **熟悉** | Lv.2 | 开始听指挥官的话（有时候） |
| **信任** | Lv.3 | 分享她对科学的纯粹热爱 |
| **暧昧** | Lv.4 | 笨拙地表达喜欢，送的礼物总是炸弹 |
| **誓约** | Lv.5 | "我最美的烟火，只为你绽放！" |

### 特殊互动
- 会拉指挥官看"漂亮的爆炸"
- 送的礼物需要先检查是否会爆炸
- 被夸奖时会像小狗一样开心

## 七、关键台词

### 战斗语音
- （开始）「轰炸号起飞！让我们创造艺术吧！」
- （攻击）「炸炸炸炸炸！」
- （受伤）「哇哦好痛——但好刺激！」
- （暴击）「这个爆炸好漂亮！」
- （波次结束）「还不够！还要更多！」
- （胜利）「哈哈哈！烟火大会结束！」
- （失败）「呜...燃料不够了...」

### 日常语音（好感度相关）
- （Lv.1）「你好你好！要不要看我的新发明？保证不会炸！...大概！」
- （Lv.2）「指挥官指挥官！今天的实验超成功！只炸了一点点！」
- （Lv.3）「其实我...只是想创造美丽的东西。爆炸也可以是艺术对吧？」
- （Lv.4）「那个...这个送你！不是炸弹！是...心形烟火！」
- （Lv.5）「我所有的烟火，都只为你绽放。这是星火的誓言！」

### 特殊语音
- （长时间未登录）「指挥官不见了！要不要我炸出一条路去找你！」
- （生日）「生日快乐！我准备了超大的烟火！...呃，消防队来了。」

## 十、设计备注

**设计意图**：
- 疯狂科学家类型
- "诺娃"（新星/超新星）致敬爆炸
- 疯狂但善良的反差
- 天才的孤独与被认可的渴望

**参考来源**：
- 疯狂科学家
- 爆破专家
- 火焰/烟火意象

**设计要点**：
- 立绘体现疯狂和活力
- 橙红黄的火焰配色
- 炸毛发型是重要标识
- 护目镜和"样品"是特色细节
