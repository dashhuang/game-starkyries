## Phase 3: 创建事件系统

### 3.1 事件系统设计

```lua
-- utils/EventBus.lua
local EventBus = {
    listeners = {},
    debug = false,  -- 调试开关
}

-- 订阅事件
function EventBus.on(event, callback, priority)
    priority = priority or 0
    EventBus.listeners[event] = EventBus.listeners[event] or {}
    table.insert(EventBus.listeners[event], {
        callback = callback,
        priority = priority,
    })
    -- 按优先级排序
    table.sort(EventBus.listeners[event], function(a, b)
        return a.priority > b.priority
    end)
end

-- 取消订阅
function EventBus.off(event, callback)
    local list = EventBus.listeners[event]
    if not list then return end
    for i = #list, 1, -1 do
        if list[i].callback == callback then
            table.remove(list, i)
        end
    end
end

-- 发射事件
function EventBus.emit(event, ...)
    if EventBus.debug then
        print("[EventBus] emit:", event)
    end
    local list = EventBus.listeners[event]
    if not list then return end
    for _, listener in ipairs(list) do
        listener.callback(...)
    end
end

-- 清空所有监听器
function EventBus.clear()
    EventBus.listeners = {}
end

return EventBus
```

### 3.2 事件定义

```lua
-- 定义标准事件名（可在 config/events.lua 中统一管理）
Events = {
    -- 敌人相关
    ENEMY_DEATH = "enemy:death",           -- (enemy)
    ENEMY_DAMAGE = "enemy:damage",         -- (enemy, damage, isCrit)
    ENEMY_SPAWN = "enemy:spawn",           -- (enemy)
    
    -- 玩家相关
    PLAYER_DAMAGE = "player:damage",       -- (damage, result)
    PLAYER_HEAL = "player:heal",           -- (amount)
    PLAYER_LEVEL_UP = "player:levelUp",    -- (newLevel)
    
    -- 战斗相关
    WAVE_START = "battle:waveStart",       -- (waveNum)
    WAVE_COMPLETE = "battle:waveComplete", -- (waveNum)
    BOSS_SPAWN = "battle:bossSpawn",       -- (boss)
    BOSS_PHASE = "battle:bossPhase",       -- (boss, phase)
    
    -- 拾取相关
    PICKUP_COLLECT = "pickup:collect",     -- (pickup)
    
    -- 武器相关
    WEAPON_FIRE = "weapon:fire",           -- (weapon, target)
    WEAPON_HIT = "weapon:hit",             -- (weapon, target, damage)
    
    -- 无人机相关
    DRONE_FIRE = "drone:fire",             -- (drone, target)
}
```

### 3.3 使用示例

**发射事件（Enemy.lua）**：
```lua
local EventBus = require("utils.EventBus")

function Enemy.Die(enemy)
    -- ... 死亡逻辑
    EventBus.emit("enemy:death", enemy)
end
```

**订阅事件（Effects.lua）**：
```lua
local EventBus = require("utils.EventBus")

-- 模块初始化时订阅
function Effects.Init()
    EventBus.on("enemy:death", function(enemy)
        Effects.CreateDeathParticles(...)
    end)
end
```

### 3.4 迁移计划

**第一批迁移（高频使用）**：
- `Enemy.onDeath` → `EventBus.emit("enemy:death", enemy)`
- `Battle.onWaveStart` → `EventBus.emit("battle:waveStart", waveNum)`
- `Battle.onWaveComplete` → `EventBus.emit("battle:waveComplete", waveNum)`
- `Pickup.onCollect` → `EventBus.emit("pickup:collect", pickup)`

**保留回调（低频/特殊）**：
- `Shop.onPurchaseWeapon` - 单一调用者，保留回调更清晰
- `Battle.getPlayerPosition` - getter 函数，不适合事件
