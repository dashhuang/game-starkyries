## 一、战舰系统概述

### 1.1 战舰数量

| 舰种 | 数量 | 特点 |
|------|------|------|
| 突击舰 | 10 | 高机动性、高攻速、近程专精 |
| 火力舰 | 8 | 高伤害、远射程、导弹专精 |
| 防御舰 | 8 | 高护盾、高装甲、生存能力强 |
| 特种舰 | 8 | 元素/异常伤害、特殊效果 |
| 支援舰 | 6 | 舰载机/召唤物专精、工程学 |
| 旗舰 | 8 | 特殊机制、独特玩法、极端设计 |
| **总计** | **48** | |

### 1.2 战舰属性结构

每艘战舰包含：
- **舰型名称**：战舰的型号名
- **舰长**：对应舰长（详见叙事文档）
- **初始武器**：开局自带的武器
- **初始模块**：开局自带的模块（部分战舰有）
- **属性加成**：正面效果
- **属性减益**：负面效果
- **特殊能力**：独特的被动技能
- **解锁条件**：如何解锁该战舰

### 1.3 正负属性设计理念（对标Brotato）

> **设计哲学**：强力的正面属性必须伴随明显的负面属性

#### 设计原则

| 原则 | 说明 | 示例 |
|------|------|------|
| **代价平衡** | 正面效果越强，负面效果越重 | 狙击号：+100%伤害 / -60%攻速 |
| **塑造特色** | 负面属性定义战舰独特玩法 | 利刃号：仅近程武器 |
| **避免必选** | 没有"纯正面"的强力战舰 | 先锋号：无负面，但加成也较弱 |
| **鼓励多样** | 不同负面催生不同Build | 导弹流 vs 近程流 |

#### 负面属性类型

| 负面类型 | 效果 | 应对策略 |
|----------|------|----------|
| **属性削弱** | 降低某项属性 | 通过模块弥补 |
| **武器限制** | 只能使用某类武器 | 专精该类武器 |
| **机制改变** | 改变游戏规则 | 适应新玩法 |
| **风险增加** | 敌人更强或自伤 | 需要更高技术 |

#### 负面属性强度分级

| 强度 | 描述 | 示例 |
|------|------|------|
| ⚪ 轻微 | 几乎无影响 | 引擎推力-5% |
| 🔵 中等 | 需要适应 | 最大护盾-25% |
| 🟣 较重 | 改变策略 | 无法使用修复包 |
| 🔴 严重 | 核心限制 | 仅能装备1武器 |

### 1.4 战舰基础数值（所有战舰统一）

| 属性 | 基础值 | 说明 |
|------|--------|------|

## 九、开局可选武器池（对标Brotato）

> **设计说明**：参考Brotato的角色可选武器池机制，每艘战舰开局时玩家可从专属武器池中选择1把武器

### 9.0 设计规律

| 规律 | 说明 | 示例 |
|------|------|------|
| **规律1：武器类型限制** | 战舰限制武器类型 → 池仅包含该类型 | 利刃号仅近程 → 仅近程武器池 |
| **规律2：特殊武器加成** | 战舰对某类武器有额外加成 → 池优先/仅该类武器 | 幽灵号虚灵+10伤害 → 仅虚灵池 |
| **规律3：特殊机制** | 战舰机制需要特定武器 → 池匹配机制 | 医疗号非医疗-100%攻速 → 仅医疗池 |
| **规律4：无限制** | 无负面限制 → 较大混合池 | 先锋号无负面 → 14种混合池 |
| **规律5：无法装备** | 无法装备武器 → 极小池或无池 | 和平号火力-100% → 仅3种辅助 |

### 9.0.1 池大小参考

| 限制强度 | 池大小 | 典型战舰 |
|----------|--------|----------|
