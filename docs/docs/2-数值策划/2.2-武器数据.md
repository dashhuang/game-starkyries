> 本文档详细记录《星河战姬》中所有舰载武器的完整信息。

## 一、武器系统概述

### 1.1 武器数量
- **近程炮**：23种
- **中程导弹**：18种
- **远程激光**：16种（+1入门级）
- **舰载机**：13种（+1入门级）
- **总计**：70种武器

### 1.2 武器槽位
- 每艘战舰最多可同时装备**6种武器**
- 所有武器同时攻击（自动锁定最近敌舰）
- 相同武器可以装备多把

### 1.3 武器等级（Tier）

> **⚠️ 重要设计变更（2026-01-05）**：
> - **废弃固定公式**：不再使用 ×1.0/×1.25/×1.5/×1.75 的固定倍率
> - **采用自定义数值**：每把武器独立定义 T1-T4 伤害值（参考 Brotato）
> - **目标缩放比例**：T1→T4 约 **2.5-3.5 倍**（原 1.75 倍过弱）

| 1 | 标准型 | 白色 | 基础伤害（各武器独立定义） | ×1.0 |
| 2 | 改良型 | 绿色 | 约 T1 的 1.4-1.6 倍 | ×2.0 |
| 3 | 精英型 | 蓝色 | 约 T1 的 2.0-2.4 倍 | ×4.0 |
| 4 | 旗舰型 | 紫色 | 约 T1 的 2.8-3.5 倍 | ×8.0 |

**关键对应**：
- **SMG = 机炮**：速射、高频率、中近程
- **Melee = 力场**：贴身攻击，受**近战强化**加成
- **Ranged = 弹炮（机炮+导弹）**：受**弹道计算机**加成
- **Elemental = 能量（电弧+激光）**：受**能量增幅**加成
- **Turret = 舰载机**：工程类，**无伤害加成**（仅受工程%加成）

### 1.5 武器合成机制

> **详细机制请参阅** [3.1-资源与经济系统](../3-系统设计/3.1-资源与经济系统.md)

#### 购买规则

| 条件 | 购买行为 | 结果 |
|------|----------|------|
| 武器槽**有空位** | 购买任意武器 | 直接加入武器槽 |
| 武器槽**已满** + 购买**同ID同Tier**武器 | 自动合成 | 2把合成为1把更高Tier |
| 武器槽**已满** + 购买**不同ID或不同Tier**武器 | 无法购买 | 提示"武器槽已满" |

#### 自动合成触发条件

**只有同时满足以下3个条件，才会触发自动合成**：
1. ✅ 武器槽**已满**（没有空位）
2. ✅ 购买的武器**ID相同**（同一种武器）
3. ✅ 购买的武器**Tier相同**（同一品级）

#### 合成路径

| 合成路径 | Tier倍率 |
|----------|----------|
| T1 + T1 → T2 | ×1.25 |
| T2 + T2 → T3 | ×1.5 |
| T3 + T3 → T4 | ×1.75 |

**注意**：
- 武器槽未满时，购买同类武器会直接加入槽位，需要玩家**手动合成**
- 合成后武器数量减少（2把→1把），需权衡"多把低级"vs"单把高级"的策略价值

## 二、近程武器 (22种)

> 特点：高射速、无限穿透、射程6-16m
> 
> **伤害加成分类**：
> - **力场系列**：受**近战强化**加成（近程武器）
> - **电弧系列**：受**能量增幅**加成（能量武器）
> - **机炮系列**：受**弹道计算机**加成（弹炮武器）

### 射程分类说明（使用米制单位）

> **米（m）**：1 米 = 玩家战舰的短边（高度）。详见 [1.3-属性系统](../1-设计文档/1.3-属性系统.md)

| 射程(m) | 武器类型 | 核心概念 | 玩家展示 | 玩家理解 |
|----------|----------|----------|----------|----------|
| **≤8m** | **力场/电弧系列** | 贴身战斗 | "近程" | 需要冒险贴近敌人 |
| **9-24m** | **导弹/冷冻系列** | 安全距离输出 | "中程" | 保持距离作战 |
| **>24m** | **激光狙击等** | 远距离狙击 | "远程" | 最安全的输出距离 |

**射程单位说明**：所有射程均使用**米（m）**作为单位，与近程武器保持一致。

> **设计说明**：通过命名让玩家直观理解武器距离：
> - 看到"力场/护盾/电弧" → 知道是最近程，需要贴身
> - 看到"脉冲/冲击" → 知道是近程，但稍有距离
> - 看到"近防/突击" → 知道是近程火力

#### ⚠️ 灼烧（DOT）机制详解

**核心规则**（与 Brotato 一致）：

| 规则 | 说明 | 设计意义 |
|------|------|----------|
| **灼烧不叠加** | 多次灼烧只刷新持续时间，不累加伤害 | 多把喷射器DOT不会成倍增长 |
| **取较高值** | 新灼烧伤害 > 当前灼烧时才覆盖 | 高伤害灼烧不会被低伤害刷掉 |
| **DOT不触发能量吸收** | 持续伤害不算"命中" | 灼烧流需要靠命中次数吸血，不是DOT |

**等离子喷射器的真正价值**：
```
❌ 错误理解：DOT伤害很高
✅ 正确理解：穿透全体 → 一次命中多个敌舰 → 多次触发能量吸收

等离子喷射器 = 吸血发动机（不是DOT发动机）
```

**配装优先级**：
1. **能量吸收** > 火力输出（穿透 × 能量吸收 = 续航核心）
2. **射击频率** > 暴击（高攻速 = 更多命中 = 更多吸血）
3. **暴击** 收益较低（DOT不吃暴击，只有命中伤害吃）

## 五、舰载机 (12种)

> 特点：召唤物、自主作战、持续输出、受**工程%**加成（工程类武器，**无伤害类型加成**）

### 5.0 舰载机系统概述 ⭐重要

舰载机是**自主作战的伴随武器**，装备后会召唤舰载机环绕在战舰周围，自动寻敌、攻击、返回。

#### 核心特性

| 特性 | 说明 | 与Brotato对比 |
|------|------|---------------|
| **自主移动** | 会飞出攻击再返回 | Brotato炮塔是静态的 |
| **锁定追杀** | 锁定目标后持续攻击直到击杀 | 更有策略感 |
| **独立攻速** | 舰载机攻速不受玩家射击频率影响 | 与Brotato一致 |
| **能量吸收生效** | 舰载机攻击**可触发能量吸收** | ⚡与Brotato不同 |
| **品级合成** | T1-T4品级，合成逻辑与普通武器相同 | 与普通武器一致 |

#### 舰载机状态机

```
┌─────────────────────────────────────────────────────┐
│                                                     │
│   【待机】──发现敌人──→【攻击】──目标死亡──→【返回】  │
│      ↑                                      │      │
│      └──────────────────────────────────────┘      │
│                                                     │
└─────────────────────────────────────────────────────┘
```

| 状态 | 行为描述 |
|------|----------|
| **待机（IDLE）** | 环绕玩家战舰旋转，等待攻击冷却完成后寻找目标 |
| **攻击（ATTACKING）** | 飞向目标敌舰，进入攻击射程后持续开火，直到目标死亡 |
| **返回（RETURNING）** | 目标死亡后，飞回玩家身边，切换回待机状态 |

**关键设计**：无人机锁定目标后会**持续攻击直到击杀**，不会中途放弃。

#### 舰载机数量

- 每装备1把舰载机武器 = 召唤1架舰载机
- 最多6架同时环绕（均匀分布）
- 合成后数量减少（2把T1 → 1把T2 = 2架变1架）

## 6.1 武器命中机制（单体 vs AOE）⭐重要

> **对标Brotato**：明确定义武器的命中目标数量机制

### 6.1.0 核心规则

| 武器大类 | 默认命中 | 说明 | Brotato对应 |
|----------|----------|------|-------------|
| **近程武器** | **AOE（多体）** | 攻击范围内的所有敌舰都会被命中 | 近战武器天然AOE |
| **导弹武器** | **单体** | 每发导弹只命中1个敌舰 | 远程武器天然单体 |
| **激光武器** | **单体** | 每道激光只命中1个敌舰（除非穿透） | 远程武器天然单体 |
| **舰载机** | **单体** | 每次攻击只命中1个敌舰 | 召唤物单体 |

### 6.1.1 近程武器的AOE机制

近程武器（力场/电弧/脉冲/近防）**天然是AOE**：

```
攻击触发 → 检测射程范围内所有敌舰 → 全部造成伤害
```

| 近程类型 | 射程 | AOE方式 | 视觉表现 |
|----------|------|---------|----------|
| **力场系列** | 1.5 m | 环绕能量场 | 360°能量波动 |
| **电弧系列** | 2 m | 电弧传导 | 闪电弧光连接多目标 |
| **脉冲系列** | 2.5 m | 冲击波扩散 | 圆形冲击波 |
| **近防系列** | 3-4 m | 扇形射击 | 多道子弹扇形散射 |

**设计意义**：
- 近程武器射程短，用AOE弥补
- 鼓励玩家冒险贴近敌群
- 配合能量吸收（攻击回盾）形成高风险高回报

### 6.1.2 远程武器变AOE的方法

导弹和激光默认单体，可通过以下方式变成AOE：

#### 方法1：穿透（Piercing）

| 武器/效果 | 穿透数量 | 说明 |
|-----------|----------|------|
| 激光狙击 | 无限 | 贯穿全屏所有敌舰 |
| 穿刺弹 | 3 | 穿透3个敌舰 |
| 穿透属性+1 | +1目标 | 每点穿透多命中1个 |

```
弹道发射 → 命中敌舰A → 穿透继续飞行 → 命中敌舰B → ...
```

#### 方法2：爆炸（Explosion）

| 武器 | 爆炸半径 | 说明 |
|------|----------|------|
| 集束导弹 | 0.5m × 6 | 命中后产生6个连锁爆炸，分布在1.5m范围内 |
| 战术核弹 | 4 m | 大范围爆炸 |
| 轰炸无人机 | 0.5m × 3/4 | 集束爆炸（T4为4个），分布在2.0m范围内 |

```
弹道命中 → 产生爆炸 → 爆炸范围内所有敌舰受伤
```

#### 方法3：连锁（Chain）

| 武器 | 连锁次数 | 连锁范围 | 备注 |
|------|----------|----------|------|
| 离子连锁炮 | 3次 | 4 m | T4升级后连锁4次 |
| 连锁电弧 | 5次 | 4 m | |

```
命中敌舰A → 跳跃到4m内敌舰B → 跳跃到敌舰C → ...
```

#### 方法4：散射（Multi-projectile）

| 武器 | 子弹数 | 说明 |
|------|--------|------|
| 霰弹导弹 | 8发 | 大范围散射 |
| 散射炮 | 5发 | 扇形散射 |

```
一次攻击 → 发射多发弹道 → 每发独立命中1个敌舰
```

### 6.1.3 命中机制与属性的交互

| 属性/效果 | 单体武器 | 穿透武器 | AOE武器 |
|-----------|----------|----------|---------|
| **能量吸收** | 1次触发 | **每个穿透敌舰各触发1次** | 每个范围内敌舰各触发1次 |
| **击毁回盾** | 击毁1个回复 | 同时击毁多个各自回复 | 同时击毁多个各自回复 |
| **DOT效果** | 附加到1个敌舰 | **附加到所有穿透敌舰** | 附加到所有范围内敌舰 |
| **击退效果** | 击退1个敌舰 | 击退所有穿透敌舰 | 击退所有范围内敌舰 |

#### ⭐ 穿透 + 能量吸收协同（吸血发动机）

**核心机制**：穿透武器每次射击可命中多个敌舰，每个命中都独立触发能量吸收

| 武器类型 | 单次射击命中数 | 能量吸收触发次数 | 评价 |
|----------|---------------|-----------------|------|
| 单体武器 | 1 | 1 | 普通 |
| 穿透3武器 | 最多3 | 最多3 | 良好 |
| **穿透全体** | **敌群数量** | **敌群数量** | **吸血发动机** |

**实战示例**（等离子喷射器 + 80%能量吸收）：
```
场景：10个敌舰排成一线
单次射击 → 穿透10个敌舰 → 10次能量吸收判定
每次80%概率回1护盾 → 期望回盾 = 10 × 0.8 = 8护盾/次
配合0.17s攻速 → 理论回盾速度远超10/秒上限 → 稳定触及上限
```

**关键结论**：
- **穿透武器**配合"能量吸收"最有效（命中次数 = 穿透数）
- AOE武器配合"能量吸收"也有效（命中次数 = 范围内敌舰数）
- **DOT伤害不触发能量吸收**，穿透武器的价值在于"命中次数"而非"DOT伤害"

### 6.1.4 伤害类型（三大类 + 工程）

> **设计说明**：参考 Brotato 的 Melee/Ranged/Elemental 分类，结合太空题材重新命名

| 类型 | 模块名称 | 属性 | 影响武器 | Brotato对应 |
|------|----------|------|----------|-------------|
| **近程** | 近战强化 | `meleeDamageBonus` | 力场系列 | Melee |
| **弹炮** | 弹道计算机 | `ballisticDamageBonus` | 机炮、导弹、炮弹 | Ranged |
| **能量** | 能量增幅 | `energyDamageBonus` | 电弧、激光 | Elemental |
| **工程** | — | — | 舰载机（炮塔、无人机、战机） | Turret |

**注意**：
- **工程类武器无伤害类型加成**，仅受"工程%"属性影响
- 每个模块+20%对应伤害，最多叠加3层（+60%）

### 6.2 伤害计算公式

```
武器最终伤害 = 基础伤害 
             × Tier倍率 
             × (1 + 火力输出%) 
             × (1 + 武器类型加成%)
             × (1 + 暴击伤害%) [如果暴击]
```

### 6.3 DPS计算

```
武器DPS = 最终伤害 × 子弹数 / 冷却时间 × (1 + 射击频率%)
        × (1 + 暴击率 × (暴击伤害 - 1))
```

## 八、武器获取方式

### 8.1 补给站购买
- 每波结束后补给站刷新4个物品
- 可花费能量晶体刷新补给站
- 物品品质受战场运势属性影响

### 8.2 初始武器

**战舰自带武器**：
- 部分战舰有自带的初始武器（见战舰数据表）
- 有初始武器的战舰：大部分自带1把，少数自带2把
- 很多战舰没有初始武器
- 初始武器为 Tier 1

**玩家选择武器**：
- 开局时玩家选择1把武器
- 选择界面会显示战舰是否自带武器

**开局武器数量**：
- 有初始武器的战舰：自带武器 + 玩家选择 = **开局2把武器**
- 无初始武器的战舰：仅玩家选择 = **开局1把武器**
- 例：疾风号自带"双联速射力场"，玩家选择"粒子机炮"，开局即有2把武器

### 8.3 补给箱掉落
- 战场中随机出现补给箱
- 补给箱可能掉落武器
- 掉落品质随机
