## 二、错误项（需修正）

### 2.1 🔴 刷新价格公式错误

**文档位置**：`3.1-资源与经济系统.md`

**星河战姬记录**：
```
首次刷新 = floor(波次 × 0.5) + 1
后续递增 = floor(波次 × 0.35)
```

**Brotato原作（Wiki确认）**：
```
首次刷新 = floor(波次 / 2)
第N次刷新 = 首次费用 + (N-1) × floor(波次 × 0.33)
```

**差异分析**：
- 首次刷新：星河战姬多加了+1
- 递增系数：0.35 vs 0.33，微小差异

**建议修正**：
```lua
-- 正确公式
首次刷新 = math.floor(波次 / 2)
第N次刷新 = 首次费用 + (N-1) * math.floor(波次 * 0.33)

-- 波次1: 0, 0, 0, 0... （免费！）
-- 波次5: 2, 3, 5, 6...
-- 波次10: 5, 8, 11, 14...
-- 波次20: 10, 16, 23, 29...
```

### 2.4 🟡 生命恢复机制理解偏差

**文档位置**：`1.3-属性系统.md`

**星河战姬记录**：
> "护盾再生 = 每5秒回复的护盾点数"

**Brotato原作机制**：
```
HP Regeneration 是持续恢复（每秒），不是间歇恢复（每5秒）
- 第1点 = 0.20 HP/秒
- 后续每点 = +0.089 HP/秒（递减效应）
- 10点 HP Regen ≈ 1 HP/秒
```

**建议**：明确护盾再生是持续恢复还是间歇恢复，并记录递减效应公式。

## 三、缺失项（需补充）

### 3.1 🔴 无尽模式系统完全缺失

**严重程度**：高

星河战姬文档中提到"无尽深空"模式，但缺少详细机制文档。

**Brotato原作关键机制**：

#### 无尽因子（Endless Factor）
```
endless_factor = current_wave - 20

敌人HP倍率 = 1 + (endless_factor × 0.125)  // 每波+12.5%
敌人伤害倍率 = 1（固定不变）
敌人速度上限 = 275%（第40波达到）
```

#### 经济变化
- 第21-25波：刷新成本固定为波次20的值
- 第26波起：刷新成本停止增长
- 第25波后：商店不再生成Tier 1物品

#### 精英波规则
- 第21波起：每5波一次精英波
- 第25波：首个无尽精英波

**建议**：创建`2.5-无尽深空系统.md`文档。

### 4.2 ❓ 能量吸收触发舰载机

**星河战姬**：舰载机攻击**可触发**能量吸收

**Brotato原作**：召唤物攻击**不触发**生命偷取

**文档已标注**：这是有意的设计差异

**建议**：确认这个差异是否经过平衡测试。

---

### 4.3 ❓ 护盾再生间隔

**星河战姬**：每5秒回复护盾再生点数

**Brotato原作**：持续回复（每秒），有递减效应

**问题**：星河战姬是有意简化还是误解了原作机制？

**建议**：
- 如果是有意简化：在文档中明确说明"与Brotato不同"
- 如果是误解：修正为持续回复机制

### 4.4 ❓ 敌人伤害缩放

**星河战姬**：敌人伤害随波次增长

**Brotato无尽模式**：敌人伤害**固定不变**，只有HP增长
