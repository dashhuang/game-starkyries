# 1.5 区域与难度系统

> 区域是危险等级的叙事包装，用于整合游戏难度与世界观。

---

## 1. 设计原则

### 1.1 方案A：区域 = 危险等级的叙事包装

| 概念 | 本质 | 玩家感知 |
|------|------|---------|
| **危险等级** | 数值层面的难度系统 | 敌人更强、精英更多 |
| **区域** | 叙事层面的包装 | 去往更远的星域探索 |

**核心关系**：选择难度 = 选择区域

```
玩家视角：
  "我要去远域探索" → 实际触发危险4的难度规则

游戏内部：
  区域选择 → 映射到危险等级 → 应用对应的数值系统
```

### 1.2 映射表

| 区域 | 危险等级 | 难度标签 | 解锁条件 |
|------|---------|---------|---------|
| **近域** | 0-1 | 简单 | 默认解锁 |
| **中域** | 2-3 | 普通 | 通关近域 |
| **远域** | 4 | 困难 | 通关中域（危险3） |
| **禁域** | 5 | 噩梦 | 通关远域（危险4） |

> **说明**：近域/中域各包含2个危险等级，玩家可细选；远域/禁域各对应1个危险等级。

## 2. 区域详细设计

### 2.1 近域（危险0-1）

> **叙事定位**：舰队刚叛逃不久，在帝国边境活动，相对安全的区域。

| 属性 | 内容 |
|------|------|
| **主要敌人** | 星际海盗、帝国边防巡逻 |
| **视觉主题** | 边境星云、废弃空间站、小行星带 |
| **背景音乐** | 紧张但不绝望，有希望感 |
| **叙事线索** | 舰队逃亡的起因、帝国的追捕 |

**难度细分**：

| 子选项 | 危险等级 | 说明 |
|--------|---------|------|
| 近域·安全航线 | 0 | 默认难度，新手友好 |
| 近域·争议区 | 1 | 更多敌舰类型提前出现 |

### 2.2 中域（危险2-3）

> **叙事定位**：进入混乱地带，帝国控制力减弱，各势力交错的区域。

| 属性 | 内容 |
|------|------|
| **主要敌人** | 混合势力（海盗+机械叛军+帝国追兵） |
| **视觉主题** | 战场残骸、殖民地废墟、贸易航线 |
| **背景音乐** | 混乱、紧张、节奏加快 |
| **叙事线索** | 各势力的恩怨、遗迹线索开始出现 |

**难度细分**：

| 子选项 | 危险等级 | 特殊规则 |
|--------|---------|---------|
| 中域·混乱地带 | 2 | 精英波首次出现（11或12阶段） |
| 中域·战争前线 | 3 | 敌舰属性+12% |

### 2.3 远域（危险4）

> **叙事定位**：深入未知宇宙，接近传说中的遗迹，极其危险的区域。

| 属性 | 内容 |
|------|------|
| **主要敌人** | 虫族（原生者）、机械叛军 |
| **视觉主题** | 深空星云、外星建筑残骸、诡异天体 |
| **背景音乐** | 压抑、神秘、史诗感 |
| **叙事线索** | 原生者的起源、机械叛军的真相 |

**难度规则**：

| 项目 | 数值 |
|------|------|
| 敌舰属性加成 | +26% |
| 精英/潮水波 | 3次（11/12、14/15、17/18阶段） |

### 2.4 禁域（危险5）

> **叙事定位**：传说中的区域，"创世遗迹"所在，宇宙的终极秘密。

| 属性 | 内容 |
|------|------|
| **主要敌人** | ??? + 双Boss机制 |
| **视觉主题** | 异常空间、时空裂隙、古老遗迹 |
| **背景音乐** | 史诗、悲壮、终章感 |
| **叙事线索** | 地球真相、帝国与AI的关系 |

**难度规则**：

| 项目 | 数值 |
|------|------|
| 敌舰属性加成 | +40% |
| 精英/潮水波 | 3次 |
| **双Boss** | 每个Boss阶段出现2个Boss，各75%护盾 |

## 5. 无尽深空模式

通关禁域（危险5）后解锁。

| 属性 | 说明 |
|------|------|
| **区域名称** | 深渊（无尽深空） |
| **起始难度** | 相当于禁域（危险5） |
| **波次规则** | 无限继续，难度持续增加 |
| **敌舰属性** | 每10波+10%属性 |
| **排行榜** | 记录最高波次 |

---

## 6. 总结

| 玩家选择 | 游戏表现 |
|---------|---------|
| 选择"近域" | 应用危险0-1的难度规则 + 边境星云视觉 + 海盗/帝国敌人 |
| 选择"中域" | 应用危险2-3的难度规则 + 战场残骸视觉 + 混合敌人 |
| 选择"远域" | 应用危险4的难度规则 + 深空星云视觉 + 虫族/机械敌人 |
| 选择"禁域" | 应用危险5的难度规则 + 异常空间视觉 + 双Boss机制 |

**核心要点**：区域是玩家看到的"故事外衣"，危险等级是背后的"数值骨架"。
