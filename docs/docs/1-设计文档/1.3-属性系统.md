## 0. 空间单位系统（虚拟米制）

> **重要**：本游戏所有距离、速度使用 **米（m）** 作为单位，实现分辨率无关的设计。
> 
> 这是**虚拟米制**，并非现实物理尺度，但便于直观理解和沟通。

### 基准定义

```
1 米 = 玩家战舰的短边（高度）

战舰示意图（横版）：
        ← 长边（约3米）→
       ┌─────────────────┐
 1米 → │  ◄══ 战舰 ══►   │
       └─────────────────┘
```

### 参考尺寸

| 概念 | 尺寸 | 说明 |
|------|------|------|
| **战舰高度** | 1 m | 基准单位 |
| **战舰长度** | ~3 m | 战舰头尾长度 |
| **能量晶体** | ~0.5 m | 掉落物 |
| **小型敌舰** | ~1 m | 孢子虫等 |
| **屏幕宽度** | ~40 m | 横版可视区域 |
| **屏幕高度** | ~22 m | 横版可视区域 |
| **竞技场** | 50×30 m | 详见 [1.4-战斗系统](1.4-战斗系统.md) |

### Brotato 像素换算表

> **换算基准**：Brotato 角色高度约 32 像素，对应本游戏 1 米

| Brotato像素 | 本游戏(米) | 说明 |
|-------------|------------|------|
| 32 px | 1 m | 角色/战舰高度（基准） |
| 100 px | 3 m | 近程射程 |
| 300 px | 9 m | 中程射程 |
| 320 px | 10 m | 玩家移动速度/秒 |
| 960 px | 30 m | 屏幕高度 |
| 1280 px | 40 m | 屏幕宽度 |

### 距离分类

| 类型 | 米 | 玩家展示 | 用途 |
|------|------|----------|------|
| 贴脸 | 0-1 m | — | 力场类武器 |
| 近程 | 1-3 m | "近程" | 电弧/脉冲武器 |
| 中程 | 3-8 m | "中程" | 导弹/激光 |
| 远程 | 8-20 m | "远程" | 追踪/狙击 |
| 全屏 | 20+ m | "全屏" | 贯穿激光 |

### 速度分类

| 类型 | 米/秒 | 占玩家速度 | 说明 |
|------|-------|-----------|------|
| 玩家基础 | **10 m/s** | 100% | 每秒10个战舰高度，约4秒穿越屏幕 |
| 慢速敌舰 | 1-3 m/s | 10%-30% | 容易风筝 |
| 中速敌舰 | 3-6 m/s | 30%-60% | 可以风筝 |
| 快速敌舰 | 6-8 m/s | 60%-80% | 追逐紧张 |
| 极速敌舰 | 8+ m/s | 80%+ | 无法甩开 |

### 程序换算

```
实际渲染距离 = 米值 × 战舰短边渲染尺寸

示例（1080p，战舰高度=48像素）：3 m = 144像素
示例（4K，战舰高度=96像素）：3 m = 288像素
```

### 1.3 武器类型加成

| 属性 | 说明 | 影响武器 | Brotato对应 |
|------|------|----------|-------------|
| **近战强化** | 近程武器专属加成 | 力场系列 | Melee Damage |
| **弹炮强化** | 弹炮武器专属加成 | 机炮/导弹/鱼雷 | Ranged Damage |

### 1.3.1 工程属性详解（对标Brotato Engineering）

> **工程（Engineering）** 是所有召唤物的核心属性，类似Brotato的工程属性。

#### 工程影响的召唤物类型

| 召唤物类型 | 说明 | 代表武器/模块 |
|------------|------|---------------|
| **无人机** | 跟随战舰、自动出击攻击敌人 | 战斗无人机、轰炸无人机 |
| **炮塔** | 固定/跟随位置，持续攻击 | 防御炮塔、激光炮塔 |
| **战斗机** | 全图追踪敌舰 | 轻型战机、轰炸机 |
| **空间站** | （规划中）大型固定建筑 | 补给站、火力支援站 |

#### 工程伤害计算

```
召唤物伤害 = 基础伤害 × (1 + 火力输出%) × (1 + 工程%)
```

**注意**：召唤物**同时受**火力输出和工程加成，但**不受**近程/弹炮/能量伤害加成。

#### 工程专属规则

| 规则 | 说明 | 与Brotato差异 |
|------|------|---------------|
| **✅ 触发能量吸收** | 召唤物攻击可回盾 | 不同于Brotato |
| **不受射击频率影响** | 召唤物有独立攻速 | 与Brotato一致 |
| **受暴击影响** | 召唤物可以暴击 | 与Brotato一致 |
| **受DOT效果影响** | 召唤物可附加燃烧等 | 与Brotato一致 |

### 1.4 功能类属性

| 属性 | 英文 | 说明 |
|------|------|------|
| **引擎推力** | Speed | 战舰移动速度加成（基础10 m/s = 100%） |
| **战场运势** | Luck | 影响掉落/商店品质 |

| **拾取范围** | Pickup Range | 自动拾取掉落物范围 |

## 2. 计算公式

### 2.1 伤害计算

```
最终伤害 = 武器基础伤害 
         × (1 + 火力输出%) 
         × (1 + 武器类型加成%)
         × 暴击倍率  ← 仅暴击时
         - 敌舰装甲

最低伤害 = 1
```

**示例计算**：
- 武器基础伤害：10
- 火力输出：+50%
- 近战强化：+30%（力场武器时）
- 暴击触发，暴击倍率：2.0

```
最终伤害 = 10 × 1.5 × 1.3 × 2.0 = 39
```

### 2.1.1 暴击系统详解 ⭐

> **暴击率和暴击倍率由武器基础值和玩家属性共同决定**

#### 暴击率计算

```
最终暴击率 = 武器基础暴击率 + 玩家暴击率属性

示例：
- 粒子机炮基础暴击率：1%
- 玩家暴击率属性：5%
- 最终暴击率 = 1% + 5% = 6%
```

#### 暴击倍率计算

```
最终暴击倍率 = 武器暴击倍率(默认200%) + 玩家暴击伤害加成

玩家暴击伤害加成 = 玩家critDamage属性 - 基础值1.5

示例：
- 粒子机炮暴击倍率：150%（配置为1.5）
- 玩家critDamage：2.0（基础1.5 + 模块加成0.5）
- 玩家加成 = 2.0 - 1.5 = 0.5
- 最终暴击倍率 = 1.5 + 0.5 = 2.0 (200%)
```

#### 不同武器的暴击配置示例

| 武器 | 基础暴击率 | 基础暴击倍率 | 特点 |
|------|-----------|-------------|------|
| 粒子机炮 | 1% | 150% | 高射速低暴击 |
| 速射导弹 | 0% | 200%(默认) | 无基础暴击 |
| 特斯拉电弧 | 5% | 200%(默认) | 中等暴击 |
| 等离子狙击炮 | 10% | 200%(默认) | 高暴击武器 |

> ⚠️ **注意**：武器未配置暴击倍率时，默认使用200%

### 2.2 攻击间隔计算

```
实际攻击间隔 = 武器基础间隔 / (1 + 射击频率%)
```

**示例**：
- 武器基础间隔：1秒
- 射击频率：+100%

```
实际攻击间隔 = 1 / (1 + 1.0) = 0.5秒
```

### 2.3 装甲减伤

```
实际受到伤害 = 原始伤害 - 装甲值
(最低伤害 = 1)
```

### 2.4 DPS计算

```
DPS = (武器伤害 × 火力加成 × 类型加成 × 暴击系数) / 攻击间隔

暴击系数 = 1 + (暴击率% × 暴击伤害%)
```

### 2.5 有效护盾

```
有效护盾 = 最大护盾 / [(1 - 规避率%) × (DPS - 装甲) / DPS] 
         + 护盾再生 × 存活时间
         + 能量吸收预期值
```

```

近程武器：有独立的更低上限
递减效应：高数值时每点加成效果降低
负值计算：与正值使用不同公式
```

| **工程** | 降低召唤物伤害/治疗量，最低1 |

> ⚠️ **装甲负值特别重要**：负装甲会让你**受到更多伤害**，而非减少！

### 3.3 护盾再生与能量吸收详细机制（Brotato Wiki精确对标）

#### 护盾再生（Shield Regen）- 持续回复机制 ⭐

> ⚠️ **重要**：护盾再生是**持续回复（每秒）**，不是间歇回复（每5秒）！

**公式（Brotato Wiki确认）**：
```
第1点护盾再生 = 0.20 HP/秒
后续每点 = +0.089 HP/秒（递减效应）

计算公式：HP/秒 = 0.20 + (护盾再生值 - 1) × 0.089
```

| 护盾再生值 | 每秒回复 | 计算过程 |
|-----------|----------|----------|
| 1 | 0.20 HP/s | 基准值 |
| 5 | 0.56 HP/s | 0.20 + 4×0.089 |
| 10 | 1.00 HP/s | 0.20 + 9×0.089 |
| 20 | 1.89 HP/s | 0.20 + 19×0.089 |
| 30 | 2.78 HP/s | 0.20 + 29×0.089 |
| 50 | 4.56 HP/s | 0.20 + 49×0.089 |

**设计意义**：
- 递减效应防止无脑堆叠
- 10点护盾再生 ≈ 1 HP/秒是实用基准
- 与能量吸收形成互补（再生稳定但慢，吸收快但依赖攻击）

#### 能量吸收（Energy Drain）- 与Brotato Lifesteal一致 ⭐

**核心规则**：
1. **每秒上限10护盾**：无论攻速多高，每秒最多回复10护盾（每0.1秒最多1次）
2. **触发机制**：每点能量吸收 = 1%概率回复1护盾
3. **100%能量吸收 = 每次攻击必回1护盾**

**⚠️ 关键机制（常见误区）**：
| 伤害来源 | 是否触发能量吸收 | 说明 |
|----------|------------------|------|
| **武器命中** | ✅ 触发 | 子弹/激光命中敌舰瞬间 |
| **穿透命中** | ✅ 每个敌舰各触发1次 | 穿透N个敌舰 = N次触发机会 |
| **AOE命中** | ✅ 每个敌舰各触发1次 | 爆炸范围内每个敌舰 |
| **DOT伤害** | ❌ **不触发** | 灼烧/腐蚀的持续伤害不算"命中" |
| **召唤物命中** | ✅ 触发 | 炮塔/无人机攻击可回盾 |

> **设计意义**：这解释了为什么"穿透全体"武器（如等离子喷射器）是"吸血发动机"——
> 一次射击穿透10个敌舰 = 10次能量吸收触发机会，而DOT本身不提供额外吸血。

**公式**：
```
回盾概率 = 能量吸收%
回盾量 = 1护盾/次
最大回复 = 10护盾/秒
```

| 能量吸收 | 触发概率 | 配合攻速的效果 |
|----------|----------|----------------|
| 20% | 20%/次 | 需要高攻速弥补 |
| 50% | 50%/次 | 中等稳定 |
| 80% | 80%/次 | 较稳定 |
| 100% | 100%/次 | 每次攻击必回1护盾 |

**优化策略**：
- **攻速流**：高攻速+高能量吸收 = 稳定回盾（但受10/秒上限）
- **多武器流**：6把武器 = 6次独立触发机会
- **目标**：建议达到60-80%能量吸收以实现稳定回盾

## 4. 边际效应

### 4.1 攻击类属性

攻击类属性叠加存在边际递减：
- 第一个+10%伤害 → 实际提升10%
- 叠加到+100%时，再+10% → 实际只提升5%

### 4.2 防御类属性

- **装甲**：对高伤害敌人效果有限
- **规避**：存在硬上限
- **护盾再生**：需配合生存时间

### 4.3 Build设计启示

| 流派 | 核心属性 | 边际考虑 |
|------|----------|----------|
| 暴击流 | 暴击率+暴击伤害 | 100%暴击后堆暴击伤害 |
| 攻速流 | 射击频率 | 配合能量吸收回盾 |
| 坦克流 | 装甲+护盾 | 注意敌人穿甲 |
| 规避流 | 规避率 | 注意60%上限 |

| **战场运势** | +5 | +10 | +15 | +20 | 点 |
| **资源回收** | +5% | +8% | +10% | +12% | % |

> ⚠️ **重要**：专项伤害（近战/弹炮/能量）的升级加成远低于通用火力输出！
> - 火力输出 T4 = +16%（通用，影响所有武器）
> - 近战强化 T4 = +8%，弹炮/能量 T4 = +4%（专项，更依赖模块获取）

### 6.2 品质保底规则（Brotato Wiki精确对标） ⭐

⚠️ **保底 = 全部 4 个选项都是该品质**

| 舰长等级 | 保底品质 | 说明 |
|----------|----------|------|
| 1-4级 | 100% Tier I | 4个选项全部是 Tier I |
| **5级** | **100% Tier II** | ⭐ 4个选项**全部是 Tier II** |
| 6-9级 | Tier I/II 混合 | 每个选项独立随机，受运势影响 |
| **10级** | **100% Tier III** | ⭐ 4个选项**全部是 Tier III** |
| 11-14级 | Tier I/II/III 混合 | 每个选项独立随机，受运势影响 |
| **15级** | **100% Tier III** | 4个选项**全部是 Tier III** |
| 16-19级 | Tier I/II/III 混合 | 每个选项独立随机，受运势影响 |
| **20级** | **100% Tier III** | 4个选项**全部是 Tier III** |
| 21-24级 | Tier I/II/III/IV 混合 | 每个选项独立随机，受运势影响 |
| **25级+** | **100% Tier IV** | ⭐ 4个选项**全部是 Tier IV** |

### 6.3 升级属性刷新规则

**升级时可以花费材料刷新 4 个属性选项**：

| 规则 | 说明 |
|------|------|
| **刷新价格** | 与商店刷新使用**相同公式** |
| **独立计价** | 升级刷新和商店刷新**分别计算次数**，互不影响 |
| **同波次共享** | 同一波次获得多次升级时，刷新价格**共享递增** |
| **跨波次重置** | 下一波次升级时，刷新次数重置 |

### 6.4 战场运势影响

战场运势（Luck）使用与商店相同的公式，提高非保底等级获得高品质属性的概率。

### 6.5 设计意义

1. **成长曲线平滑**：前期低品质，后期高品质，匹配难度曲线
2. **运势属性多用途**：不仅影响商店，还影响升级品质
3. **里程碑获得感**：5/10/15/20/25 级有明确的品质提升节点
4. **随机中的确定性**：保底机制防止运气过差
5. **刷新机制**：提供"花钱换选择"的策略深度

### 7.1 标签页分类

#### 主要属性列表

| 分类 | 属性 | 值类型 |
|------|------|--------|
| **状态** | 舰桥等级、晶体、经验值 | 基础值 |
| **防御** | 最大护盾、护盾再生、装甲 | 基础值 |
| **防御** | 规避率、能量吸收、受伤减免 | 加成值 |

#### 次要属性列表

| 分类 | 属性 | 值类型 |
|------|------|--------|
| **专精伤害** | 近程伤害、弹炮伤害、能量伤害、召唤伤害 | 加成值 |

### 7.2 属性显示格式

区分 **基础值** 和 **加成值** 两种显示方式：

| 值类型 | 显示格式 | 示例 |
|--------|----------|------|
| **基础值** | `属性名: 数值` | `最大护盾: 30`、`武器槽位: 6` |
| **加成值** | `%属性名: 数值` | `%火力输出: 15`、`%射击频率: 20` |

**设计理由**：
- 基础值是实际数值，直接影响战斗（如护盾30点）
- 加成值是百分比修正，名称前加 `%` 明确表示这是加成而非实际值
- 与 Brotato 界面风格保持一致

### 7.3 颜色编码

| **零值** | ⚪ 白色 | (180, 185, 195) | 无加成 |
| **状态值** | ⚪ 白色 | (220, 225, 235) | 等级、晶体等（不使用颜色编码） |
