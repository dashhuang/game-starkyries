# 1.2 核心玩法

> 游戏核心循环概述

---

## 1. 单局游戏流程

```
┌─────────────────────────────────────────────────────────────┐
│                         母港                                 │
│  选择战舰 → 选择初始武器 → 确认出击                          │
└─────────────────────────────────────────────────────────────┘
                              ↓
┌─────────────────────────────────────────────────────────────┐
│                      战役阶段 1-20                           │
│  ┌─────────────┐    ┌─────────────┐    ┌─────────────┐      │
│  │  战斗阶段   │ → │  舰桥升级   │ → │   补给站    │       │
│  │  20-90秒   │    │ (波次结束后) │    │  (每阶段后) │       │
│  └─────────────┘    └─────────────┘    └─────────────┘      │
│         ↑                                      ↓             │
│         └──────────── 下一阶段 ←──────────────┘             │
└─────────────────────────────────────────────────────────────┘
                              ↓
┌─────────────────────────────────────────────────────────────┐
│                        战斗结算                              │
│  统计数据 → 好感度增加 → 解锁检查 → 返回母港                 │
└─────────────────────────────────────────────────────────────┘
```

---

## 2. 核心循环说明

### 2.1 战斗阶段

玩家在竞技场中战斗，击杀敌舰获取经验和能量晶体。

| 要素 | 说明 | 详细文档 |
|------|------|----------|
| **竞技场** | 50×30米战斗区域，镜头跟随玩家 | → [1.4-战斗系统](1.4-战斗系统.md) |
| **掉落物** | 经验值自动获取，晶体需拾取 | → [1.4-战斗系统](1.4-战斗系统.md) |
| **战役时长** | 第1阶段20秒 → 第20阶段90秒 | → [1.4-战斗系统](1.4-战斗系统.md) |
| **难度系统** | 危险等级0-5，影响敌舰属性 | → [1.4-战斗系统](1.4-战斗系统.md) |

### 2.2 舰桥升级

累计经验值触发升级，从4个随机属性中选择1个强化。

| 要素 | 说明 | 详细文档 |
|------|------|----------|
| **升级机制** | 经验值达到阈值，波次结束后触发 | → [3.1-资源与经济系统](../3-系统设计/3.1-资源与经济系统.md) |
| **品质系统** | Tier I-IV，特定等级保底高品质 | → [3.1-资源与经济系统](../3-系统设计/3.1-资源与经济系统.md) |
| **属性刷新** | 花费晶体刷新4个选项 | → [3.1-资源与经济系统](../3-系统设计/3.1-资源与经济系统.md) |

### 2.3 补给站

每个战役阶段结束后，使用能量晶体购买武器和模块。

| 要素 | 说明 | 详细文档 |
|------|------|----------|
| **购买物品** | 显示4个随机物品 | → [3.1-资源与经济系统](../3-系统设计/3.1-资源与经济系统.md) |
| **刷新补给** | 花费晶体刷新物品 | → [3.1-资源与经济系统](../3-系统设计/3.1-资源与经济系统.md) |
| **武器合成** | 购买相同武器，2把合成为更高Tier | → [3.1-资源与经济系统](../3-系统设计/3.1-资源与经济系统.md) |
| **物品品质** | 阶段越高，高品质概率越大 | → [3.1-资源与经济系统](../3-系统设计/3.1-资源与经济系统.md) |

## 3. 胜利与失败

| 条件 | 结果 |
|------|------|
| **存活至阶段结束** | 进入下一阶段 |
| **通关第20阶段** | 游戏胜利 |
| **战舰护盾归零** | 游戏结束 |
