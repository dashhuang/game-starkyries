# 1.5 主界面系统

> 游戏启动界面、选项设置、图鉴系统

---

## 1. 主界面

> 游戏启动后的首个界面，提供核心功能入口

### 1.1 界面布局

```
┌─────────────────────────────────────────────────────────────┐
│                                                             │
│                     ⭐ 星河战姬 ⭐                           │
│                    STAR CAPTAIN                             │
│                                                             │
│                  ┌─────────────────┐                        │
│                  │   ▶ 开始游戏    │                        │
│                  └─────────────────┘                        │
│                  ┌─────────────────┐                        │
│                  │   ▷ 继续游戏    │  ← 仅有存档时显示       │
│                  └─────────────────┘                        │
│                  ┌─────────────────┐                        │
│                  │   ⚙ 选项        │                        │
│                  └─────────────────┘                        │
│                  ┌─────────────────┐                        │
│                  │   📖 图鉴        │                        │
│                  └─────────────────┘                        │
│                                                             │
│                                            v1.0.0           │
└─────────────────────────────────────────────────────────────┘
```

### 1.2 按钮功能

| 按钮 | 功能 | 说明 |
|------|------|------|
| **开始游戏** | 进入母港，开始新的一局 | 选择战舰 → 选择武器 → 出击 |
| **继续游戏** | 从即时存档恢复游戏 | 仅在有存档时显示，详见 [3.3-存档系统](../3-系统设计/3.3-存档系统.md) |
| **选项** | 打开设置菜单 | 音量、语言等 |
| **图鉴** | 查看已解锁内容 | 成就、武器、模块、敌人 |

## 2. 选项菜单

```
┌─────────────────────────────────────────┐
│              ⚙ 选项                     │
├─────────────────────────────────────────┤
│                                         │
│  🔊 音乐    [■■■■■■■□□□] 70%    [✓]    │
│                                         │
│  🔈 音效    [■■■■■■■■■□] 90%    [✓]    │
│                                         │
│  🌐 语言    [ 简体中文 ▼ ]              │
│                                         │
├─────────────────────────────────────────┤
│          [保存]      [返回]             │
└─────────────────────────────────────────┘
```

| 选项 | 控件 | 说明 |
|------|------|------|
| **音乐** | 滑块 + 开关 | 0-100% 音量，可单独关闭 |
| **音效** | 滑块 + 开关 | 0-100% 音量，可单独关闭 |
| **语言** | 下拉选择 | 简体中文 / 繁體中文 / English |

## 3. 图鉴系统

```
┌─────────────────────────────────────────┐
│              📖 图鉴                     │
├─────────────────────────────────────────┤
│                                         │
│    🏆 成就        ⚔️ 武器               │
│                                         │
│    🔧 模块        👾 敌人               │
│                                         │
├─────────────────────────────────────────┤
│                [返回]                   │
└─────────────────────────────────────────┘
```

| 分类 | 内容 | 解锁条件 |
|------|------|----------|
| **成就** | 成就列表、完成进度、奖励 | 达成对应条件 |
| **武器** | 已遇见武器、属性数据、Tier信息 | 首次获得该武器 |
| **模块** | 已遇见模块、效果说明 | 首次获得该模块 |
| **敌人** | 已遇见敌人、属性、掉落 | 首次击败该敌人 |
