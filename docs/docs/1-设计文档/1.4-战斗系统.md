# 1.4 战斗系统

> 竞技场设计、战斗机制、战役阶段与难度系统

---

## 1. 竞技场设计（对标Brotato）

> **核心概念**：竞技场 > 屏幕，玩家在更大的战斗区域中自由移动，镜头跟随玩家。

### 1.1 尺寸设计

| 概念 | 尺寸 | 说明 |
|------|------|------|
| **屏幕可视区域** | 40×22 m | 玩家实际看到的区域（880 m²） |
| **竞技场（战斗区域）** | **50×30 m** | 实际战斗空间（1500 m²） |
| **竞技场面积比** | **约1.7倍屏幕** | 对标Brotato的1.5-2倍设计 |

> **单位说明**：1 米 = 玩家战舰的短边（高度）。详见 [1.3-属性系统](1.3-属性系统.md)

### 1.2 镜头行为

| 玩家位置 | 镜头行为 |
|----------|----------|
| 远离边界 | 镜头跟随玩家，玩家居中 |
| 接近边界 | 镜头停止，玩家偏离中心 |
| 贴近边界 | 镜头锁定边界，玩家可见边缘 |

### 1.3 敌舰刷新规则

| 规则 | 说明 |
|------|------|
| **刷新位置** | 竞技场边缘（屏幕外），360°全方位 |
| **刷新距离** | 距离竞技场边界1-2米处跃迁进入 |
| **刷新方向** | 朝向竞技场中心或玩家位置 |

### 1.4 与Brotato对比

| 项目 | Brotato | 星河战姬 | 说明 |
|------|---------|----------|------|
| 屏幕尺寸 | 960×540像素 | 40×22 m | 基准可视区 |
| 竞技场尺寸 | 1200×800像素 | 50×30 m | 扩展战斗区 |
| 竞技场/屏幕比 | ~1.85倍 | ~1.7倍 | 略小于Brotato |
| 敌人刷新 | 屏幕外360° | 竞技场边缘360° | 一致 |

### 1.5 设计意义

- ✅ 玩家有走位空间，可主动拉开距离
- ✅ 敌人从屏幕外进入，有反应时间
- ✅ 边界提供战术价值（背靠边界减少被包围方向）
- ✅ 对标Brotato的1.5倍屏幕设计，保持紧凑战斗节奏

## 2. 战斗阶段

### 2.1 基本规则

| 项目 | 说明 |
|------|------|
| **持续时间** | 第1阶段20秒 → 第20阶段90秒（逐步增加） |
| **敌舰刷新** | 从竞技场边缘（屏幕外）360°全方位跃迁进入 |
| **视角** | 横版视角，镜头跟随玩家 |
| **自动攻击** | 武器自动锁定最近敌舰 |
| **玩家操作** | 移动、规避、拾取掉落物 |

### 2.2 战斗操作

```
┌─────────────────────────────────────────┐
│           横版战场示意图                 │
│                                         │
│   敌舰 →  ●  ●  ●                      │
│                    ▲                    │
│                   战舰                  │
│                    ↕ 移动               │
│         ●  ●  ● ← 敌舰                 │
│                                         │
│   [方向键/摇杆] 移动战舰                 │
│   武器自动开火                          │
└─────────────────────────────────────────┘
```

### 2.3 掉落物

| 掉落物 | 图标 | 效果 | 获取方式 |
|--------|------|------|----------|
| **经验值** | ⭐ 金色光点 | 累计升级进度 | **自动获取**（击杀后自动结算） |
| **能量晶体** | 💎 绿色 | 唯一货币（补给站购买） | **需要拾取** |
| **修复包** | ❤️ 红色 | 恢复护盾值 | 需要拾取 |
| **补给箱** | 📦 金色 | 额外奖励（晶体/模块） | 需要拾取 |

> **对标Brotato**：经验值自动获取，消除拾取压力；能量晶体需要主动拾取，保留风险-收益决策。

### 2.4 掉落保留机制（对标Brotato）

**核心机制**：未拾取的能量晶体会在下一阶段开始时重新掉落

| 规则 | 说明 |
|------|------|
| **保留规则** | 本阶段未拾取的晶体 → 下阶段开始时重新掉落 |
| **视觉标识** | 重新掉落的晶体显示 "×2" 标记 |
| **经验值** | ✅ 击杀时自动获取，无需拾取，不受保留机制影响 |
| **修复包** | 不保留，本阶段内失效 |

**设计意义**：
- ✅ **降低惩罚**：因走位失误漏拾的晶体不会永久丢失
- ✅ **避免冒险死亡**：不必为拾取晶体冒险送命
- ⚠️ **仍需冒险**：下阶段需要在新敌人中拾取，保持风险-收益
- 📌 **培养意识**：提醒玩家上一阶段的遗漏

### 2.5 拾取的风险-收益设计

**核心设计**：拾取掉落物需要冒险靠近敌舰

| 玩家水平 | 行为 | 结果 |
|----------|------|------|
| **新手** | 谨慎拾取，部分遗漏 | 依赖保留机制 |
| **熟练** | 高效拾取，偶有遗漏 | 资源获取稳定 |
| **高手** | 精准走位，全部拾取 | 最大化资源+生存 |

**策略价值**：
- 高"资源回收"属性 → 每颗晶体价值更高 → 更值得冒险
- 高"拾取范围"属性 → 更安全地拾取
- 高"引擎推力"属性 → 更灵活地走位拾取

## 3. 战役阶段设计

### 3.1 时间表（对标Brotato）

> **时长设计**：1-9阶段递增（20→60秒），10-19阶段固定60秒，20阶段Boss波90秒

| 阶段 | 持续时间 | 敌人密度 | 势力 | 特殊事件 |
|------|----------|----------|------|----------|
| 1-5 | 20-40秒 | 低→中 | 虫族 | 阶段5：虫母Boss |
| 6-9 | 45-60秒 | 中→高 | 机械叛军 | - |
| **10** | **60秒** | **高** | **机械叛军** | **AI核心Boss** |
| 11-15 | 60秒 | 高→极高 | 星际海盗 | 阶段15：海盗王Boss |
| 16-19 | 60秒 | 极高 | 异星帝国 | - |
| **20** | **90秒** | **疯狂** | **异星帝国** | **帝国母舰Boss** |

### 3.2 胜利条件

- **存活至阶段结束**：进入下一阶段
- **通关第20阶段**：游戏胜利
- **战舰损毁**：游戏结束

## 4. 难度系统

> **完整设计**：区域与难度系统的详细设计（包括区域叙事、视觉主题、敌人分布）请参见 [1.6-区域与难度系统](1.6-区域与难度系统.md)

### 4.1 危险等级（对标Brotato）

> **设计说明**：危险0-2无属性加成，从危险3开始敌舰属性增强。精英波/潮水波从危险2开始出现。

| 等级 | 敌舰属性 | 精英/潮水波 | 特殊效果 | 解锁条件 |
|------|----------|-------------|----------|----------|
| 0 | 无加成 | 无 | 标准难度 | 默认 |
| 1 | 无加成 | 无 | 更多敌舰类型提前出现 | 通关1次 |
| 2 | 无加成 | 1次（11或12阶段） | 精英/潮水波首次出现 | 通关危险1 |
| **3** | **+12%** | 1次（11或12阶段） | 敌舰属性首次提升 | 通关危险2 |
| **4** | **+26%** | 3次（11/12、14/15、17/18阶段） | 更多精英/潮水波 | 通关危险3 |
| **5** | **+40%** | 3次 | 双Boss（各75%护盾） | 通关危险4 |

### 4.2 无尽深空模式

- 通关第20阶段后解锁
- 战役阶段无限继续
- 难度持续增加
- 用于刷分和挑战
