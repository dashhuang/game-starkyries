## 游戏简介

《星河战姬》是一款结合了 **Brotato 核心玩法**与**二次元太空战舰**题材的游戏。玩家扮演舰队指挥官，操控战舰自动射击、走位躲避，在 20 波跳跃中收集武器、搭配模块，抵御虫族入侵。

### 核心特色

| 特色 | 说明 |
|------|------|
| **自动射击** | 专注走位，战舰自动锁定攻击 |
| **48 艘战舰** | 每艘战舰对应一位可养成的女性舰长 |
| **68 种武器** | 力场/电弧/机炮/导弹/激光/载机 六大类 |
| **144 种模块** | 丰富的 Build 构建可能性 |
| **四大势力** | 虫族、机械叛军、星际海盗、异星帝国 |
| **舰长养成** | 立绘、语音、好感度、专属剧情 |
| **3D 俯视角** | 3D 渲染 + 俯视 2D 玩法，兼具表现力与操作性 |
| **多平台** | 同时支持 PC 和手机 |

| 内容 | 已完成 | 完整版目标 | 进度 |

| 战舰 | 2 | 48 | 4% |

| 敌人 | 6 + 2 Boss | 30+ | 20% |
| 波次 | 20 | 20 | 100% |
| 角色文档 | 49 | 48 | 100% |

**已实现的系统**：
- 完整战斗循环（移动、自动射击、碰撞、拾取）
- 波间商店与舰桥升级
- 超空间跳跃动画
- 漂浮残骸与补给箱
- 视觉小说对话系统（立绘 + 表情 + 语音）
- 云存档与加载
- 教程系统
- 自动战斗 AI
- 虚拟摇杆（移动端）
