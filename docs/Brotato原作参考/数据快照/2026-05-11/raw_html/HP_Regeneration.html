HP Regeneration

HP Regeneration is a form of Healing. When you have HP regeneration, your HP will slowly regenerate whenever you're below your Max HP.

You regenerate 1 HP every x seconds. If this is negative, it acts like the stat is at 0.

  • 1 = Heal 1 HP every 5 seconds (0.2 /sec).
  • 10 = Heal 1 HP every 1 second.
  • 21 = Heal 1 HP every 0.51 seconds (1.98 /sec, ~2 HP /sec).
  • 33 = Heal 1 HP every 0.33 seconds (3.04 /sec, ~3 HP /sec).
  • For exact changes per stat point, see Formulas & Values below.
  • It can be approximated to the first point of HP regeneration giving 0.2 HP/s and all following points increasing it by 0.089 HP/s


See the Healing page for comparisons with other forms of healing.

The Torture item prevents you from using HP Regeneration. In return it provides a healing which regenerates 5 HP/s, the same as exactly 55 worth of , but at the cost of all other healing. Note that torture heals every second in chucks of 5, while 55 HP regeneration would heal 1 HP every 0.20 sec.

Formulas & Values

The formula used in the game for HP regen is: 

HPEveryXSeconds = 5.0 / ( 1.0 + ( ( HP_Regen - 1) / 2.25 ) )

which means: 

HPEveryXSeconds = 11.25 / (1.25 + )

HPPerSecond = /11.25 + 1/9.

So 11.25 additional will heal you 1 additional HP per second.
