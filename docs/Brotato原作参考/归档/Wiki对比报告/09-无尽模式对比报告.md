## Wiki无尽模式完整内容摘要

### 1. 解锁与启用

| 项目 | 说明 |
|------|------|
| 解锁条件 | 无需解锁，在难度选择时切换 |
| 启用方式 | 可在菜单选择Danger时启用，或在第20波前启用 |
| 胜利条件 | 坚持到20波后，死亡时计为胜利 |

### 2. 波次结构（20波后）

| 规则 | 说明 |
|------|------|
| 波次分组 | 每10波为一组 |
| Boss波 | 每10波（30/40/50...）双Boss出现 |
| 精英波 | 根据Danger等级0/1/3次，每过10波+1精英 |
| 潮水波 | 20波后不再出现 |
| 第20波变化 | 90秒，杀死Boss不结束波次 |

### 3. Endless Factor核心公式

```
Endless_wave = max(0, current_wave - 20)
Endless_mult = 2.0 + max(0, (current_wave - 35) * 0.2)
Endless_factor = max(0, ((endless_wave * (endless_wave + 1)) / 2) / 100) * endless_mult
```

| 波次 | Endless Factor | 说明 |
|------|----------------|------|
| 20 | 0 | 开始计算 |
| 30 | 1.10 | 适度增强 |
| 35 | 2.40 | 乘数开始增加 |
| 40 | 6.30 | 急剧增强 |
| 50 | 15.50 | 极难 |

### 4. 敌人缩放

| 属性 | 公式 | 波次30 | 波次40 |
|------|------|--------|--------|
| 伤害 | `Damage × (1 + Factor)` | +110% | +630% |
| HP | `HP × (1 + Factor × 2.25)` | +248% | +1418% |
| 速度 | `+7.5% × Factor` (上限275%) | 108% | 147% |

### 5. 商店变化

| 项目 | 公式/说明 | 波次40效果 |
|------|-----------|------------|
| 物品价格 | 受Endless Factor影响 | +126% |
| 刷新成本 | 大幅增加 | 数倍于普通 |
| 回收价值 | **不受影响** | 仍为25%/60% |

### 6. 特殊机制变化

| 机制 | 公式/说明 | 波次40效果 |
|------|-----------|------------|
| 材料掉落 | 每波-1.5%，最低50% | 50% |
| 消耗品掉落 | `÷(1 + Factor)` | 13.7% |
| 无敌帧 | `÷max(1, Factor)` | 15.9% |
| 百分比伤害 | `÷(Factor × 0.2)`（39波起） | 79% |
| Piggy Bank | 20波后停止生效 | 失效 |
| 收获 | 不再+5%/波，改为-20%/波 | 递减 |
