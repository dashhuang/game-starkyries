### 2. 本地文档的设计分析

本地文档包含Wiki没有的深度分析：

**目标优先级型敌人**：
- Buffer/Stonefish: 最高优先级（强化效果太强）
- Healer/Hermit: 高优先级（持续治疗）

**冲锋威胁型敌人**：
- Pursuer: 加速型（150→600），必须快速击杀
- Charger系列: 周期冲锋

**远程/弹道型敌人**：
- Fly: 特殊机制（反弹远程弹道）
- Spitter: 逃离型远程

**召唤/分裂型敌人**：
- Spawner: 死亡召唤
- Blobfish: 死亡分裂（近战减少）

## 数据格式差异（非错误）

### DLC敌人护甲数据

Wiki格式：
```
Armor: 0.6 (+2/wave)
```

本地格式（护甲列）：
```
| 护甲 |
| - |（无护甲）
| 1 |（有护甲值）
```

**说明**: 本地文档将护甲简化为布尔值或整数，Wiki提供了详细的护甲成长公式。这是格式选择差异，不影响实际使用。

**建议**: 如需精确护甲成长数据，可参考Wiki原文。
