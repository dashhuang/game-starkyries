## 1. 模式概述

### 解锁与启用

| 项目 | 说明 |
|------|------|
| 解锁要求 | 无，默认可用 |
| 启用方式1 | 选择Danger等级时切换开关 |
| 启用方式2 | 第20波Boss战前可选择启用 |
| 胜利判定 | 坚持到20波后死亡计为胜利 |

### 与普通模式对比

| 项目 | 普通模式 | 无尽模式 |
|------|----------|----------|
| 波次上限 | 20波 | 无限 |
| Boss战 | 第20波1次 | 每10波1次 |
| 难度曲线 | 固定 | 指数级增长 |
| 收获加成 | 每波+5% | 每波-20% |

## 2. 核心公式：Endless Factor

### 计算公式

```lua
function CalculateEndlessFactor(current_wave)
    -- 从21波开始计算
    local endless_wave = math.max(0, current_wave - 20)
    
    -- 36波后乘数增加
    local endless_mult = 2.0 + math.max(0, (current_wave - 35) * 0.2)
    
    -- 最终因子
    local endless_factor = math.max(0, 
        ((endless_wave * (endless_wave + 1)) / 2) / 100
    ) * endless_mult
    
    return endless_factor
end
```

## 3. 敌人缩放

### 缩放公式

| 属性 | 公式 | 说明 |
|------|------|------|
| **伤害** | `round(Damage × (1 + Factor))` | 线性增长 |
| **最大HP** | `round(HP × (1 + Factor × 2.25))` | 2.25倍系数增长 |
| **速度** | `Speed × (1 + Factor × 0.075)`，上限275% | 有上限 |

### 敌人属性增长表

| 波次 | 伤害倍率 | HP倍率 | 速度倍率 |
|------|----------|--------|----------|
| 20 | 100% | 100% | 100% |
| 25 | 130% | 168% | 102% |
| 30 | **210%** | **348%** | 108% |
| 35 | 340% | 640% | 118% |
| 40 | **730%** | **1518%** | 147% |
| 50 | 2425% | 5231% | 274% |
| 51+ | - | - | **275% (上限)** |

### 速度上限说明

- 敌人速度最高275%
- 约在51波达到上限
- 之后速度不再增加

## 5. 经济系统变化

### 物品价格

```
无尽模式物品价格 = 普通价格 × (1 + Factor × 系数)
```

| 波次 | 价格增加 |
|------|----------|
| 30 | +22% |
| 40 | +126% |
| 50 | +465% |

### 刷新成本

```lua
刷新成本 = max(1, 上次刷新成本 + max(1, 0.5 × 波次 × sqrt(1 + Factor)))
```

刷新成本增长极快，后期几乎无法刷新。

### 回收价值

**关键**: 回收价值**不受**Endless Factor影响！

| 来源 | 回收比例 | 无尽模式 |
|------|----------|----------|
| 默认 | 25% | 25%（不变） |
| Recycling Machine | 60% | 60%（不变） |
| Entrepreneur | 85% | 85%（不变） |

这意味着后期回收价值远低于购买价格。

## 6. 掉落机制变化

### 材料掉落

```
材料掉落率 = max(50%, 100% - (波次 - 20) × 1.5%)
```

| 波次 | 掉落率 |
|------|--------|
| 20 | 100% |
| 27 | 89.5% |
| 30 | 85% |
| 34+ | **50% (下限)** |

### 消耗品掉落

```
消耗品掉落率 = 基础率 × (1 + Luck/100) / (1 + Factor)
```

| 波次 | Factor | 掉落率（相对） |
|------|--------|---------------|
| 20 | 0 | 100% |
| 30 | 1.10 | 47.6% |
| 40 | 6.30 | **13.7%** |
| 50 | 23.25 | 4.1% |

**例外**: Looters、精英、Boss仍保证掉落消耗品。

### 传奇箱子

- 20波后精英和Boss**不再保证**掉落传奇箱子
- 掉落率受Endless Factor削弱

## 7. 战斗机制变化

### 无敌帧（Iframes）

```
实际无敌帧 = 基础无敌帧 / max(1, Factor)
```

| 波次 | Factor | 无敌帧比例 |
|------|--------|------------|
| 20 | 0 | 100% |
| 30 | 1.10 | 90.9% |
| 40 | 6.30 | **15.9%** |
| 50 | 23.25 | 4.3% |

### 百分比生命伤害削弱

**从第39波开始**，百分比伤害效果被削弱：

```
实际百分比伤害 = floor(当前敌人血量 × 伤害百分比 / max(1, Factor × 0.2))
```

**受影响道具/武器**:
- Chainsaw（链锯）
- Giant Belt（巨人腰带）
- Greek Fire（希腊火）

| 波次 | Factor | 有效率 |
|------|--------|--------|
| 39 | 5.49 | 91% |
| 40 | 6.30 | **79%** |
| 50 | 23.25 | 22% |
| 100 | 485.40 | **1%** |

## 8. 道具特殊表现

### 失效道具

| 道具 | 无尽模式表现 |
|------|--------------|
| Piggy Bank | 20波后完全失效 |
| 收获类道具 | 价值大幅下降（-20%/波） |

### 价值下降道具

| 道具 | 原因 |
|------|------|
| 百分比伤害道具 | 39波后效果削弱 |
| 消耗品触发道具 | 掉落率大幅下降 |

### 价值上升道具

| 道具 | 原因 |
|------|------|
| 固定伤害道具 | 不受Factor削弱 |
| 生存类道具 | 敌人伤害增长快于HP增长 |
| 速度类道具 | 敌人速度有上限 |

## 9. 策略建议

### 构筑重点

| 阶段 | 重点 |
|------|------|
| 1-20波 | 普通构筑 + 为无尽准备 |
| 21-35波 | 提升固定伤害输出 |
| 36-50波 | 生存为主，放弃百分比伤害 |
| 50波+ | 极限生存，依赖无敌/闪避 |

### 避免的陷阱

1. **不要过度投资收获** - 20波后反向增长
2. **不要依赖Piggy Bank** - 20波后失效
3. **不要依赖百分比伤害** - 39波后急剧削弱
4. **不要频繁刷新商店** - 成本增长极快
