## 详细对比

### 1. 核心价格公式 ✅

**Wiki**:
```
Final Price = (Base_Price + Wave + (Base_Price × 0.1 × Wave)) × Shop_Price
```

**本地**:
```
最终价格 = (基础价格 + 波次 + (基础价格 × 0.1 × 波次)) × 商店价格修正
```

**状态**: 完全一致 ✅

### 2. 刷新成本公式 ✅

**Wiki**:
```
Reroll Increase = Rounddown(0.40 × Wave), minimum 1
First Reroll = Rounddown(Wave × 0.75) + Reroll Increase
```

**本地**:
```
刷新递增值 = 向下取整(波次 × 0.40)，最小值为1
首次刷新价格 = 向下取整(波次 × 0.75) + 刷新递增值
```

**状态**: 完全一致 ✅
