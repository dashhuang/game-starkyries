## 二、主属性计算公式差异（重要）

### 2.1 HP Regeneration（生命回复）- 公式需修正

**本地描述**：
> 每5秒恢复固定生命值，基础每5秒恢复1点

**Wiki正确公式**：
```
第1点HP Regen = 0.20 HP/秒
后续每点 = +0.089 HP/秒

例：10点HP Regen = 0.20 + 9×0.089 = 1.001 HP/秒
```

**差异**：本地描述为"每5秒恢复"是错误的，实际是持续恢复（每秒），且有递减效应。

---

### 2.2 Life Steal（生命偷取）- 上限需补充

**本地描述**：
> 数值为1 = 1%概率恢复1点生命，建议点满100点

**Wiki补充**：
```
上限：每0.1秒最多触发1次，即最大 10 HP/秒
机制：角色生命偷取 + 武器基础生命偷取 = 总触发概率
```

**差异**：本地缺少"每0.1秒上限"的重要机制说明。

---

### 2.3 Attack Speed（攻击速度）- 上限需补充

**本地描述**：
> 可能有攻速上限

**Wiki明确**：
```
上限：最快每秒12次攻击
近战武器：有独立的更低上限
计算：高数值时有递减效应（Diminishing Returns）
负值：计算方式与正值不同
```

---

### 2.4 Dodge（闪避）- 上限需修正

**本地描述**：
> 可能有上限

**Wiki明确**：
```
基础上限：60%
Cryptid角色：70%
Ghost角色：90%
负值：视为0处理
```

**差异**：本地未明确上限数值和角色特殊上限。

---

### 2.5 Range（射程）- 机制补充

**本地描述**：
> 对近战武器+X = 射程+X/2像素（减半），且降低攻速

**Wiki补充**：
```
近战武器：射程增加会增加冷却时间（更长的攻击动画）
最小范围：25像素
```
