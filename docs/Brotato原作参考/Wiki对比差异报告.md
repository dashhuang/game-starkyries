### 2.8 无尽模式 ✅ 已补充

**状态**: 已创建 `11-无尽模式.md` 主文档

#### Wiki无尽模式公式:

**无尽因子计算**:
```lua
-- 波数1-20: 正常游戏
-- 波数21-34: 线性增长
-- 波数35+: 指数增长

if wave <= 20 then
    endless_factor = 1.0
elseif wave <= 34 then
    endless_factor = 1.0 + (wave - 20) * 0.1
else
    endless_factor = 2.4 * (1.15 ^ (wave - 34))
end
```

**无尽模式敌人缩放**:
- 生命值: ×endless_factor
- 伤害: ×(1 + endless_factor × 0.5)
- 数量: 基于波数增加

**Boss出现规则**:
- 每10波出现Boss
- 无尽模式Boss属性随endless_factor缩放
