# 资源目录规范

## 目录结构

```
assets/
├── image/
│   ├── {角色名}/          # 每个角色独立文件夹
│   │   ├── 头像.jpg       # 舰长头像（战舰选择界面）
│   │   ├── 坚定.jpg       # 表情立绘（对话系统）
│   │   ├── 微笑.jpg
│   │   └── ...
│   ├── scene/             # 场景背景图
│   └── logo.png           # UI图标等
└── ...
```

## 图片格式规范

| 类型 | 格式 | 说明 |
|------|------|------|
| 角色图片 | **JPG** | 头像、立绘、表情等，节省空间 |
| UI元素 | PNG | 图标、按钮等需要透明通道的素材 |
| 场景背景 | JPG | 大尺寸背景图 |

## 分辨率规范

| 类型 | 分辨率 | 说明 |
|------|--------|------|
| 头像 | **512×512** | 正方形，用于战舰选择界面 |
| 立绘 | **2048×2048** | 正方形，用于对话系统 |
| 场景背景 | 1920×1080 | 横屏背景 |

## 角色图片命名规范

- `头像.jpg` - 角色头像（正方形，用于选择界面）
- `{表情名}.jpg` - 对话立绘（如：坚定、微笑、紧张、加油）
- `战败_开心.jpg` - 带状态前缀的变体

## 质量设置

- JPG 压缩质量：90%
- 推荐使用 ImageMagick 转换：
  ```bash
  # 头像：转换为 512x512 JPG
  convert input.png -resize 512x512! -quality 90 头像.jpg
  
  # 立绘：转换为 2048x2048 JPG
  convert input.png -resize 2048x2048! -quality 90 output.jpg
  ```
