# 星河战姬 Starkyries

一款使用 **TapTap Maker** 开发的太空战舰 Roguelike 自动射击游戏。

玩法参考了《土豆兄弟 Brotato》，题材换成了太空战舰与虫族入侵。操控战舰活过 20 波跳跃，收集武器、搭配模块，还有简单的 Boss 弹幕战和对话剧情。

> **这个项目的所有内容——美术、文案、音乐、代码——全部通过 AI 辅助完成。**
> 没有美术，没有策划，没有音乐人，一个人 + TapTap Maker，从零开始做出来。
> 它也是一个样本，让大家看看现阶段靠 AI 辅助到底能把一款游戏做到什么程度。

## 试玩

在 TapTap 搜索「星河战姬」，或直接访问[游戏页面](https://www.taptap.cn/app/820063)试玩。

## 关于 TapTap Maker

[TapTap Maker](https://maker.taptap.cn/) 是 TapTap 推出的游戏创作工具，基于 UrhoX 引擎，使用 Lua 编写游戏逻辑，支持 AI 辅助编程，可直接发布到 TapTap 平台（移动端 + PC）。

本项目是一个 TapTap Maker 的开源示例，展示了如何用这套工具链完成一款完整游戏的开发。

## 如何使用

1. 打开 [TapTap Maker](https://maker.taptap.cn/)
2. 在聊天窗口中粘贴本项目的 GitHub URL，让 AI 帮你把项目复制到本地
3. 剩下的交给 AI
## 游戏特色

- **自动射击** — 武器自动瞄准最近的敌人，玩家专注移动和走位
- **20 波战役** — 难度逐步递增，第 10/15 波有中 Boss，第 20 波是最终 Boss
- **武器系统** — 力场、电弧、机炮、导弹、激光、载机 6 大类，15 种武器可选
- **模块搭配** — 火控、防御、引擎、战术、实验、特殊、舰桥 7 类被动升级模块
- **波间商店** — 每波结束后购买/出售装备，标签匹配加成，层级概率递进
- **舰桥升级** — 经验升级后选择 Perk，类似 Brotato 的升级选择
- **剧情对话** — 视觉小说风格的对话系统，角色立绘 + 表情 + 语音
- **舰长收藏** — 多位舰长角色，各有专属立绘、性格设定和语音

## 项目结构

```
scripts/           # 游戏源码（Lua）
  main.lua         # 入口文件
  config/          # 数据配置（战舰、武器、模块、敌人、波次）
  core/            # 核心系统（游戏状态、战斗循环、商店、存档）
  entities/        # 游戏实体（玩家、敌人、弹幕、拾取物）
  ui/              # UI 界面（全部基于 NanoVG 渲染）
  render/          # 3D 渲染（背景、材质、舰船模型）
  data/            # 剧情对话数据
  utils/           # 工具库（数学、事件总线、对象池、空间哈希）
assets/            # 资源文件
  audio/           # 音乐 + 音效 + 语音
  image/           # 角色立绘、场景图、Logo
  images/modules/  # 模块图标
docs/              # 设计文档（游戏设计、数值策划、系统设计、叙事设计）
```

## 技术栈

| 层 | 技术 |
|---|------|
| 引擎 | UrhoX（TapTap Maker 内置引擎） |
| 语言 | Lua 5.4 |
| 渲染 | 3D 俯视角 + NanoVG 全量 UI |
| 平台 | 移动端 + PC（通过 TapTap Maker 发布） |

**代码规模**：103 个 Lua 文件，约 47,000 行。
